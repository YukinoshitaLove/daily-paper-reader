## 摘要
在编辑操作下生成一致的视频需要持久性：当编辑修改场景外观或布局时，后续生成应保持时间和视点上的一致。然而，现有的记忆设计在此类修改后难以维持长期一致性，因为存储的上下文可能变得过时或无效。为了解决这个问题，我们提出了PermaVid，一个新颖的框架，它建立在多模态上下文记忆之上，将空间上下文解耦为语义外观和几何结构，并配备了一种编辑感知的记忆更新和检索策略，使记忆演化与后续观察保持一致。具体来说，我们开发了两个互补的记忆库：一个RGB上下文记忆，捕获外观感知的观察同时隐式编码几何，以及一个深度上下文记忆，保留仅与语义解耦的几何结构。基于此设计，我们引入了一个记忆引导的视频生成模型，该模型在从混合模态记忆上下文中提取的参考条件下执行多模态特征融合。实验表明，我们的方法在编辑后保持了强大的长期语义和结构一致性，显著优于现有最先进方法。

## Abstract
Consistent video generation under editing operations requires persistence: when edits modify scene appearance or layout, subsequent generations should remain coherent across time and viewpoints. However, existing memory designs struggle to maintain long-term consistency after such modifications, as stored contexts may become outdated or invalid. To address this, we propose PermaVid, a novel framework built upon a multi-modal context memory that disentangles spatial context into semantic appearance and geometric structure, together with an edit-aware memory update and retrieval strategy that keeps memory evolution aligned with subsequent observations. Specifically, we develop two complementary memory banks: an RGB context memory that captures appearance-aware observations while implicitly encoding geometry, and a depth context memory that preserves geometry-only structure disentangled from semantics. Building on this design, we introduce a memory-guided video generation model that performs multi-modal feature fusion under reference conditions drawn from mixed-modality memory contexts. Experiments demonstrate that our method maintains strong long-term semantic and structural consistency after edits, significantly outperforming state-of-the-art methods.

## 论文核心问题与研究动机
- **核心问题**：现有的视频生成模型在经历编辑操作（如全局风格转换、局部物体替换）后，难以维持长期的一致性。当摄像机视角移动并“回访”先前编辑过的区域时，生成的内容容易出现语义错乱、回归到编辑前状态或结构失真。
- **研究背景与动机**：当前的记忆增强视频模型通常将历史上下文（包括语义外观和几何结构）混杂存储在统一表示中。当编辑操作（特别是全局语义编辑）发生时，整个记忆库都会变得过时。即便场景的几何结构依然有效，也因其与过时外观的耦合而无法被有效复用。因此，亟需一种能将外观与几何解耦的记忆机制，以实现编辑后场景的持久一致生成。

## 方法论：PermaVid 框架
- **核心思想**：将空间上下文分解为两个动态特性不同的独立组分——**语义外观**（动态、易变）和**几何结构**（稳定、局部变化），并为它们设计解耦的记忆库以及编辑感知的更新与检索策略。
- **关键技术细节**：
  - **解耦多模态上下文记忆（Disentangled Multi-modal Context Memory）**：
    - **RGB 记忆库**（\(M_{rgb}\)）：存储外观观测，但因其天然耦合外观与几何，在全局编辑后会整体失效。
    - **深度记忆库**（\(M_{dep}\)）：存储纯净的几何结构信息，不受纹理、光照、风格等编辑影响，可以长期保留复用。
  - **编辑感知的记忆更新与检索（Edit-Aware Memory Update and Retrieval）**：
    - **更新策略**：
      - **全局编辑**（如风格变换）：RGB 记忆库被整体清空并标记新的全局语义版本；深度记忆库完全保留。
      - **局部编辑**（如物体替换）：仅将与编辑区域 \(Ω_e\) 视锥重叠的 RGB（或深度，若几何改变）记忆单元无效化。
    - **检索策略**：在生成时，基于目标摄像机轨迹 \(P\) 的空间相关性（视锥重叠度 \(s_i\)）和语义有效性（RGB 记忆的版本 \(g_i\) 需与当前全局版本 \(g^*\) 匹配）进行筛选，生成混合模态的参考集 \(R_{mem}\)。
  - **记忆引导的视频生成模型（Memory-Guided Video Generation）**：
    - 基于扩散 Transformer（DiT）架构（Wan2.1-14B），并增设一个由 DiT 层复制的**记忆上下文分支（Context Block）**。
    - 该分支独立编码混合模态的稀疏参考记忆 \(R_{mem}\)，并与主 DiT 令牌（融合了文本和摄像机姿态条件）进行特征融合，以指导一致的新视角合成。
  - **数据集构造**：利用虚幻引擎 5 （Unreal Engine 5）和内置导航智能体，自动合成了一个包含 4000 个长视频、每个视频 1000 帧的 **UE-Mem 数据集**。该数据集具有精确的 6-DoF 相机姿态和丰富的回访轨迹。

## 主要结论与发现
- PermaVid 通过将空间记忆解耦为语义和几何两个独立部分，能有效应对视频编辑后的长期一致性问题。
- 该框架在全局编辑下能够持久地传播更新后的语义，同时保持稳定的几何结构。
- 在局部编辑下，当摄像机回访时，模型能可靠地回忆起编辑后的内容，并保持结构完整性。
- 此方法在结构和语义一致性指标上**显著超越**了所有对比的现有最先进方法。

## 优点
- **新颖的解耦视角**：首次明确将视频空间记忆解耦为外观与几何，从根本上解决了编辑后记忆失效的问题，构思巧妙且逻辑清晰。
- **完备的更新检索机制**：针对全局/局部编辑设计的差异化记忆更新与检索规则，确保了理论的有效性和实践的可行性。
- **强大的实验支撑**：不仅进行了全面的 SOTA 对比和消融实验，还专门分析了内存开销，展示了方法在效果和效率上的双重优势。
- **数据集贡献**：自建的 UE-Mem 数据集填补了长回访轨迹视频数据的空缺，对领域有潜在贡献。
- **架构设计的工程灵巧性**：通过冻结预训练的记忆上下文分支（来自 VACE）和分阶段训练策略，有效利用了现有模型能力并降低了训练难度。

## 不足与局限
- **依赖深度预测器**：在推理时需要外部深度预测器来为生成帧构建深度记忆，这可能引入深度估计的误差，成为一个级联误差来源。
- **对编辑类型的粒度假设**：编辑感知的更新策略主要基于全局/局部两种预设类别，可能难以处理更模糊或混合的编辑指令。
- **合成数据的泛化性风险**：训练核心的 UE-Mem 数据集完全来自虚幻引擎合成，尽管质量高，但模型在极端复杂的真实世界长视频编辑下的泛化能力仍有待验证。
- **局部编辑的自动化检测**：文中未详细论述如何从自然语言编辑指令中自动精确获取局部编辑的掩码区域 \(Ω_e\)，这在实际应用中是一个需要前置解决的问题。
