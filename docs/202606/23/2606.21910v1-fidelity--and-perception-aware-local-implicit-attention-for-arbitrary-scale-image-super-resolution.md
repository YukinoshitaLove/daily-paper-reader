## 摘要
任意尺度图像超分辨率（ASISR）旨在从低分辨率输入中重建出高分辨率图像，且尺度因子可连续变化。传统的像素回归方法常产生过于平滑的结果，缺乏真实细节；而近期的扩散方法能生成更清晰、更逼真的纹理。然而，这些扩散技术往往引入结构幻觉的风险。为解决这些问题，我们提出保真度与感知感知局部隐式注意力（FPLIA）框架，该框架有效融合保真度导向特征到扩散流程中，以在ASISR中生成逼真且忠于原图的重建结果。我们引入保真度与感知注意力模块（FPAM），对保真度导向和感知特征应用自注意力和交叉注意力，以增强表示能力。为进一步利用其互补性，我们设计了保真度与感知选择模块（FPSM），自适应选择最具代表性的特征用于RGB值预测。我们进行了大量实验验证这些组件的有效性。定性和定量结果表明，FPLIA在标准ASISR基准上实现了卓越的感知真实感，同时保持了重建精度。源代码可在以下仓库获取：https://github.com/XUSean0118/FPLIA。

## Abstract
Arbitrary-scale image super-resolution (ASISR) aims to reconstruct high-resolution images from low-resolution inputs over a continuous range of upscaling factors. While traditional pixel-regression approaches often produce overly smooth results that lack realistic details, recent diffusion methods can produce sharper and more realistic textures. However, these diffusion techniques frequently introduce the risk of structural hallucinations. To address these issues, we propose Fidelity- and Perception-Aware Local Implicit Attention (FPLIA), a framework that effectively integrates fidelity-oriented features into a diffusion pipeline to produce realistic and faithful reconstructions for ASISR. We introduce a Fidelity and Perception Attention Module (FPAM), which applies both self-attention and cross-attention to fidelity-oriented and perceptual features to enhance representational capacity. To further exploit their complements, we design a Fidelity and Perception Select Module (FPSM) that adaptively selects the most representative features for RGB values prediction. We conduct extensive experiments to validate the effectiveness of these components. Both qualitative and quantitative results show that FPLIA delivers superior perceptual realism while maintaining reconstruction accuracy on standard ASISR benchmarks. The source code is accessible at the following repository: https://github.com/XUSean0118/FPLIA.

### 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心矛盾**：任意尺度图像超分辨率（ASISR）中，像素回归方法（如 LIIF、LIT）以 ℓ₁/ℓ₂ 损失优化，重建忠诚但过度平滑，丢失高频纹理；扩散方法（如 IDM、Kim et al.）虽能生成逼真细节，却因缺乏刚性结构约束，易产生内容“幻觉”（如伪纹理、错误文字）。
- **问题本质**：两种特征流——保真导向特征（Fidelity‑oriented）与感知特征（Perceptual）——统计特性异质，在空间和尺度两个轴上存在互补性：平坦区域和低放大因子时保真特征更可靠，纹理区和高放大因子时感知特征不可缺少。现有方法要么固守单一范式，要么仅进行浅层拼接，未能灵活利用这种互补。
- **研究目标**：设计一个能在每个查询坐标上自适应选择最优特征组合的框架，在不牺牲峰值信噪比的前提下，大幅提升感知相似度，消除扩散方法的幻觉。

### 2. 论文提出的方法论
**总体流程**  
给定低分辨率图像 \(I_{LR}\)，通过冻结的保真编码器 \(E_\theta\)（如 SwinIR）和以此编码器特征为条件训练的感知生成器 \(G_\theta\)（如 Stable Diffusion）分别提取 \(F_f, F_p\)。随后，FPLIA 通过以下两个核心模块在连续坐标空间中预测 RGB 残差，并与双线性上采样相加得到最终 HR 图像。

- **Fidelity and Perception Attention Module (FPAM)**  
  - 对 \(F_f\) 和 \(F_p\) 分别生成查询、键、值投影。  
  - 在局部网格内执行四种方向的跨尺度局部注意力：
    1. \(\tilde{f}_f\)：保真查询→保真键/值（自注意力）
    2. \(\tilde{f}_p\)：感知查询→感知键/值（自注意力）
    3. \(\tilde{f}_{fp}\)：保真查询→感知键/值（保真锚定+感知细节）
    4. \(\tilde{f}_{pf}\)：感知查询→保真键/值（感知驱动+结构约束）
  - 每个方向的输出为一个增强特征向量和对应的注意力图，构成四种候选特征，覆盖“保真‑感知”光谱的不同区域。

- **Fidelity and Perception Select Module (FPSM)**  
  - 利用轻量卷积从增强特征图预测全局可靠性置信图，并结合各分支的注意力图与尺度信息（cell size \(c\)）计算每个坐标上四种候选的得分 \(s = [s_f, s_p, s_{pf}, s_{fp}]\)。  
  - 训练时用 Gumbel‑Softmax 松弛离散选择以允许梯度传播（温度从 1 退火至 0.001）；推理时硬选择得分最高的特征 \(\tilde{f}'\)。  
  - 解码器 \(D_\theta\) 同时监督所有分支输出，确保每个候选都具备独立预测能力。

**关键公式/机制**  
- 注意力计算：\(\tilde{f}, a = \text{CSLA}(q,k,v,\delta x, c)\)，包含位置编码偏置和分辨率 cell 编码。  
- 置信度融合：\(s = \text{FC}(\text{Concat}(a_f \times \text{conf}_f, a_p \times \text{conf}_p, a_{pf} \times \text{conf}_f, a_{fp} \times \text{conf}_p, c))\)。  
- 损失函数：联合最小化 \(\mathcal{L}= \mathcal{L}_{\text{select}} + \mathcal{L}_f + \mathcal{L}_p + \mathcal{L}_{pf} + \mathcal{L}_{fp}\)，其中回归损失为 MSE，感知分支损失用 \(\mu/\sigma\) 加权以模拟扩散去噪过程。

- FPLIA 在所有基准上稳定占据 PSNR‑LPIPS 帕累托前沿：相较最优回归方法，LPIPS 最高提升 29.6%，PSNR 损失不超过 2.6%；相较最优扩散方法，PSNR 和 LPIPS 基本实现双提升。
