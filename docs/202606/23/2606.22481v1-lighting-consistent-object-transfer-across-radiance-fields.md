## 摘要
3D高斯散射（3DGS）被广泛用于捕获和渲染真实场景。将一个捕获中的物体合成到另一个中在视觉特效、建筑与室内设计或市场营销等众多领域都有应用。然而，从源场景中提取物体并直接粘贴到目标场景，由于两场景间光照条件不同，无法产生逼真效果。为解决此问题，我们引入一个扩散模型，用于协调光照不一致的朴素合成图像。该模型使用异构的图像对数据集（不一致的合成输入，一致的输出）进行训练，结合了合成、生成和真实数据。我们完整的3D解决方案允许用户从源场景中提取物体，并将其合成到目标场景中。由此渲染出带合成物体的目标场景（不一致的）视图。我们的扩散模型协调这些视图中的每一个，最终通过后优化步骤在3DGS表示中整合。我们的方法提供了视觉上引人注目的结果，使3DGS间的物体迁移易于使用，且与前人方法相比显著提升了质量。

## Abstract
3D Gaussian Splatting (3DGS) is widely used to capture and render real scenes. Compositing objects from one capture into another has applications in many domains, such as VFX, architecture and interior design, or marketing. However, extracting an object from a source scene and naively pasting it into a target scene will fail to produce realistic results due to the different lighting conditions between the two scenes. To address this problem, we introduce a diffusion model that harmonizes naively composited images with inconsistent lighting. The model is trained with a heterogeneous dataset of image pairs (inconsistent composite input, consistent output), combining synthetic, generated, and real data. Our complete 3D solution allows a user to extract an object from the source scene and composite it into the target scene. From this, the (inconsistent) views of the target scene with the composite object are rendered. Our diffusion model harmonizes each one of these views, which are finally consolidated in a 3DGS representation with a post-optimization step. Our method provides visually compelling results, making object transfer between 3DGS easy to use and significantly improving quality compared to previous methods.

### 2. 论文提出的方法论
DOT3D的方法论分为三个核心阶段：交互式物体合成、单视图光照协调以及多视图一致化整合。

#### 2.1 交互式物体合成
- **物体提取**：首先为源3DGS场景中的每个高斯体赋予一个额外的一维可学习特征。使用预训练的`BiRefNet`生成二维前景蒙版，通过交叉熵损失优化特征，最终根据特征阈值（0.75）分割出属于物体的高斯体。
- **物体放置**：提供一个三维交互界面，用户可将分割出的物体加载到目标3DGS场景中，自由调整其位置、旋转和缩放，生成一个光照不一致的“合成（Naively Composited）”场景。

#### 2.2 单视图光照协调（扩散模型）
此为该论文的技术核心，目标是解决“粘贴”后的单个图像光照不和谐问题。
- **基础架构**：基于先进的一步式文本到图像生成模型`FLUX.1-schnell`进行微调，采用校正流匹配框架。
- **输入与条件**：模型接收噪声潜在变量`zt`，并以合成图像潜在变量`zcomp`和物体蒙版潜在变量`zmask`作为额外条件（两者沿通道维度拼接，且不参与加噪过程）。文本条件设为空字符串。
- **训练损失**：网络`vθ`学习预测速度`ϵ - z0`，目标是使模型依据光照不一致的合成图，生成光照协调的目标图像。

#### 2.3 异构训练数据集构建
为解决缺乏高质量训练数据的问题，构建了一个混合三类数据源的异构数据集来训练上述扩散模型（详细构成见第5项实验部分）。

#### 2.4 多视图一致性整合（3DGS后优化）
由于单视图协调是在每张图上独立进行的，直接结果必然是多视图不一致的，需要将其提升回一致的3D表示。
- **初始化**：用合成场景的3DGS表示作为初始状态，物体的高阶球谐函数（SH）系数被重置为零，以消除源场景的视角依赖效应。
- **优化策略**：只优化高斯体的颜色参数（所有SH波段），而位置、大小等几何属性保持不变。
- **损失函数**：采用**感知损失**（LPIPS），而非像素级的光度损失（L1/SSIM），因为感知损失对语义相似度更敏感，对跨视图的微小不一致性更具鲁棒性。
- **优势**：该步骤仅需运行一次扩散模型（即仅一次数据集更新），比流行的迭代数据集更新（如Instruct-NeRF2NeRF）策略更快、效果更好。

### 8. 不足与局限
- **高频细节丢失**：由于扩散模型内部VAE的潜在空间压缩，对精细纹理或高频结构的物体会产生模糊或块状伪影。
