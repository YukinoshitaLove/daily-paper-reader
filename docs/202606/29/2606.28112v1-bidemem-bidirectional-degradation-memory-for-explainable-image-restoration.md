## 摘要
退化感知提示、条件和潜在先验在图像修复中日益得到应用，但它们通常仅由单一终点评判：即修复图像是否获得了更高的PSNR。这是对语义的弱测试。条件可以通过增加容量、充当全局校正偏置或利用数据集捷径来提供帮助，而不成为可解释的退化先验。我们提出BiDeMem，一种面向可解释图像修复的双向退化记忆。由修复特征和输入统计构建的查询检索出紧凑的top-k记忆槽子集。所选槽的标识在推理时支持修复路径，并在仅训练时前向退化解释路径中发挥作用。研究聚焦于受控多退化NAFNet设置下的可验证性。新的控制分离了仅来自校正头、稠密查询先验和静态全局先验的增益：这些变体分别比BiRank低0.2588、0.2586和0.2839 dB。强残差监督和更宽的退化头也仍低于完整的双向记忆模型。干预探针显示BiRank在保持修复质量的同时增加了错误先验和原生先验的敏感性，将退化记忆构建为既是修复模块又是可证伪的解释机制。

## Abstract
Degradation-aware prompts, conditions, and latent priors are increasingly used in image restoration, yet they are usually judged by a single endpoint: whether the restored image obtains higher PSNR. This is a weak test of semantics. A condition can help by adding capacity, acting as a global correction bias, or exploiting dataset shortcuts, without becoming an interpretable degradation prior. We propose BiDeMem, a bidirectional degradation memory for explainable image restoration. A query built from restoration features and input statistics retrieves a compact top-k subset of memory slots. The same selected slot identity supports the restoration path at inference time and a training-only forward-degradation explanation path. The study centers on verifiability in a controlled multi-degradation NAFNet setting. New controls separate the gain from a correction head alone, a dense query prior, and a static global prior: these variants are 0.2588, 0.2586, and 0.2839 dB below BiRank, respectively. Strong residual supervision and a wider degradation head also remain below the full bidirectional memory model. Intervention probes show that BiRank preserves restoration quality while increasing wrong-prior and native-prior sensitivity, framing degradation memory as both a restoration module and a falsifiable explanation mechanism.

## 1. 研究动机与核心问题

- **背景**：当前图像修复模型广泛使用退化感知的提示（prompts）、条件（conditions）或潜先验（latent priors）来提升性能。然而，这些机制通常仅通过单一的终点评判——如修复后图像的 PSNR 或 SSIM 提升——来评估其有效性。
- **核心问题**：仅靠端点图像质量的提升无法证明退化条件真正编码了有意义的、可解释的退化先验。增益可能来自额外模型容量、全局偏置、稠密条件向量或数据集捷径，而非对输入退化的语义理解。因此，缺少一种可验证的退化先验，其语义合理性能够被严格检验（如错误先验是否会破坏修复效果、同一路径是否能反向解释退化过程等）。
- **整体含义**：本文旨在将退化记忆设计为一种可证伪的解释机制，通过双向记忆结构和受控干预实验，使退化先验既服务于修复任务，又能在反事实检验下表现出可预期的敏感性变化。

## 2. 方法论

- **核心思想**：提出 BiDeMem，一种双向退化记忆。模型从修复特征与输入统计中构建查询，检索紧凑的 top-k 记忆槽。同一组被选中的槽标识在推理时用于调制修复解码器，在训练时同时驱动一个前向退化解释分支，从而强制记忆槽具备双向可验证性。
- **技术细节**：
  - **查询构建**：利用 NAFNet 瓶颈特征 \(z\) 和输入图像的 22 维观察统计（RGB 统计、亮度统计、梯度幅值、FFT 频带能量等），通过线性映射和规范化得到 128 维查询向量 \(q\)。
  - **紧凑 top-k 记忆检索**：记忆包含 \(N=12\) 个槽，维度 \(d=128\)。通过键表 \(K\) 与查询的点积计算分数，经温度 \(\tau=0.2\) 的 softmax 后，执行 top-k（\(k=4\)）选择，仅激活选中的槽，避免均匀分布或静态全局效应。
  - **修复路径**：由激活槽读取的修复先验 \(p_r\) 通过 FiLM 调制 NAFNet 解码器的所有阶段；同时，一个槽路由的校正头接收输入、基础修复结果及二者残差，预测全局校正和槽索引基校正，最终修复图像 \(\hat{y}\) 显式依赖激活槽。
  - **双向解释路径**（仅训练时）：同样的槽标识读取退化先验 \(p_d\)，配合干净目标 \(y\) 预测退化图像 \(\hat{x}\)。分支通过像素、频域、低频和统计空间的多项损失进行监督，迫使同一组槽能同时解释退化过程。
  - **训练目标**：包括修复像素损失 \(L_{pix}\)，先验排序损失 \(L_{rank}\)（区分原生与错误先验），以及退化解释损失等。BiRank Memory 总损失为 \(L_{BiRank} = L_{pix} + \alpha L_{rank} + \beta L_{deg} + \gamma L_{explain}\)。

## 6. 主要结论与发现

- **双向记忆在维持修复性能的同时显著提升了先验的可解释性**：BiRank Memory 在 8 个常见测试集上平均 PSNR/SSIM 为 29.7529/0.8865，略高于单向 Rank Memory，且明显优于所有控制变体。
- **增益不可归因为简单因素**：仅校正头、稠密先验、静态先验、单纯强化退化监督或扩大头容量均无法填补与双向记忆之间的差距，验证了双向槽路由的必要性。
- **干预证据表明先验真正对齐了退化语义**：相比单向模型，双向训练使错误先验造成的 PSNR 降幅从 0.2365 dB 大幅增至 1.0430 dB，原生与非原生先验的差距从 0.3484 dB 扩大到 0.6134 dB。同时，屏蔽激活槽会导致性能下降，而屏蔽非激活槽几乎无影响，说明模型确实依赖选中的少量槽。
- **泛化与复用**：在未见/混合退化上，记忆变体仍带来稳定的 PSNR 提升；在低数据适配情景下，预训练记忆槽可显著提升小样本训练效果，显示了记忆的可复用性。

## 7. 优点

- **创新性理念**：首次将退化记忆明确构建为可证伪的解释对象，提出“复原有用性+干预敏感性”的双重评价标准，超越仅凭 PSNR 判断先验有效性的传统范式。
- **精巧的双向约束**：通过共享槽标识的修复与退化解释双路径，在训练中注入额外的语义约束，使学习到的先验具备反向验证能力，设计具有自洽性。
- **严密的控制实验**：设计了多层控制变体，逐一排除容量、稠密条件、静态偏置、额外监督等可能的混淆解释，逻辑严谨，可重复性强。
- **可审计的干预测试**：引入多种反事实干预（错误先验、洗牌、槽掩码等），将可解释性定量化，为退化先验的检验提供了标准化工具。
- **跨骨架验证**：在 AirNet、PromptIR 等多个外部骨架上进行测试，增强了方法的可信度和潜在通用性。

## 8. 不足与局限

- **效率问题**：虽然参数量紧凑，但记忆检索和校正头导致推理延迟增加，计算效率不升反降，限制了在实时或资源受限场景中的应用。
- **骨架依赖性**：外部骨架测试中增益幅度波动，且 SSIM 并不总是最优，表明双向记忆的优势可能依赖于特定骨架与训练配置，普适性仍有待更广泛验证。
- **实验覆盖局限**：退化类型限于高斯噪声、雨、雾等合成退化，未涉及真实场景中更复杂的混合退化或真实噪声、模糊等情况；未见在更高分辨率或更大规模数据集上的评估。
- **设计与评估的定制性**：记忆槽数、top-k 选择等超参数可能需针对不同任务调优；干预测试虽设计精巧，但可能过度简化了真实世界中退化先验的复杂性。
- **理论深度不足**：对双向记忆为何能提升干预敏感性的内在机理缺乏更深层的理论分析，更多依赖实验现象的解释。
