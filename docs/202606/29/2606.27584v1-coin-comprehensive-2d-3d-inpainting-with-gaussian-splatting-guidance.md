## 摘要
三维场景修复对于重建因遮挡或视角受限而损坏的区域至关重要。尽管近期方法利用高斯溅射（GS）进行高效的三维编辑，但它们通常依赖精确的多视图分割掩码，且本质上局限于物体移除任务。我们提出CoIn，一个通过多阶段一致性流水线连接二维修复模型与3DGS的新颖框架。我们的方法首先使用扩散模型生成初始修复图像，支持任意形状的掩码和物体插入等多种任务。然后，我们引入带有特征注意力的参考自适应GS，通过自适应加权参考视图重建粗三维场景（2D->3D）。该三维表示通过基于GS的参考特征变形为扩散过程提供几何指导，确保多视图一致性（3D->2D）。最后，纹理增强鉴别器优化三维场景以实现高度逼真的光照真实感（2D->3D）。实验表明，CoIn有效利用双向信息流，实现了最先进的性能，并能用灵活的掩码输入有效处理物体移除和物体插入。

## Abstract
3D scene inpainting is essential for reconstructing areas corrupted by occlusions or limited viewpoints. While recent methods leverage Gaussian Splatting (GS) for efficient 3D editing, they often depend on precise multi-view segmentation masks and are inherently constrained to object removal tasks. We propose CoIn, a novel framework that bridges 2D inpainting models and 3DGS through a multi-stage consistency pipeline. Our approach first generates initial inpainted images using a diffusion model, enabling the use of arbitrary-shaped masks and diverse tasks like object insertion. We then introduce Reference Adaptive GS with Feature Attention to reconstruct a coarse 3D scene by adaptively weighing towards a reference view (2D -> 3D). This 3D representation provides geometric guidance to the diffusion process via GS-based Reference Feature Warping, ensuring multi-view consistency (3D -> 2D). Finally, a Texture-Enhancing Discriminator refines the 3D scene to achieve high photometric realism (2D -> 3D). Experiments show that CoIn, effectively leveraging bidirectional information flow, achieves state-of-the-art performance and effectively handles both object removal and object insertion with flexible mask input.

## 1. 核心问题与整体含义
- 三维场景修复（3D inpainting）在遮挡、传感器噪声或视角受限等情况下不可或缺，传统的 NeRF 方法优化复杂、编辑困难，而最近的高斯溅射（3DGS）方法虽更高效，却普遍采用 **“3D‑first”流水线**：先在原始多视图上重建 3DGS，再通过剪枝高斯点实现物体移除。
- 此类方法需要**极其精确的多视图分割掩码**，否则会在 3D 空间中产生过量删除或不一致（图 1）；且**任务单一，只能做移除**，难以扩展至物体插入。
- 另一类“2D‑first”流水线（先在每张图上做 2D 修复，再用光流/网格等保一致）虽更灵活，却常面临跨视图不一致、需要额外 3D 标签或微调等问题。
- **论文试图解决的核心问题**：如何构建一个既支持**任意形状掩码、物体插入与移除**，又能保证**多视图几何与外观一致**的高效 3D 修复框架。
- 整体含义：通过“2D 修复→粗 3D 重建→3D 引导 2D 再修复→3D 精化”的多阶段双向流水线，打通了生成式 2D 模型与显式 3D 表示之间的壁垒。

## 2. 方法论
CoIn 分为四个模块化的核心阶段，整体流程见图 2。

### 2.1 初始 2D 修复（Initial Stage）
- 使用 Stable Diffusion 2.0 的修复 pipeline，对每张视图的掩码区域进行 2D 补全，得到初始修复图像组 \(\{ \hat{I}_n \}\)。
- 该结果在单张视图上视觉合理，但跨视图一致性差。

### 2.2 参考自适应高斯溅射与特征注意力（Ref‑GS, Coarse Stage）
- 在初始修复图像上训练 3DGS，但引入两项改进：
  - **自适应参考权重**：为每个视图 \(n\) 计算损失权重 \(W_n\)，对参考视图 \(k\) 上权至 \(\lambda_r\)，其他视图根据其光测损失变化通过 sigmoid 函数降权，强制 GS 场景向参考视图对齐。
  - **锚点特征单向注意力**：针对 Scaffold‑GS 的锚点结构，对修复区域内锚点特征 \(f^A_{\text{in}}\)，使用修复区域外侧锚点特征 \(f^A_{\text{side}}\) 进行 Query‑Key‑Value 注意力更新，使内外纹理一致。
- 同时使用单目深度估计生成参考视图的深度，并通过最小二乘对齐后施加深度损失 \(L_D\)，稳定参考视图几何。

\[\text{总损失： } L = W_n \cdot L_R(R_n, \hat{I}_n) + \lambda_D L_D\]

### 2.3 一致性损失引导（CLG, Guidance Stage）
- 将粗重建的点云 \(P_k\) 和 GS 渲染作为 3D 先验，通过能量函数引导扩散模型在去噪过程中保持一致。
- **GS 基准参考特征变形**：在参考视图 \(k\) 与当前视图 \(n\) 之间插值出一个中间视角 \(\phi\)，利用 \(P_k\) 将高频特征（DINO+FeatUp）和 RGB 特征投影到 \(\phi\)，计算参考图与当前去噪输出 \(X^{(n,t)}\) 之间的余弦相似度（高频）和 L2 距离（RGB），构成 3D 一致性损失 \(L_{3D}\)（公式 6）。
- 利用点云支撑掩码 \(m_{P_n}\) 区分有 3D 对应点的区域与无对应区域，无对应区域直接用 GS 渲染 \(R_n\) 进行 L1 引导，形成最终能量函数（公式 7）。
- 在扩散采样的每一步中，沿负梯度方向更新隐空间样本，从而在生成过程中注入多视图一致性。

### 2.4 纹理增强鉴别器（TE‑D, Fine Stage）
- 在最后一个阶段，对 GS 场景进行微调，采用**局部补丁对抗训练**：以初始修复图像 \(\hat{I}_n\) 的补丁为“真”，以当前 GS 渲染的补丁为“假”，使用条件 GAN 损失，将 \(\hat{I}_n\) 中的高频纹理迁移到渲染中。
- 同时保持渲染与 CLG 修复图像 \(\tilde{I}_n\) 之间的光度一致性，从而在提升真实感的同时不破坏几何结构。

## 3. 实验设计
### 3.1 数据集
- **SPIn‑NeRF**[28]：10 个场景，每场景 60 物体存在图像 + 40 物体缺失图像，提供分割掩码及导出的包围盒掩码。
- **IMFfine**[37]：20 个场景，按视角覆盖度分 90°、120°、180°、360° 四类，每场景 125 物体存在 + 75 物体缺失图像，含分割掩码及相机参数。

### 3.2 评估指标与基准
- **物体移除**：LPIPS、PSNR、FID，并特别计算掩码内部指标 m‑LPIPS、m‑PSNR、m‑FID。
- **物体插入**：CLIP 方向相似度（CLIP dir）及用户研究（多视图一致性、总体质量、文本对齐、无伪影）。
- **对比方法**：NeRF 系（SPIn‑NeRF、MVIP‑NeRF、MALD‑NeRF）；3DGS 系（Gaussian Grouping、3DGIC、GScream、IMFfine）；插入任务则对比 Gaussian Editor、Infusion 等。

### 3.3 消融与补充实验
- 消融研究：逐步移除 Ref‑GS（含自适应权重、特征注意力）、CLG、TE‑D，验证每个组件的必要性。
- 额外实验：包围盒掩码效果、嵌入 3D‑first 流程处理 360° 场景、对不规则掩码的鲁棒性、对参考视图选择的鲁棒性。

## 6. 主要结论与发现
- CoIn 在 **物体移除** 任务上取得最先进性能，尤其 m‑LPIPS、m‑FID 等掩码内指标大幅领先，且从分割掩码切换到包围盒掩码时性能下降微小（Tab 1），证明了对掩码形状的鲁棒性。
- 在 **物体插入** 任务上，CLIP dir 分数与用户偏好均超越现有方法，能生成与场景无缝融合且遵循文本提示的新物体。
- 消融实验证实：参考自适应权重、特征注意力、一致性引导、纹理鉴别器四个组件缺一不可，任一缺失都会导致指标显著下降或模糊/伪影加重。
- 将 CoIn 的 2D‑first 流程与 3D‑first 的精确掩码剔除相结合，可有效处理大视角覆盖场景（如 360°），兼具两派优势。
- 深层发现：用一致性引导后的 2D 修复图像训练 3DGS，不仅提高渲染真实性，还改善了深度图的几何稳定性（图 6B）。

## 7. 优点（亮点）
- **灵活性突破**：一举摆脱传统 3D 修复对精准分割掩码的依赖，支持包围盒、不规则涂抹，甚至物体插入，大大拓展了 3D 编辑的实用边界。
- **多阶段双向融合**：将 2D 生成模型的语义多样性与 3DGS 的显式几何约束通过 CLG 紧密结合，实现了从 2D→3D→2D→3D 的信息闭环。
- **轻量高效**：单卡 4090、2.5 小时可完成全流程，优于多数 NeRF 或训练型基线。
- **实验设计扎实**：覆盖两个主流数据集、多类掩码、移除与插入双重任务，并采用客观指标+主观用户研究，可信度高。

## 8. 不足与局限
- 基础生成模型受限：2D 修复依赖 Stable Diffusion 2.0，其文本理解和掩码填充质量影响整体上限；当掩码未能完全覆盖目标时，移除质量会下降。
- 大视角失败的场景：当掩码占比大且视角变化超过 180° 时，纯 2D‑first 流程会因上下文不足产生模糊纹理（图 3c）；虽可通过嵌入 3D‑first 缓解，但该混合策略需要在准确掩码存在的条件下才能施展。
- 参考视图选择的依赖：虽然实验表明对参考视图不敏感，但论文仍建议选择最有代表性的角度，实际操作中可能需要人工判断。
- 评测的局限性：插入任务的 CLIP 方向相似度和用户研究虽较全面，但用户规模（18 人）偏小，可能影响统计稳定性。
- 补充数据集覆盖度：虽然使用了 SPIn‑NeRF 和 IMFfine，但场景类型和多样性仍然有限，真实世界复杂场景下的泛化性尚未验证。
