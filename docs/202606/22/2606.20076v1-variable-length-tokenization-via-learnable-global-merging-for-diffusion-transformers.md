## 摘要
潜在扩散模型（LDMs）已成为视觉合成的主流，但其质量与计算量的权衡主要受分词器固定压缩比的制约。变长分词器（VLTs）通过改变 token 数量实现自适应压缩，使扩散模型能够灵活平衡质量与计算量。然而，传统 VLT 通过截断有序 token 序列来调节长度，这导致 token 语义依赖于 token 位置，破坏了不同长度之间的表示对齐。这会引起潜在分布中的跨长度偏移，阻碍单个变长扩散模型有效运行。为解决这一问题，我们提出了一种新颖的变长分词器，它通过合并 token 来调节长度。我们证明，当扩散变换器按照合并模式运行时，鼓励相似 token 合并能够实现直接的跨长度表示对齐。由于传统合并方法依赖于数据，导致在生成过程中无法获取合并模式，我们引入了数据无关的可学习全局合并，以确保与扩散变换器的兼容性。在 ImageNet 256×256 生成任务上，与之前的 VLT 方法相比，我们基于合并的变长分词器与扩散变换器结合，实现了更优的 gFID-计算量权衡。代码可在 [此链接](https://github.com/movinghoon/lgm) 获取。

## Abstract
Latent Diffusion Models (LDMs) have become dominant in visual synthesis, but their quality-compute trade-off is largely constrained by the tokenizer's fixed compression ratio. Variable-length tokenizers (VLTs) promise adaptive compression by varying token counts, allowing diffusion models to flexibly balance quality and compute. However, conventional VLTs modulate length by truncating ordered token sequences, which makes token semantics depend on token position and breaks representational alignment across lengths. This leads to a cross-length shift in the latent distribution that hinders a single variable-length diffusion model from operating effectively. To address this, we propose a novel variable-length tokenizer that modulates length by merging tokens. We show that encouraging similar tokens to merge enables direct cross-length representation alignment when the diffusion transformer operates according to the merging pattern. Since conventional merging methods are data-dependent, making the merging pattern inaccessible during generation, we introduce learnable global merging, which is data-independent, to ensure compatibility with diffusion transformers. On ImageNet 256$\times$256 generation, our merging-based variable-length tokenizer integrated with a diffusion transformer achieves a superior gFID-compute trade-off compared to prior VLT methods. ## 2. 论文提出的方法论

### 2.1 基于合并的长度调制
- 不采用截断，而是通过将全长度 token 序列 **合并（merge）成更少的 token** 来调节长度。
- 利用一个给定的合并分配矩阵 \(\Gamma \in \{0,1\}^{N \times (N-K)}\)，将 \(N\) 个 token 归入 \(N-K\) 个簇，通过加权平均得到调制后的 token \( \tilde{z} = \bar{\Gamma}^\top z\)。
- **关键洞察**：在解码器和扩散模型中若使用**比例注意力**（根据合并簇的大小 \(m_j\) 调整注意力分数），则对合并后 token 的处理等效于对一个投影后的全长度 token 矩阵 \(Wz\) 进行标准注意力操作。这使得不同长度下的表示可以直接对齐，对齐目标为最小化 \(\|Wz - z\|^2\)，即**把相似的 token 合并到同一簇**。

### 2.2 可学习全局合并（Learnable Global Merging）
- 传统数据依赖的合并（如 ToMe）在生成时无法获得合并模式，因而无法使用比例注意力或正确的合并位置编码。
- 提出**数据无关的全局合并**：引入一组可学习嵌入 \(e \in \mathbb{R}^{N \times D}\)，对 \(e\) 进行层次聚类得到合并分配 \(\Gamma\)，再将其用于 token 合并。
- 为使嵌入 \(e\) 可训练，使用**直通估计**：计算软分配 \(\Gamma_{\text{soft}} = \text{softmax}(e c^\top / \tau)\)，并在合并时使用 \([\Gamma - \Gamma_{\text{soft}}]_{\text{sg}} + \Gamma_{\text{soft}}\)。
- 引入**对齐损失** \(L_{\text{align}}\)，鼓励 token 间的内积相似度与嵌入间的相似度对齐（仅对超过阈值 \(\delta\) 的正相关部分惩罚），从而促使全局合并倾向于合并相似 token。
- 整体训练目标：\( \min_{e, E, D} L_{\text{total}} + \lambda_{\text{align}} L_{\text{align}} \)。

### 2.3 扩散 Transformer 中的集成
- 在 DiT 中采用比例注意力，并使用与 token 合并方式相同的合并位置嵌入 \(\tilde{pe} = \bar{\Gamma}^\top pe\)。
- 训练时随机采样不同 token 长度 \(K\)，联合优化噪声预测目标。
- 提出后训练微调策略：在联合训练后引入长度专属的 LoRA 参数，仅需少量额外训练 epoch 即可进一步提升特定长度下的生成质量。

## 6. 论文的主要结论与发现

- 基于合并的长度调制比基于截断的方法更有利于保持跨长度的表示对齐，大幅减少联合训练时不同 token 数下的质量退化。

## 8. 不足与局限

- **全局合并并非数据最优化**：合并模式独立于输入图像，极限压缩下重建/生成质量可能弱于数据依赖的合并方法（尽管差距不大，但依然存在）。
