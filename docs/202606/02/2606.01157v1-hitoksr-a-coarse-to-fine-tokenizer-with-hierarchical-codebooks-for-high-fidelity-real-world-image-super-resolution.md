## 摘要
矢量量化（VQ）生成模型在真实世界图像超分辨率（Real-ISR）中展现出 promising 的结果。然而，现有方法通常依赖于一个单一的潜空间，该空间将低频结构与高频纹理纠缠在一起。这种纠缠迫使单个码本去捕捉结构-纹理配对的组合复杂性，从而限制了表示能力和码本利用率。为解决这一问题，我们提出 HiTokSR，一种分层 token 预测框架。HiTokSR 不是使用单一码本，而是沿着通道维度将潜空间划分为频率感知的组，并用独立的子码本对每个组进行量化。这种由粗到细的设计将全局结构与细节解耦，增强了组合表达能力，同时避免了高维最近邻查找的优化不稳定性。为进一步提升语义一致性，我们的生成器通过自适应特征调制、多尺度类别 token 和表示对齐损失整合了视觉基础模型先验。此外，我们在解码器微调过程中引入索引级扰动策略，以弥合离散 token 预测中的训练-测试差异。在真实世界基准上的大量实验表明，HiTokSR 在感知质量和重建保真度方面均达到了最先进的性能。

## Abstract
Vector-quantized (VQ) generative models have shown promising results in real-world image super-resolution (Real-ISR). However, existing methods typically rely on a monolithic latent space that entangles low-frequency structures with high-frequency textures. This entanglement forces a single codebook to capture a combinatorially complex set of structure-texture pairings, which constrains representational capacity and limits codebook utilization. To address this issue, we present HiTokSR, a hierarchical token prediction framework. Instead of using a single codebook, HiTokSR partitions the latent space along the channel dimension into frequency-aware groups, quantizing each with an independent sub-codebook. This coarse-to-fine design disentangles global structures from fine details, enhancing combinatorial expressiveness while circumventing the optimization instability of high-dimensional nearest-neighbor lookups. To further improve semantic consistency, our generator integrates priors from a vision foundation model via adaptive feature modulation, multi-scale class tokens, and a representation alignment loss. Additionally, we introduce an index-level perturbation strategy during decoder fine-tuning to bridge the train-test discrepancy in discrete token prediction. Extensive experiments on real-world benchmarks demonstrate that HiTokSR achieves state-of-the-art performance in both perceptual quality and reconstruction fidelity.

## 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**：现有基于矢量量化（VQ）的真实世界图像超分辨率（Real-ISR）方法通常使用单一潜空间，该空间将低频结构（如边缘、轮廓）与高频纹理（如细节、噪声）纠缠在一起。这种设计迫使单个码本必须记忆大量结构-纹理配对组合，导致表示能力受限、码本利用率低下。
- **背景**：GAN 和扩散模型在超分中各有缺陷（GAN 不稳定、伪影；扩散模型多步推理计算成本高、语义漂移）。VQ 方法具备单次前向推理优势，但受到码本设计的根本限制——扩展码本容量会导致高维最近邻查找不稳定（维度灾难），且结构-纹理纠缠进一步加剧优化困难。
- **整体含义**：HiTokSR 提出一种分层码本框架，通过解耦结构与纹理来同时增强表示能力和推理效率，实现感知质量与重建保真度的平衡。

## 2. 论文提出的方法论

### 2.1 核心思想：粗到细的分层分词器
- 将 ViT 编码器输出的连续潜特征沿通道维度划分为 N 个非重叠组，每组独立使用一个低维子码本进行矢量量化。
- 低维子空间避免了维度灾难，提高了最近邻查找的可靠性；不同组可自然专注于不同频带。

### 2.2 关键技术细节

#### 2.2.1 分组矢量量化
- 公式：`z = [z1, z2, ..., zN]`，`zi ∈ R^{h×w×(C/N)}`；每个组 `zi` 用子码本 `Ci ∈ R^{K×(C/N)}` 量化。
- 每个组的 token 索引 `ki = argmin ||zi - Ci[k]||₂`。

#### 2.2.2 组掩蔽策略（Group Masking）
- 训练时随机掩蔽后缀组，活跃组数 M 按重尾分布采样：90% 概率使用全部 N 组，10% 概率 M ∈ {1,...,N-1} 均匀采样。
- 掩蔽后的表示 `˜z` 中，被掩组置零。迫使早期组学习粗结构，后期组学习残差细节。

#### 2.2.3 频率匹配监督（Frequency-Matched Supervision）
- 对 GT 图像执行 (N-1) 层离散小波变换，获得从粗到细的近似序列 T^(0)~T^(N-1)。
- 重建目标 `Ttarget = T^(N-M)`，与活跃组数 M 一一对应。当 M 小（粗）时目标为低频近似，M 大时引入高频细节。
- 结合重建损失（L1+LPIPS+GAN）和承诺损失训练 tokenizer。

#### 2.2.4 语义引导的 token 生成
- 采用 CNN-Transformer 混合生成器，从低分辨率输入预测 token 索引。
- 集成视觉基础模型（VFM，默认 DINOv3-B）先验：
  - **自适应特征调制**：VFM patch token 预测通道尺度/偏移参数，通过自适应归一化调制 Transformer 特征。
  - **多尺度 CLS token**：从 VFM 各层提取 CLS token 拼入生成器输入序列，提供全局语义摘要。
  - **表示对齐损失（REPA）**：对齐低质量输入与 GT 的 VFM 特征，使低质特征接近清晰语义。
- VFM 通过 LoRA 微调以适应退化分布。

#### 2.2.5 解码器微调（索引级扰动）
- 冻结生成器与子码本，仅更新解码器。
- 从生成器 softmax 输出的 top-K 预测中随机采样 token 作为解码器输入（而非确定性 top-1），迫使解码器适应预测误差。
- 微调后推理时仍使用 top-1，实现鲁棒性。

### 2.3 训练流程（三阶段）
1. **Tokenizer 训练**：重建损失 + 承诺损失，600k 轮。
2. **Generator 训练**：交叉熵损失 + 像素损失（L1, LPIPS, GAN）+ REPA 损失，600k 轮。
3. **Decoder 微调**：L1+LPIPS+GAN 损失，50k 轮。
