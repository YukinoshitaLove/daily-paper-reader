## 摘要
基于扩散先验的方法在真实世界图像超分辨率（ISR）中取得了令人印象深刻的结果，但仍存在两个关键挑战：平衡像素级保真度与语义质量，以及适应多样化的退化。现有的双分支方法在语义训练期间冻结像素模块，但语义分支仍可以在像素子空间内扩展容量，从而阻碍了真正的感知质量提升。此外，使用单一静态适配器无法泛化到异构的真实世界退化类型。为了解决这两个问题，我们提出了FreqOrtho-SR，它包括：频率引导的LoRA专家混合（FreqMoE），它通过一个非参数的基于FFT的退化特征提取器将输入路由到专业专家，该提取器编码频域特征，从而在不同退化类型上实现稳定且可解释的专门化；以及正交梯度投影（OGP），它将双目标优化重新构建为一个子空间约束问题：通过SVD对组合的专家权重增量提取像素保真度子空间，并将语义梯度投影到其零空间，OGP保证了两个目标之间的正交性，从而实现真正互补的学习而不会相互干扰。实验表明，FreqOrtho-SR在多个基准测试上取得了有竞争力的整体性能，并在高效的单步推理中实现了强大的保真度-感知权衡。我们的方法源代码可在 https://github.com/sonhm3029/FreqOrtho-SR 找到。

## Abstract
Diffusion prior-based methods have shown impressive results in real-world image super-resolution (ISR), yet two key challenges persist: balancing pixel-level fidelity with semantic quality, and adapting to diverse degradations. Existing dual-branch approaches freeze the pixel module during semantic training, but the semantic branch can still expand capacity within the pixel subspace, precluding genuine perceptual improvement. Moreover, using a single static adapter cannot generalize across heterogeneous real-world corruptions. To address both issues, we propose FreqOrtho-SR, which comprises: $\textbf{Freq}$uency-guided Mixture of LoRA Experts (FreqMoE), it routes inputs to specialized experts via a non-parametric FFT-based degradation-feature extractor that encodes frequency-domain signatures, enabling stable and interpretable specialization across corruption types; and $\textbf{Ortho}$gonal Gradient Projection (OGP), which reframes the dual-objective optimization as a subspace-constrained problem: by extracting the pixel-fidelity subspace via SVD on combined expert weight deltas and projecting semantic gradients onto its null space, OGP guarantees orthogonality between the two objectives, enabling genuinely complementary learning without mutual interference. Experiments show that FreqOrtho-SR achieves competitive overall performance and a strong fidelity-perception trade-off across multiple benchmarks with efficient single-step inference. The source code of our method can be found at $\href{https://github.com/sonhm3029/FreqOrtho-SR}{\texttt{sonhm3029/FreqOrtho-SR}}$.

## 1. 核心问题与研究动机
- **核心挑战**：基于扩散先验的真实世界图像超分辨率（ISR）存在两大难题。
  - **平衡像素保真度与语义质量**：回归型损失能获得高 PSNR/SSIM，但结果过于平滑；生成式方法增强纹理但容易引入伪影。
  - **适应异构退化**：真实世界退化类型多样（模糊、噪声、JPEG 压缩等），单一静态适配器难以应对所有情况。
- **现有方法的不足**：
  - 双分支架构（如 PiSA‑SR）虽然冻结像素分支再训练语义分支，但语义分支仍可能在像素子空间中扩张，造成表示冗余，无法实现真正的互补。
  - 缺乏对语义更新方向的显式约束，且缺乏依据退化类型动态调配模型容量的机制。
- **整体目标**：在单步扩散框架下，通过频率引导的专家混合与正交梯度投影，实现保真度与感知质量的更好权衡，同时具备退化自适应性。

## 2. 方法论
### 2.1 总体框架
- 基于 Stable Diffusion 2.1 的 VAE 潜空间，采用单步残差学习：\(z_H = z_L - \epsilon_\theta(z_L)\)。
- 引入双 LoRA 设计：**像素级 FreqMoE**（频率引导的 LoRA 专家混合）与**语义 LoRA**，并采用两阶段训练。
  - 阶段一：训练 FreqMoE 以恢复高频结构。
  - 阶段二：冻结 FreqMoE，训练语义 LoRA 并施加正交梯度投影（OGP）。

### 2.2 频率引导的 LoRA 专家混合（FreqMoE）
- **动机**：不同退化在频域具有特征性指纹（如模糊抑制高频，噪声抬升全频带，JPEG 产生 8×8 块状伪影等）。
- **退化特征提取器**（非参数、无梯度）：
  - 将输入 LR 转换为灰度图，计算 2D FFT 幅度谱，划分成 \(K\) 个径向波段，得到归一化能量分布 \(e\)。
  - 同时计算三个标量指标：模糊分数 \(s_{blur}\)、JPEG 块效应分数 \(s_{jpeg}\)、噪声分数 \(s_{noise}\)。
  - 最终特征向量 \(f \in \mathbb{R}^{K+3}\) 作为全局退化描述符。
- **门控网络**：
  - 对于 UNet 每一层的输入 \(x\) 和全局退化特征 \(f\)，计算门控 logits：\(\ell = W_g x + \phi(f)\)，其中 \(\phi\) 为小型 MLP。
  - 采用 top‑k 稀疏路由（默认 k=2），通过 softmax 得到专家权重 \(g_i\)。
- **专家结构**：
  - 每层包含 \(N\) 对 LoRA 专家（默认 N=4），输出为加权和 \(\Delta h_{pix} = \sum_{i \in \text{Top-k}} g_i B_i A_i x\)。
- **训练稳定措施**：使用负载均衡损失 \(L_{lbl} = \alpha N \sum_i f_i P_i\)，防止专家坍缩。

### 2.3 正交梯度投影（OGP）
- **问题形式化**：语义 LoRA 直接训练可能学到的仍是像素子空间内特征，导致表示冗余。
- **子空间提取**：
  - 阶段一结束后，将所有 FreqMoE 专家的权重增量按列拼接：\(\Delta W_{all} = [\Delta W_1|\cdots|\Delta W_N]\)。
  - 对 \(\Delta W_{all}\) 进行 SVD，保留前 \(\tilde{k}\) 个左奇异向量（覆盖 95% 能量），构成像素保真度子空间基 \(U_{\tilde{k}}\)。
- **梯度投影**：
  - 阶段二训练时，对语义 LoRA 的 B 矩阵梯度进行投影：\(G_{\text{ortho}} = G - U_{\tilde{k}}(U_{\tilde{k}}^T G)\)。
  - 理论上，仅投影 B 矩阵即可保证完整权重增量 \(\Delta W_{\text{sem}} = B A\) 与像素子空间正交（因为 \(U_{\tilde{k}}^T \Delta W_{\text{sem}} = (U_{\tilde{k}}^T B) A\)，若 \(U_{\tilde{k}}^T B = 0\) 则正交）。
- **训练策略**：
  - 阶段一：Loss = \(\lambda_{l2} L_2 + \lambda_{lbl} L_{lbl}\)。
  - 中间：SVD 提取并保存 \(U_{\tilde{k}}\)。
  - 阶段二：Loss = \(\lambda_{l2} L_2 + \lambda_{lpips} L_{lpips} + \lambda_{csd} L_{csd}\)，同时开启 OGP 钩子。
- **可调节推理**：通过像素引导尺度 \(\lambda_{pix}\) 和语义引导尺度 \(\lambda_{sem}\) 实现可控的保真-感知权衡，无需重新训练。

## 6. 主要结论与发现
- FreqOrtho‑SR 在多个基准上取得**有竞争力或最优的全参考/无参考指标**，尤其在真实世界测试集上显著提升感知质量（LPIPS/FID），同时保持高保真度（PSNR/SSIM）。
- **频率引导的 MoE** 使不同专家自然分化为针对不同退化类型的专长，FFT 特征提供了比纯空间门控更稳定的路由信号，避免了专家崩溃。
- **正交梯度投影** 通过数学约束将语义 LoRA 的学习限制在像素子空间的零空间，有效消除表示冗余，带来 FID、DISTS、NIQE 等多指标的同步改善。
- 可调节推理使模型能够灵活控制保真与感知的折衷，扩展了应用场景。

## 7. 优点
- **新颖且实用的设计**：将频率分析自然融入 MoE 路由，无需大型预训练模型，非参数特征提取轻量且可解释。
- **严格的数学框架**：引入 OGP 将多目标优化转化为子空间约束问题，提供了正交性保证，区别于单纯的参数冻结。
- **性能与效率的平衡**：在单步扩散中达到甚至超过多步方法的感知质量，参数与显存开销与最优方法相当。
- **丰富的实验验证**：消融详尽，分析深入（如子空间重叠的量化），可调节机制实用，OOD 测试表明一定泛化能力。

## 8. 不足与局限
- **固定的 SVD 基**：OGP 假设像素分支在训练语义分支时完全冻结；若未来需要联合微调或像素子空间漂移，固定基不再适用。
- **推理延迟略高**：MoE 的 top‑k 计算和 FFT 特征提取使单次推理时间（0.30 s）高于 PiSA‑SR 等极简方法，虽可通过剪枝或结构化稀疏优化。
- **训练数据依赖**：使用 RealESRGAN 合成退化，尽管在 RealLR200 上进行了 OOD 测试，但更复杂的真实退化（如多重压缩、传感器噪声）的泛化仍需进一步检验。
- **专家数目固定**：N=4 为手工设定，未来可探索动态专家数量以及更自适应的路由策略。
- **仅适用扩散先验**：方法论与 SD 2.1 绑定，未验证在其他扩散 backbone 或非扩散模型上的迁移性。
