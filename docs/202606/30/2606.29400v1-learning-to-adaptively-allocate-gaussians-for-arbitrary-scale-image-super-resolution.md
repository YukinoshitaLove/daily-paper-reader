## 摘要
在计算机图形学中，视觉内容不断被扭曲、缩放和重采样。这发生在引擎提升帧率、用户缩放三维场景或注视点虚拟现实应用可变缩放时。处理这些变换需要任意尺度超分辨率（ASR）。传统模型专为固定尺度设计，通常预测较低整数倍（如4倍）并依赖次优插值获得连续分辨率，损害了质量。此外，多数方法对像素均匀处理。由于精细细节稀疏，这带来开销；效率要求仅将资源集中在结构复杂性需要的地方。虽然隐式模型和高斯泼溅（GS）支持连续表示，但GS因自适应致密化而具有优势。然而，将GS转变为用于ASR的前馈模型并非易事。标准GS优化需要高分辨率梯度来驱动基元生长，这在推理时不可用。因此，网络必须从低分辨率输入自主预测GS致密化。为解决此问题，我们提出QuADA-GS。在将输入编码到潜在空间后，神经路由架构评估局部复杂性以分配全局预算，为特征分配特定的上采样因子以避免冗余处理。基于这些因子动态致密化特征，形成不规则拓扑并解码为二维高斯基元。为在解码前协调特征，我们引入层次化指针卷积。这种非网格算子实现O(1)邻域查找复杂度，促进高效空间通信并绕过密集瓶颈。实验表明QuADA-GS实现了最先进的ASR性能，同时保持低延迟和精简的内存占用。

## Abstract
In computer graphics, visual content is continuously warped, zoomed and resampled. This occurs when engines upscale frames, users zoom into 3D scenes, or foveated VR applies varying scaling. Handling these transformations requires Arbitrary-Scale Super-Resolution (ASR). Traditional models, designed for fixed scales, typically predict at a lower integer scale (e.g., x4) and rely on sub-optimal interpolation for continuous resolutions, compromising quality. Furthermore, most methods process pixels uniformly. Since fine details are sparse, this creates overhead; efficiency dictates concentrating resources only where structural complexity demands it. While implicit models and Gaussian Splatting (GS) enable continuous representation, GS is advantageous due to adaptive densification. However, transitioning GS into a feed-forward model for ASR is non-trivial. Standard GS optimization needs high-resolution gradients to drive primitive growth, which are unavailable during inference. Thus, the network must autonomously predict GS densification from low-resolution inputs. To solve this, we propose QuADA-GS. After encoding inputs into a latent space, a Neural Routing Architecture evaluates local complexity to distribute a global budget, assigning specific upsampling factors to features to avoid redundant processing. Features are dynamically densified based on these factors, forming an irregular topology decoded into 2D Gaussian primitives. To coordinate features before decoding, we introduce Hierarchical Pointer Convolution. This non-grid operator achieves O(1) neighbor lookup complexity, facilitating efficient spatial communication and bypassing dense bottlenecks. Experiments show QuADA-GS achieves state-of-the-art ASR performance, maintaining low latency and a lean memory footprint.

- **核心问题**：如何在任意连续缩放因子下，高效且高质量地完成单幅图像的超分辨率（Arbitrary‑Scale Super‑Resolution, ASR）。现有主流方法存在两大痛点：
  - **刚性上采样**：传统模型常固定预测某一整数倍（如×4），再通过插值适应非整数尺度，极大损害重建质量。

- **研究动机**：计算机图形学、VR/AR、自适应显示等场景要求对图像进行任意、动态缩放，而高斯泼溅（Gaussian Splatting, GS）虽具备自适应稠密化的潜力，却无法直接用于前馈推理（因为稠密化依赖高分辨率梯度，推理时不可得）。因此，作者旨在设计一种能从低分辨率输入自主预测高斯分布的前馈式 ASR 模型，实现“按需分配”的稀疏计算和高质量重建。

### 2. 论文提出的方法论

- **核心思想**：**QuADA‑GS** 将高斯泼溅转化为可学习的、显式自适应分配的前馈模型。通过神经路由评估局部重构复杂度，动态决定每个低分辨率（LR）像素应分裂成多少个子特征（即高斯基元），并用指针卷积在非规则四叉树拓扑上高效实现邻域通信，最后解码为2D高斯并渲染。
- **关键技术细节**：
  - **特征提取与尺度感知精炼**（Sec. 3.1）：从LR输入提取特征图 \(F\)，通过可复制的高斯嵌入 \(E_{gs}\) 与尺度嵌入 \(e_s\)，经窗口交叉注意力和自注意力，获得精炼的潜在特征 \(E_{gs}^{(refined)}\)。
  - **结构引导的自适应高斯路由**（Sec. 3.2）：
    - **可微分神经路由架构**（图1）：路由器 \(R_\theta\) 为每个潜在像素输出三种上采样等级（\(\{L_0,L_1,L_2\}\)，对应分裂为4、16、64个子特征）的离散概率；通过 Gumbel‑Softmax 和直通估计器生成离散路由图 \(M_{route}\)，保持梯度。
    - **预算约束**：利用HR真值图像的结构张量 \(S\) 及其特征值，生成本地复杂度分数 \(S_{map}\)，再下采样并量化得到全局目标预算 \(B_{target}\)。对比路由器预测的总成本 \(C_{router}\)，施加带容忍边缘的预算匹配损失 \(\mathcal{L}_{budget}\)，防止过度分配或分配不足。
    - **路由质量损失** \(\mathcal{L}_{rout}\)：取脱离渲染误差图，惩罚路由器在误差高处不使用精细等级，促使等级分配与重构误差一致。
  - **层次化指针卷积（HPC）**（Sec. 3.3）：
    - **级联分裂**：先密集上采样至 \(L_0\)，再根据路由图选择性对 \(L_{target}\ge 1\) 和 \(L_{target}=2\) 进行1→4分裂，形成层次化四叉树拓扑。
    - **指针网格（Pointer Grid）**：维护一张最大分辨率虚拟网格 \(P\)，记录每个单元格所属的特征索引。通过特征的中心坐标自适应地在网格中投影出足迹（分辨率随层级变化）。
    - \(O(1)\) **邻域查找**：利用足迹边界自适应偏移采样（图4），直接一次访存检索到拓扑邻居，从而避免密集稀疏卷积，将邻居特征拼接后经残差卷积更新。
  - **2D高斯预测与光栅化**（Sec. 3.4）：使用共享MLP预测不透明度、颜色、协方差相关参数及偏移。通过层级感知的缩放调制 \(m_{l_i}\) 稳定训练。用alpha‑blending在连续归一化空间渲染任意分辨率，总损失为渲染损失、预算损失和路由质量损失的加权和。
