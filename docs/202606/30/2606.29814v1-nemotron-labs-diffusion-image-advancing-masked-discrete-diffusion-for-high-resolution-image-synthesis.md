## 摘要
我们提出Nemotron-Labs-Diffusion-Image，这是一种用于高分辨率文本到图像合成的先进掩码离散扩散模型（MDM）。与先前掩码图像生成工作相比，Nemotron-Labs-Diffusion-Image解决了两个关键挑战。首先，与逐步细化整个图像潜在表示的连续扩散模型不同，标准MDM缺乏自校正能力，因为离散标记一旦取消掩码就无法修改。其次，虽然增加离散图像分词器的词汇量可以提高重建保真度，但由于每个标记的训练信号变得越来越稀疏，这给生成建模带来了优化困难。为了解决第一个挑战，Nemotron-Labs-Diffusion-Image引入了一种标记编辑机制，使模型能够在推理过程中动态修改已取消掩码的标记，类似于雕塑家逐步完善其作品。为了应对第二个挑战，我们提出了一个分组交叉熵（GCE）目标，该目标为嵌入空间中与真实标记相邻的标记分配正学习信号，从而缓解信号稀疏性。为了进一步提高训练效率，我们为GCE实现了一个自定义融合算子，该算子在大词汇量设置下显著减少了VRAM使用。实验结果表明，这些创新显著提高了掩码离散图像生成器的训练效率和图像保真度，在GenEval上得分为0.90，在DPG上得分为86.9，在HPSv3上得分为10.76。

## Abstract
We propose Nemotron-Labs-Diffusion-Image, a state-of-the-art masked discrete diffusion model (MDM) for high-resolution text-to-image synthesis. Compared with prior work on masked image generation, Nemotron-Labs-Diffusion-Image addresses two key challenges. First, unlike continuous diffusion models which progressively refine latent representations across the entire image, standard MDMs lack self-correcting capability because discrete tokens cannot be modified once they are unmasked. Second, although increasing the vocabulary size of discrete image tokenizers improves reconstruction fidelity, it introduces optimization difficulties for generative modeling as the per-token training signal becomes increasingly sparse. To address the first challenge, Nemotron-Labs-Diffusion-Image incorporates a token-editing mechanism that enables the model to dynamically revise already-unmasked tokens during inference, similar to how a sculptor iteratively refines their work. To tackle the second challenge, we propose a Grouped Cross-Entropy (GCE) objective that assigns positive learning signals to tokens neighboring the ground truth in embedding space, thereby alleviating signal sparsity. To further improve training efficiency, we implement a custom fused operator for GCE that significantly reduces VRAM usage in large-vocabulary settings. Experimental results demonstrate that these innovations substantially improve both training efficiency and image fidelity of masked discrete image generators, achieving a score of 0.90 on GenEval, 86.9 on DPG and 10.76 of HPSv3.

### 1. 论文的核心问题与整体含义

- **研究背景**：高阶文本到图像生成长期由连续潜在扩散模型（LDM）主导，其核心优势在于推理时能逐步修正全图表征。离散图像生成近年兴起，它先将图像压缩为离散 token 序列，再用掩码离散扩散模型（MDM）或自回归模型生成，天然兼容语言模型、便于序列打包和 KV 缓存。然而现有 MDM 存在两大根本缺陷：
  1. **缺乏自校正能力**：一旦 token 被去掩码就彻底固定，即使预测错误也无法在后续步骤修改，导致误差积累。
  2. **大码本训练信号稀疏**：为提升重建保真度需要扩大码本（如 13 万+），但每 token 出现频次极低，传统 one-hot 交叉熵无法利用语义相近 token 的相似性，训练困难。

- **整体含义**：论文提出 **Nemotron‑Labs‑Diffusion‑Image**，通过引入 token 编辑机制和分组交叉熵（GCE）目标，在 8B 参数规模上同时解决了 MDM 的“不可自我修正”和“大词汇量监督稀疏”两大难题，实现高分辨率（1024px）文本到图像合成的先进水平。

### 2. 论文提出的方法论

#### 2.1 模型架构与训练框架

- 采用**单一 decoder‑only Transformer**同时处理文本提示和图像 token，初始权重来自预训练扩散语言模型（已习得文本理解能力）。
- 与 Meissonic 等编码器‑解码器结构相比，纯解码器架构不受冻结文本编码器的上下文长度限制，可天然利用序列打包优化，训练更高效。
- 模型总参数量 **8B**，使用 131K 码本的 image tokenizer。

#### 2.2 自校正：Token 编辑机制

- **训练阶段**：修改前向扩散过程，不仅随机掩码一部分 token，还对未被掩码的“干净 token”进行腐蚀（corruption）。
  - 腐蚀策略：①用同一图像内的其他 token 替换；②用嵌入空间中最近邻 token 替换（利用预计算 K‑means 簇）。避免纯随机 token 因分布不匹配带来的训练‑推理鸿沟。
  - 在损失函数中，**对所有 token 位置**（而非仅掩码位置）计算交叉熵：
    \[
    \mathcal{L}_{\text{ELBO}}^{\text{Edit}} = \mathbb{E}_{y_0,t,y_t\sim q'}\bigg[-\frac{1}{t}\sum_{i=1}^{L}\log p_\theta(y_0^i|y_t)\bigg]
    \]
    这使得模型学习“该位置当前 token 是否为正确 token，若不是应修正为何”。

- **推理阶段**：在每一扩散步，先执行标准去掩码，再对已去掩码 token 计算预测概率，若模型对某替代 token 的信心超过阈值 \(\tau\)（论文选 0.6），则用新高信心 token 替换原 token，实现迭代精修。

#### 2.3 缓解码本稀疏：分组交叉熵（GCE）

- 对大规模码本（如 131K）预先执行 K‑means 聚类（如 16,384 和 8,192 两组），将语义相似 token 分到同一簇。
- 训练时在标准 token 级 one‑hot 交叉熵基础上，**额外对簇标签施加交叉熵损失**，簇概率由簇内 token 概率求和得到：
  \[
  J_{\text{GCE}}(y_0^i, y_t) = \log p_\theta(y_0^i|y_t) + \sum_{j}\log P\big(C_j(y_0^i)|y_t\big)
  \]
  - 该设计使真实 token 获得来自所有层级（top‑1、细簇、粗簇）的正梯度，而其近邻 token 依据所处簇的层级获得部分正信号，越远 token 正信号越弱，从而引导模型捕捉嵌入空间的几何语义结构，而非将所有非 ground‑truth 等同视作负类。
- 对比 SNCE（距离平滑标签）：GCE 不需要存储全词汇大小的软标签，目标可完全达到零损失，且梯度分配允许模型动态集中概率质量，缓解训练不稳定。

#### 2.4 GCE 高效算子实现

- 朴素 PyTorch 实现需创建 \(N\times|V|\) 的张量并做无用掩码，显存开销巨大。
- 自定义前向/反向算子：仅分配 \(N\times |C_{\max}|\) 的缓冲区（最大簇规模远小于 |V|），前向利用 torch.logsumexp 高效计算对数归一化项；反向手动推导梯度并利用 scatter_add 原地操作，将内存开销降至与标准 CE 相当，延迟增加仅约 8ms（单步训练约 3.2s），几乎无负担。

- 提出的 **token 编辑机制**首次在图像 MDM 中实现迭代自我修正，有效减少采样误差积累，提升图像细节和真实感，且可在较低 NFE 下达到较高 NFE 的质量。

- 综合方案在 **GenEval 0.90、DPG 86.9、HPSv3 10.76** 等指标上达到同期先进水平，且推理速度远超同类自回归模型，具备少步生成能力。
- 架构上，纯解码器 Transformer 直接从预训练扩散语言模型启动，可高效适应图像生成，并为未来统一多模态提供范本。

### 7. 优点

- **原创性**：首次将 token 编辑思想适配到图像 MDM 中，设计了适合固定长度图像序列的腐蚀策略。
- **有效缓解码本难题**：GCE 利用几何近邻关系提供结构化正监督，思路清晰，比 soft label 方法更稳定且零额外内存。
- **系统优化务实**：提供自定义融合算子，将学术 idea 转化为可大规模训练的实现，显存和延迟几乎无负担。
- **全面且深入的实验**：多数据集、多指标、多基线对比，消融细致，兼顾生成质量、速度与训练成本。

### 8. 不足与局限

- **残余伪影**：token 编辑虽提升整体质量，但仍无法完全消除所有瑕疵，极限质量可能仍落后于最顶尖的连续扩散模型。
- **算子优化未到极致**：当前 GCE 融合算子仍在 PyTorch 层面定制，未使用更低级的 CUDA kernel，理论上仍有进一步压缩延迟的潜力。
- **偏见与安全**：模型训练数据源自互联网大规模图文对，可能继承文本和图像中的偏见，论文明确指出仅作研究用途，不推荐直接用于生产。
- **评估局限性**：GenEval、DPG 等主要衡量文本‑图像对齐，FID 在大分辨率下参考价值有限；HPSv3 虽更可靠但仍是自动化评分，缺乏大规模人类主观评测。
- **计算成本**：8B 模型仍需 64 块 H100 训练 16 天，门槛较高，不过相比其他 10B+ 模型已算高效。
