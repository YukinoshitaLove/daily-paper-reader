## 摘要
线性循环单元（LRU）基于稳定线性循环的原则化公式设计，在长程依赖任务上已展现出良好的准确性和鲁棒性。然而，其静态参数化和单次扫描方法限制了其在二维视觉任务中的适用性。在本研究中，我们提出了一种基于LRU的恢复网络，并带有语义调制单元（SMU），以实现单图像超分辨率中性能与效率的和谐平衡。SMU发挥三个关键作用：LRU调制、空间分类以及通过学习的原型进行特征增强。大量实验表明，我们的方法在定量和定性上超越了最近的最先进方法。值得注意的是，我们的方法在计算复杂度与现有方法相当的情况下实现了优越的性能。源代码和模型可在 https://github.com/MingyuChoi-run/LSM 获取。

## Abstract
Linear recurrent unit (LRU), designed with a principled formulation for stable linear recurrence, has demonstrated promising accuracy and robustness on long-range dependency tasks. However, its static parameterization and single-scan method limits its applicability to 2D vision tasks. In this study, we propose a LRU-based restoration network with a semantic modulating unit (SMU) to achieve a harmonious balance between performance and efficiency in single-image super-resolution. The SMU plays three key roles: LRU modulation, spatial categorization, and feature enhancement through learned prototype. Extensive experiments demonstrate that our method quantitatively and qualitatively surpasses recent state-of-the-art methods. Notably, our approach achieves superior performance with computational complexity on par with existing methods. The source code and models are available at https://github.com/MingyuChoi-run/LSM

- **核心问题**：线性循环单元（LRU）虽在长序列建模中表现优异，但其**静态参数化**与**单次扫描**机制难以直接适应图像超分辨率任务。
- **研究动机**：
  - 现有 Transformer/Mamba 等方法虽性能好，但动态参数化与多扫描机制带来较高计算复杂度和解释性困难。
  - LRU 具备线性复杂度与长程建模能力，但其固定递归行为缺乏对空间变化模式的适应能力，限制了在2D图像复原中的表达力。
- **目标**：在保持 LRU 高效稳定特性的前提下，赋予其像素级自适应能力，实现性能与效率的平衡。

## 二、方法论

### 2.1 整体思路
提出 **LSM（Linear recurrent unit with Semantic Modulation）**，通过**语义调制单元（SMU）** 对 LRU 的转移矩阵进行输入依赖的动态调制，并结合语义分类与全局特征增强，使静态单扫描 LRU 适用于图像超分辨率。

### 2.2 基础 LRU 公式
论文采用的 LRU 更新公式为：  
\[
\bar{h}_k = \text{diag}(\lambda)\bar{h}_{k-1} + \gamma \odot (\bar{B}u_k), \quad \bar{y}_k = \bar{C}\bar{h}_k + Du_k
\]  
其中 \(\lambda\)、\(\bar{B}\)、\(\bar{C}\) 均为静态参数，\(\gamma\) 为归一化因子。

### 2.3 语义调制单元 (SMU)
SMU 通过一个可学习的**字典 \(D\)** 实现三大功能：

1. **LRU 调制**：  
   对输入特征 \(U\) 与字典计算余弦相似度后，经 softmax 分块生成调制令牌 \(M_k = \{M^\lambda_k, M^B_k, M^{C_{Re}}_k, M^{C_{Im}}_k\}\)，动态调制 LRU 的隐藏状态更新与输出：
   \[
   \bar{h}_k = (\lambda \odot M^\lambda_k) \odot \bar{h}_{k-1} + \gamma \odot (\bar{B}u_k) \odot M^B_k, \quad \bar{y}_k = \bar{C}\bar{h}_k \odot M^{C}_k + \bar{D}u_k
   \]

2. **语义分类**：  
   利用 Gumbel-softmax + argmax 对像素进行硬分类，将空间上远离但语义相似的像素归入同一序列，缓解单次扫描的上下文局限。

3. **特征增强**：  
   通过 softmax 相似度对字典值矩阵 \(V_D\) 做交叉注意力，获得全局特征 \(Y_{enhance}\)，与 LRU 输出拼接。

### 2.4 网络架构
- **块结构**：局部窗口多头自注意力（WMS）块 + 类别调制 LRU（CML）块，各块带门控 MLP 与可学习残差权重。
- **堆叠方式**：重复 n_group 个层次组，最后用 pixel-shuffle 上采样重建高分辨率图像。

## 六、主要结论与发现

1. LSM 是首个成功将 LRU 用于图像超分辨率的模型，同时引入语义调制突破静态扫描限制。
2. SMU 的**三层角色（调制、分类、增强）协同作用**，显著提升 SR 质量，尤其在需要长程依赖的高倍数放大（×4）上优势更明显。
3. 合理的 λ 初始化（较大幅值与相位范围）对 LRU 视觉性能有重要影响，与长序列 NLP 任务的经验存在差异。
4. 在相近参数量和 FLOPs 下，LSM 优于主流 Transformer 和 Mamba 方法，且 GPU 内存占用更低，为高效 SR 提供新范式。
