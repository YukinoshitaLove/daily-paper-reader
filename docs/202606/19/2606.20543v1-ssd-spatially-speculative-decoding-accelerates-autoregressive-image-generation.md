## 摘要
自回归模型通过将图像视为离散标记的一维序列来进行视觉生成，类似于语言建模。然而，这种扁平化丢弃了视觉信号固有的二维空间局部性，在推理过程中造成了严重的计算瓶颈。我们提出了空间推测解码（SSD），这是一个使预测目标与图像的自然几何结构对齐的框架。我们的模型不仅预测一维序列中紧接的下一个标记，还同时预测相邻的水平标记和其正下方的标记。通过利用这种二维空间相关性，空间推测解码克服了视觉推理中的内存墙。我们的方法将自回归图像生成速度提升了高达13.3倍，同时在DPG-Bench和GenEval上保持高保真度。我们的结果表明，尊重视觉的底层几何结构释放了巨大的计算效率，为实时、高分辨率自回归生成模型铺平了道路。

## Abstract
Autoregressive models excel in visual generation by treating images as 1D sequences of discrete tokens, mirroring language modeling. However, this flattening discards the intrinsic 2D spatial locality of visual signals, creating severe computational bottlenecks during inference. We introduce Spatially Speculative Decoding (SSD), a framework that aligns the predictive objective with the natural geometry of images. Rather than predicting only the immediate next token in a 1D sequence, our model simultaneously predicts the adjacent horizontal token and the token directly below it. By capitalizing on this 2D spatial correlation, spatially speculative decoding overcomes the memory wall in visual inference. Our approach accelerates autoregressive image generation by up to 13.3x while maintaining high fidelity on DPG-Bench and GenEval. Our results suggest that respecting the underlying geometry of vision unlocks massive computational efficiencies, paving the way for real-time, high-resolution autoregressive generative models.

### 1. 论文核心问题与整体含义

- **核心问题**：现代自回归视觉生成模型将图像的二维网格扁平化为一维离散标记序列，模仿语言建模进行“下一个标记预测”。这种处理丢弃了图像天然的二维空间局部性，导致推理时必须逐 token 生成整个序列，形成严重的计算瓶颈（O(n²) 次前向传递，n 为网格边长），被称为“内存墙”。
- **整体含义**：作者指出，直接移植自然语言处理领域的推测解码（如多标记预测、Jacobi 解码）到图像生成中，因仍然受限于一维序列假设，仅能获得约 2–4 倍的有限加速。该工作试图从根本上放弃一维序列假设，将预测目标与图像的二维几何对齐，从而突破推理效率瓶颈，为实时、高分辨率自回归生成模型打开空间。

### 2. 方法论

- **核心思想**：将自回归生成的预测分解为两个正交的一维方向——水平（同一行内下一标记）和垂直（正下方标记）。利用二维空间相关性，在完成当前行后，一次性并行预测后续整行的 token，从而将每行生成成本从 O(n) 降至 O(1)，整体复杂度从 O(n²) 降至 O(n)。
- **空间推测解码（SSD）流程**：
  - **水平草稿**：使用少量水平预测头（如 +1, +2, +3 偏移）在当前行内顺序预测若干标记，直至补全当前行。
  - **垂直草稿**：一旦某行完成，使用垂直预测头（对应 +1 行、+2 行等偏移）同时预测该行下方整行的所有标记。由于列间独立，这些预测可一次性并行完成。
  - **验证与自动校正**：将草稿块送入冻结的主干模型进行并行验证。对每个位置独立应用标准推测接受规则（公式4）。被拒绝的标记并非全部丢弃，而是利用同一前向传递从残差分布（公式5）中采样修复。可重复多轮验证（r 轮），最终仅需 r+1 次前向传递即可完成一个草稿块的验证提交。
- **关键技术细节**：
  - **在连续隐空间中进行草稿预测**：预测目标选为最后一层 Transformer 的 RMSNorm 之前的隐藏状态（h_pre），而非离散标记。输入为当前位置的隐藏状态与标记嵌入的拼接，输出为偏移位置的隐藏状态预测（公式2）。训练损失为平滑 L1 损失（公式3），使用自蒸馏方式从冻结的主干模型中提取目标。
  - **轻量级预测头结构**：每个偏移和方向拥有一个独立的预测头，由线性投影 + SwiGLU 前馈网络组成，参数量极小（每个头约 134M，而主干模型有 7–8B）。多个头的草稿开销低于 0.1 个 AR 步骤（图3）。
  - **多行验证调度**：针对多行垂直草稿，采用分阶段验证：先对前导行做若干轮基础验证并提交，再以修正后的前导行为上下文对尾随行做增量验证，避免多行间相关性导致的不稳定。

### 6. 主要结论与发现

- **显著加速**：SSD 在三个模型上分别实现了 5.74×（Janus-Pro）、12.19×（Lumina-mGPT）、13.28×（Emu3）的壁钟时间加速，步数加速比更高（最高 16.74×），且生成质量与 AR 基线相当或轻微下降，远优于 1D-MTP。
- **二维空间依赖性**：实验证实，垂直方向上的预测准确率与相同空间距离的水平预测相当，说明二维局部性是被一维序列假设掩盖的本质属性。
- **连续潜空间预测优于离散标记**：直接在隐藏状态空间训练草稿头比预测标记稳定得多，能提供更强的监督信号，避免离散分布过平带来的低接受率。
- **自动校正验证有效**：通过“验证即自动校正”策略，在单次前向传递中同时完成接受和修复，避免了整块回滚，使得基于空间块的草稿高效利用。
- **加速效果随网格尺寸增大而更明显**：更大的网格意味着更多行可被并行草稿和验证，加速优势会进一步放大。

### 7. 优点

- **即插即用**：无需修改或重新训练已有的大型主干模型，仅添加轻量预测头，可随时移除。
- **几何对齐**：从根源上解决一维序列假设与二维图像本质的矛盾，将推测解码从一维扩展到二维，理论复杂度从 O(n²) 降至 O(n)。
- **连续空间预测**：避开离散标记预测的困难，使草稿头训练稳定且高效，接受率高。
- **自动校正机制**：巧妙利用并行验证的隐藏状态输出进行修补，避免严格回滚带来的浪费，极大提升了垂直草稿的实用性。
- **实验全面扎实**：覆盖多个不同规模和架构的模型，消融实验充分，可视化清晰，对加速比计算进行了仔细的公平性处理。

### 8. 不足与局限

- **训练数据量和分布**：目前草稿头只在 Midjourney 提示分布的小规模数据（5K–60K）上自蒸馏训练，数据量仍有提升空间（表7显示未饱和），可能限制在多样化、复杂提示下的泛化能力。
- **质量-速度权衡**：虽然能在高加速比下保持不错的质量，但未验证轮数较低时（r=0 或 r=1）质量明显下降（表5）；达到 AR 级质量通常需要 r≥2，此时加速比会有所降低。
- **对更大网格和更新模型的拓展性未充分探索**：实验集中在 7B/8B 量级模型和最高 90×90 网格，对于更细粒度 token 化或更大参数模型，并行草稿和验证的显存开销可能增加，文中未深入讨论。
- **未结合文本条件动态调整**：SSD 未考虑提示内容对空间预测置信度的影响，所有行的草稿轮数和深度采用统一超参数，可能仍有优化空间。
- **适用范围限制**：仅适用于将图像表示为二维 token 网格并进行自回归生成的模型，对扩散模型或其他范式不适用。
