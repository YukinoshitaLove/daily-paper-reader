## 摘要
扩散变换器（DiT）已被广泛采用于生成式扩散领域，通过注意力机制和前馈（\text{FFN}）层推动查询令牌的去噪。FFN实际上充当了解码视觉内容的键值词汇表，其中值嵌入了视觉语义知识。我们发现，关注对应于更复杂细节的关键查询令牌，并鼓励模型改进这些令牌，对于细粒度视觉生成至关重要。为此，我们提出FocusDiT，它应用一种掩码方案来专注于关键查询令牌，这些令牌仅被馈送到FFN。掩码查询可以从FFN词汇表中检索视觉令牌，并用它们来解码其视觉细节。大量的文本到图像实验验证了令牌掩码在增强生成性能方面的有效性。

## Abstract
Diffusion transformer (DiT) has been widely adopted in the generative diffusion field, advancing the denoising of query tokens through attention and Feed-Forward (\text{FFN}) layers. FFN actually acts as the key-value vocabulary for decoding visual contents where the value embeds the visual semantical knowledge. We present that focusing on critical query tokens corresponding to more complex details and encouraging the model to improve these tokens is essential for fine-grained visual generation. To this end, we propose FocusDiT, which applies a Masking scheme to focus on critical query tokens that are exclusively fed into FFN. The masked queries can retrieve visual tokens from the FFN vocabularies, and use them to decode their visual details. Extensive text-to-image experiments validate the effectiveness of token masking in enhancing generative performance.

## 1. 核心问题与整体含义（研究动机和背景）
- **研究背景**：扩散变换器（DiT）已成为视觉生成领域的主流方法，其前馈网络（FFN）实际上充当了存储视觉语义知识的键值词汇表，查询令牌通过FFN访问这些视觉令牌进行内容解码。
- **核心问题**：现有DiT对所有查询令牌一视同仁，导致对应复杂细节的关键令牌（如前景物体、精细纹理）被大量对应背景或低频区域的非关键令牌淹没，无法充分利用FFN词汇表中的视觉知识。同时，不同层的FFN词汇表利用不充分，存在大量未激活项。
- **研究动机**：提出一种机制，使关键查询令牌能够优先利用FFN词汇表，从而提升细粒度视觉生成质量。

## 2. 方法论：核心思想与关键技术细节
### 2.1 核心思想
- **查询令牌掩码（Query Masking）**：设计一个掩码预测网络（Q-MaskGen），动态辨识关键查询令牌（对应复杂细节），仅将这些关键令牌的FFN更新输出进行加权保留，非关键令牌的FFN贡献被抑制（掩码值接近0）。
- **词汇重分布（Vocabulary Redistribution, VR）**：调整不同FFN层的词汇容量，将浅层和深层的多余容量重新分配给中层，使词汇分配与各层解码需求对齐。

### 2.2 关键技术细节
- **Q-MaskGen网络**：输入包括当前块的交叉注意力输出 \(Z^{in}_t\)、上一块的掩码 \(M^{i-1}_Q\)、时间步 \(t\)，通过多层感知机（MLP）预测当前块的掩码 \(M^i_Q\)。时间步信息使用AdaLN-single编码，使得掩码随去噪进程自适应（高噪声时区分度低，低噪声时突出关键令牌）。
- **掩码应用公式**：最终输出 \(Z^{out}_t = Z^{in}_t + M_Q \cdot \text{FFN}(Z^{in}_t)\)，其中 \(M_Q\) 逐元素作用于FFN输出。反向传播时，梯度也乘以 \(M_Q\)，从而只更新对应关键令牌的词汇权重。
- **词汇重分布公式**：设总FFN层数为 \(L\)，基础中间维度为 \(M\)，第 \(i\) 层FFN的中间维度设计为：当 \(i \le L/2\) 时，维度为 \(P + (i-1)\Delta\)；当 \(i > L/2\) 时，维度为 \(P + (L-i)\Delta\)。参数 \(P=3M\)，\(\Delta = \lfloor 2/(L/2-1) \rfloor M\)，保证总参数量不变，形成中间大、两端小的纺锤形分布。

- FocusDiT通过查询令牌掩码显著提升了细粒度图像生成质量，尤其在头发纹理、建筑结构、抽象概念（如“穿柔道服的猫”）等复杂细节上优于PixArt-α和SD3。
