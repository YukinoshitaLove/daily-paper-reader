## 摘要
真实图像编辑能够精确操控视觉内容，但现有方法在复杂的多目标场景中常常失败，导致语义混合、目标重复或编辑不完全。我们将这些失败归因于注意力泄漏，即在去噪过程中，跨空间区域和文本 token 的信号相互纠缠。具体来说，我们识别出两种不同形式的泄漏：编辑 token 泄漏，模糊的 token-区域对齐导致目标混合；以及源主导泄漏，未改变的源目标的 token 压倒了用于目标实体的注意力。为解决这些泄漏，我们提出了 BindEdit，它在单个扩散轨迹内强制施加注意力级别的约束。为抑制编辑 token 泄漏，BindEdit 联合正则化交叉注意力和自注意力，使得每个目标 token 组与其对应的空间区域绑定，同时保持实例级分离。为抑制源主导泄漏，交叉注意力再平衡机制放大目标 token 的影响并衰减可编辑区域内的残余源语义。此外，区域保真度项确保每个目标概念在整个编辑掩码中连贯表达。我们还提出了一个全面的多目标基准，涵盖多样化的目标数量和类别。大量实验表明，BindEdit 在单个扩散轨迹中始终优于现有方法，在单目标和多目标编辑场景中均保持稳健性能。

## Abstract
Real image editing enables precise manipulation of visual content, yet existing methods often fail in complex multi-object scenarios, causing semantic blending, object duplication, or incomplete edits. We attribute these failures to attention leakage, where signals across spatial regions and text tokens become entangled during the denoising process. Specifically, we identify two distinct forms of leakage: Edit-Token Leakage, where ambiguous token-region alignment leads to object blending, and Source Dominance Leakage, where tokens of unchanged source objects overwhelm the attention intended for target entities. To resolve these leakages, we propose \textbf{BindEdit}, which enforces attention-level constraints within a single diffusion trajectory. To suppress Edit-Token Leakage, BindEdit jointly regularizes cross- and self-attention so that each target token group is bound to its corresponding spatial region while maintaining instance-level separation. To suppress Source Dominance Leakage, a cross-attention re-balancing mechanism amplifies target token influence and attenuates residual source semantics within editable regions. Moreover, a region fidelity term ensures that each target concept is expressed coherently across the entire editing mask. Additionally, we propose a comprehensive multi-object benchmark encompassing diverse object counts and categories. Extensive experiments demonstrate that BindEdit consistently outperforms existing methods within a single diffusion trajectory, maintaining robust performance across both single- and multi-object editing scenarios.

## 1. 核心问题与研究动机
- **研究背景**：基于扩散模型的文本引导图像编辑在单目标场景取得显著进展，但真实图像常包含多个需要同时编辑的独立实体。现有方法在多目标编辑时频繁出现语义混合、目标重复或编辑不完整。
- **核心问题**：论文将多目标编辑的失败归因于**注意力泄漏**，具体表现为两种形式：
  - **编辑令牌泄漏**：不同目标 token 的注意力信号相互纠缠，导致物体身份混合（例如“两只鸟”被编辑为“一只泰迪熊和一只猫”时，两者特征融合为一个对象）。
  - **源主导泄漏**：当源图像中存在多个同类别物体且仅编辑其中一部分时，未编辑物体对应的源 token 保持强烈的注意力绑定并溢出到可编辑区域，压制新目标 token 的表达（例如保留犬类特征而非目标对象）。
- **研究目标**：在**单一扩散轨迹**内，通过注意力级约束同时解决两类泄漏，实现精确、连贯的多目标编辑。

## 2. 方法论
- **总体框架**：BindEdit 是一个**免训练**方法，基于编辑友好的 DDPM 反演获取初始噪声潜变量，在去噪过程中直接对潜变量施加梯度更新，不修改预训练扩散模型参数。
- **关键组件**：

  **a) 注意力绑定引导（L_bind）——抑制编辑令牌泄漏**
  - 联合正则化**交叉注意力**和**自注意力**。
  - 对每个编辑目标，定义注意力集中比率 \( S(o) = \frac{W_{in}}{W_{in}+W_{out}+\epsilon} \)。
  - **交叉注意力绑定**：要求目标 token 组的注意力集中在对应语义掩码内（允许同类别间激活，以保持自然上下文）。
  - **自注意力绑定**：要求同一实例的空间 tokens 主要相互交互，强化实例级分离。
  - **联合优化**：采用乘积形式 \( L_{\text{bind}} = \frac{1}{|O|}\sum_{o}(1 - S_{\text{cross}}(o) \cdot S_{\text{self}}(o)) \)，确保语义绑定与实例隔离同时满足。
  - **自适应强度**：引入时间步依赖的缩放因子 \( \lambda_t = 1 - \frac{\text{区域内注意力质量}}{\text{总注意力质量}} \)，在早期强绑定，后期自然减弱。

  **b) 源主导抑制（L_supp）——解决源主导泄漏**
  - 在**交叉注意力**的可编辑区域内，使用**对比二元交叉熵损失**重塑注意力分布。
  - 正向项 \( L^+_{BCE} \)：放大目标 token 组的概率。
  - 负向项 \( L^-_{BCE} \)：惩罚其他所有 token（含源 token 和背景），并通过系数 \( \gamma > 1 \) 放大负向信号（默认 \( \gamma = 10 \)）以有效抑制残余源语义。
  - 对每个目标独立计算后取平均。

  **c) 区域保真度正则化（L_fidelity）——连贯表达**
  - 将编辑掩码内的归一化注意力分布 \( p_o \) 拉近以掩码质心为中心的截断高斯先验 \( q_o \)，采用 KL 散度损失。
  - 作用：防止目标注意力分散为多个峰值，确保每个掩码内生成单一、完整的对象实例。
  - 可选增强：排除非全局最大值，进一步锐化焦点。

- **整体优化目标**：
  \[
  L_{\text{edit}} = \lambda_{\text{bind}} L^{(t)}_{\text{bind}} + \lambda_{\text{supp}} L_{\text{supp}} + \lambda_{\text{fidelity}} L_{\text{fidelity}}
  \]
  平衡系数默认 \( \lambda_{\text{bind}}=20, \lambda_{\text{supp}}=10, \lambda_{\text{fidelity}}=2 \)。
  采样时执行 \( z_t \leftarrow z_t - \eta \nabla_{z_t} L_{\text{edit}} \)，仅在选定时间步更新。
