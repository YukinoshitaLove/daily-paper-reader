## 摘要
扩散模型在拖拽式编辑中展现出潜力。以往的工作主要关注基于点的拖拽，这本质上具有歧义性。本文关注基于区域的拖拽，并引入了一种新颖的基于上下文的区域拖拽（ICRDrag）方法。在上下文学习框架下，ICRDrag 接收源图像、源区域掩码和目标区域掩码，生成目标拖拽图像。在基本的上下文学习模型基础上，我们引入了两种新颖的注意力正则化：1）图像-掩码注意力一致性，以确保目标区域在图像和掩码模态中关注相似的源区域；2）源-目标注意力对应性，以确保源区域和目标区域之间的相互对应。为了促进基于区域的拖拽，我们还构建了配对区域数据集（PRD），这是一个带有配对掩码和图像的大规模数据集。大量实验表明，ICRDrag 在定量指标和用户研究中均显著优于现有方法，实现了卓越的编辑精度和视觉保真度。数据集、代码和模型可在 https://github.com/bcmi/ICRDrag-Region-Drag-Editing 获取。

## Abstract
Diffusion models have shown promise in drag-style editing. Previous works mainly focus on point-based drag, which is inherently ambiguous. This paper focuses on region-based drag and introduces a novel In-Context Region-based Drag (ICRDrag) method. Under the in-context learning framework, ICRDrag consumes a source image, a source region mask, and a target region mask, producing the target dragged image. Built upon the basic in-context learning model, we introduce two novel attention regularization: 1) image-mask attention consistency to ensure that a target region attends to similar source regions for image and mask modalities; 2) source-target attention correspondence to ensure the mutual correspondence between source and target regions. To facilitate region-based drag, we also construct Paired Region Dataset (PRD), a large-scale dataset with paired masks and images. Extensive experiments show that ICRDrag significantly outperforms existing methods in both quantitative metrics and user studies, achieving superior editing accuracy and visual fidelity. The dataset, code, and model are available at https://github.com/bcmi/ICRDrag-Region-Drag-Editing.

- **整体含义**：论文探索了如何将结构化的掩码信息深度融入到扩散模型的生成过程中，以替代模糊的稀疏点控制，从而实现更准确、更高保真的图像编辑。

### 2. 论文提出的方法论

论文提出了一种新的上下文学习框架 **ICRDrag (In-context Region-based Drag)**，核心思想与关键技术如下：

- **整体架构**：
    - 基于 DiT（Diffusion Transformer）架构构建，具体依托于 Next-DiT 模型。
    - 采用 Flow Matching 范式，接收包含**源图像、源掩码**和**目标掩码**的上下文，预测速度场以生成目标图像。
    - **模态特定 LoRAs**：在 Transformer 的前馈网络中引入独立的低秩适应（LoRA）模块，分别处理图像特征（纹理、细节）和掩码特征（稀疏空间结构），避免不同模态间的特征混淆。

- **注意力正则化机制**：
    - **图像-掩码注意力一致性（IMAC，Image-Mask Attention Consistency）**：
        - **假设**：目标区域在图像模态上应关注的源区域，应当与掩码模态上关注的源区域一致。
        - **实现**：提取目标图像局部块在源图像上的注意力图，以及对应的目标掩码局部块在源掩码上的注意力图，通过最小化两者的均方误差（MSE）来强制对齐，确保视觉生成过程锚定在掩码定义的空间结构上。
    - **源-目标注意力对应性（STAC，Source-Target Attention Correspondence）**：
        - **假设**：源图像和目标图像的相关区域应相互关注，建立双向对应关系。
        - **实现**：构建目标到源的注意力矩阵与源到目标的注意力矩阵，通过最大化这两个矩阵乘积的迹（Trace），鼓励模型建立互惠的对应关系，从而增强细节保持能力。

- **两阶段课程训练策略**：
    - **第一阶段（简单任务）**：使用**完整区域掩码**（所有物体语义区均被标注）进行预训练，让模型建立掩码与图像关系的基础先验。

### 7. 优点

- **解决本质歧义**：用密集区域掩码替代稀疏点对，从根本上消除了拖拽编辑中的多义性，是任务定义上的进步。
- **精巧的注意力设计**：IMAC 和 STAC 正则化机制具有强可解释性，通过约束注意力分布巧妙地将掩码结构信号与图像外观信号绑定。
- **高质量数据基础**：构建并开源了首个大规模区域级拖拽数据集 PRD 及配套基准 PRDBench，为社区填补了数据空白，推动了该方向的标准化评估。

### 8. 不足与局限

- **推理延迟较高**：单张图片推理需 35 秒，相比一些点拖拽或简单特征粘贴方法（如 RegionDrag 的 0.85秒）在实时交互体验上存在劣势。
- **掩码质量敏感**：ICRDrag 严重依赖用户提供的掩码质量。当源与目标掩码语义不一致或不精确时，模型容易产生显著伪影。
- **领域迁移泛化有限**：由于训练数据多基于真实世界视频，模型在处理严重偏离真实感的领域（如动画、艺术插画）时，非编辑区域可能出现不可控的畸变。
- **对复杂语义理解不足**：虽然解决了空间歧义，但无法理解隐含的深层物理规则（如重物落地形变），纯粹的掩码对应可能导致在极端拓扑变化下生成违反物理常识的结果。
