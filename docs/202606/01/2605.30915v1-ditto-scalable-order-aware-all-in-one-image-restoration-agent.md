## 摘要
现实世界的图像很少仅受单一退化影响，而去除退化的顺序会显著影响最终恢复质量，这推动了基于代理的图像恢复（IR）方法的发展，其中视觉语言模型调度一组预构建的恢复专家。然而，现有的基于训练的代理每张图像需要$\mathcal{O}((N^{\mathbf{D}})^{2})$次恢复专家调用来构建最优恢复动作轨迹数据集（ORTD），其中$N^{\mathbf{D}}$表示宇宙$\mathbf{D}$中的退化类型数量，并且将代理训练耦合到固定的恢复专家池中，导致无法扩展到新引入的恢复专家而无需完全重新训练。为了克服这些效率和可扩展性瓶颈，我们提出了\textbf{DiTTo}，一种新颖的顺序感知图像恢复代理框架，由DiTTo模拟器和DiTTo代理组成。DiTTo模拟器结合了用于单步恢复动作模拟的$\cup$S-IR和用于每动作质量预测的AiO-IQA，将ORTD构建减少为每张图像$\mathcal{O}(N^{\mathbf{D}})$次模拟器调用；DiTTo代理通过在模拟器生成的ORTD上进行SFT训练，随后进行\textbf{顺序感知恢复对齐（ORA）}，该对齐沿着独立轴对齐退化识别、恢复动作顺序和输出格式。这使得\textbf{即插即用的可扩展性}成为可能：添加新的恢复专家只需更新轻量级的ORA阶段。在具有最多五种并发退化的MiO-100评估集上，我们的DiTTo代理在现有的基于代理的IR方法中实现了最先进的多退化恢复质量。

## Abstract
Real-world images rarely suffer from a single degradation, and the order in which degradations are removed substantially affects the final restoration quality, motivating agent-based image restoration (IR), where a vision-language model schedules a pool of pre-built restoration-experts. However, existing training-based agents require $\mathcal{O}((N^{\mathbf{D}})^{2})$ restoration-expert calls per image to construct the Optimal Restoration-action Trajectory Dataset (ORTD), where $N^{\mathbf{D}}$ denotes the number of degradation types in the universe $\mathbf{D}$, and couple agent training to a fixed restoration-expert pool, preventing extension to newly introduced restoration-experts without full retraining. To overcome these efficiency and extensibility bottlenecks, we propose \textbf{DiTTo}, a novel order-aware image restoration agent framework consisting of the DiTTo Simulator and the DiTTo Agent. The DiTTo Simulator combines $\cup$S-IR for single-step restoration-action simulation and AiO-IQA for per-action quality prediction, reducing ORTD construction to $\mathcal{O}(N^{\mathbf{D}})$ simulator calls per image; the DiTTo Agent is trained by SFT on the simulator-generated ORTD, followed by \textbf{Order-aware Restoration Alignment (ORA)} that aligns degradation identification, restoration-action-ordering, and output format along independent axes. This enables \textbf{plug-and-play scalable extensibility}: adding a new restoration-expert requires updating only the lightweight ORA stage. On the MiO-100 evaluation set with up to five concurrent degradations, our DiTTo Agent achieves state-of-the-art multi-degradation restoration quality among previous agent-based IR methods.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **现实图像的多重退化**：真实场景中的图像通常同时受多种退化影响（如雾、雨、噪声、模糊等），且退化去除的**顺序**显著影响最终恢复质量——例如先去低光增强再去噪可能放大噪声，先去雾再去雨可能改变雨条纹分布。
- **现有方法的局限性**：
  - 传统单任务或全合一方法无法建模恢复顺序。
  - **基于智能体的图像恢复（Agentic IR）** 利用视觉语言模型调度预构建的恢复专家池，但存在两大瓶颈：
    - **效率瓶颈**：构建最优恢复动作轨迹数据集（ORTD）需要为每张图像枚举所有专家组合，代价为 \(\mathcal{O}((N^{\mathbf{D}})^{2})\) 次真实专家调用（\(N^{\mathbf{D}}\) 为退化类型数）。
    - **可扩展性瓶颈**：训练过程与固定专家池耦合，新增专家需完全重新训练，无法即插即用。
- **本文目标**：解决上述效率和可扩展性问题，提出一种**可扩展的顺序感知图像恢复代理框架**。

## 2. 论文提出的方法论
### 2.1 核心思想
- 提出 **DiTTo** 框架，包含两个组件：
  - **DiTTo 模拟器（DiTTo Simulator）**：用学习到的低成本模拟器代替真实专家调用，将 ORTD 构建复杂度从 \(\mathcal{O}((N^{\mathbf{D}})^{2})\) 降至 \(\mathcal{O}(N^{\mathbf{D}})\)。
  - **DiTTo 智能体（DiTTo Agent）**：两阶段训练——先在模拟器生成的 ORTD 上进行监督微调（SFT），再通过**顺序感知恢复对齐（ORA）** 在少量真实专家 ORTD 子集上优化，修正模拟器与真实专家的分布差异，并支持即插即用新增专家。

### 2.2 关键技术细节
- **DiTTo 模拟器**由两个模块构成：
  - **∪S-IR（单步退化恢复模拟器）**：基于动作条件化的特征混合机制，在频率带上进行自适应门控融合，模拟单步恢复动作的效果（移除目标退化，保留其他退化）。采用 SD3 风格流匹配目标训练。
  - **AiO-IQA（全合一恢复动作评分器）**：根据当前中间图像状态和退化轨迹，为所有候选动作预测质量分数，选出最高分对应的动作。用 ∪S-IR 生成候选下一状态，再由 IQA 指标组合（MANIQA、MUSIQ、CLIP-IQA、NIQE 等）提供监督信号，训练成对排名损失。
- **DiTTo 智能体训练**：
  - **阶段1 SFT**：在 ∪S-IR 和 AiO-IQA 构建的模拟 ORTD 上微调 VLM（Qwen3-VL-8B-Thinking + LoRA），使其具备退化感知推理、顺序恢复和结构化工具调用能力。
  - **阶段2 ORA**：基于 DPO 对分解的规划轴（退化感知推理 DP、顺序恢复 OR、工具调用格式 Tool）分别计算偏好梯度，使用少量真实专家 ORTD 子集作为 chosen 轨迹、模拟轨迹作为 rejected，对齐模拟器与真实专家的决策。
- **即插即用扩展**：新增专家时，∪S-IR、AiO-IQA 和 SFT 检查点冻结，仅需更新轻量 ORA 阶段（约 11 小时 vs. JarvisIR 的 410 小时）。

1. **状态最优**：在 MiO-100 上，DiTTo 智能体在所有无参考指标上超越之前所有基于智能体的方法（相同专家池），且 ★DiTTo 智能体扩展后进一步提升。

4. **对齐重要性**：ORA（分解轴 DPO）显著优于无对齐的 SFT 和通用 DPO，验证了退化感知、顺序恢复和工具格式三个解耦维度的互补作用。

## 7. 优点
- **高效 ORTD 构建**：利用模拟器 ∪S-IR 和评分器 AiO-IQA，大幅降低数据构建成本，可扩展到更大退化集合。
- **解耦训练与扩展**：两阶段设计将大规模训练（SFT）与轻量对齐（ORA）分离，实现专家池的即插即用。
- **精细对齐策略**：提出分解轴 DPO，避免通用 DPO 对共享令牌的偏好信号稀释，更精准地优化退化感知和顺序决策。
- **综合评估**：消融实验覆盖主要设计选择（频率带、对齐轴、评分准确率），提供深入洞察。
- **代码和数据集发布**：论文承诺公开训练好的模拟器、评分器、ORTD 数据集及部分示例，促进社区复现和扩展。

## 8. 不足与局限
- **模拟器-专家分布差距**：∪S-IR 和 AiO-IQA 虽可近似专家行为，但 ORA 仅能在有限真实子集上修正，当新专家类型或退化类型与训练分布差异较大时，对齐可能不充分。
- **依赖有参考/无参考 IQA 指标**：AiO-IQA 的监督信号来自固定 IQA 组合（如 MANIQA, MUSIQ 等），这些指标与人类感知不完全一致，可能引入偏差。
- **退化覆盖不全**：当前宇宙 \(\mathbf{D}\) 仅包含 6 种退化（噪声、低光、雾、散焦模糊、雨、雪），未包含运动模糊、JPEG 压缩伪影、膜拜等常见真实退化；新增退化类型需重新合成数据并微调 ∪S-IR，虽可扩展但成本仍存在。
- **测试数据为合成**：MiO-100 是合成数据集，缺乏真实多退化图像的验证，方法在实际应用中的泛化性能尚待检验。
- **未见 ablation 对 ORA 数据规模敏感性分析**：仅提到“小规模”真实子集，未探索性能随子集大小的变化曲线。
