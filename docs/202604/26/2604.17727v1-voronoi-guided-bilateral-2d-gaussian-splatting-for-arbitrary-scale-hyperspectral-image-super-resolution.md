## 摘要
现有的大多数高光谱图像超分辨率方法需要针对不同的尺度进行修改，因此在任意尺度重建方面的灵活性受到限制。二维高斯喷射提供了一种连续表示形式，可与任意尺度的超分辨率任务相兼容。现有方法通常依赖栅格化策略，这可能限制空间建模的灵活性。将其扩展到高光谱图像超分辨率仍具有挑战性，因为该任务需要在保持光谱保真度的同时进行自适应空间重建。本文提出了一种基于高斯喷射的高光谱图像任意尺度超分辨率框架——GaussianHSI。我们设计了一种用于空间重建的 Voronoi 引导双边二维高斯喷射方法。在预测一组用于表示输入的高斯函数后，该方法通过 Voronoi 引导的选择将每个目标像素与相关的高斯函数关联。目标像素随后通过参考感知的双边加权聚合所选高斯函数进行重建，该加权同时考虑几何相关性和与低分辨率特征的一致性。我们进一步引入了光谱细节增强模块以提升光谱重建质量。在多个基准数据集上的大量实验表明，GaussianHSI 在任意尺度高光谱图像超分辨率任务中优于现有的先进方法。

## Abstract
Most existing hyperspectral image super-resolution methods require modifications for different scales, limiting their flexibility in arbitrary-scale reconstruction. 2D Gaussian splatting provides a continuous representation that is compatible with arbitrary-scale super-resolution. Existing methods often rely on rasterization strategies, which may limit flexible spatial modeling. Extending them to hyperspectral image super-resolution remains challenging, as the task requires adaptive spatial reconstruction while preserving spectral fidelity. This paper proposes GaussianHSI, a Gaussian-Splatting-based framework for arbitrary-scale hyperspectral image super-resolution. We develop a Voronoi-Guided Bilateral 2D Gaussian Splatting for spatial reconstruction. After predicting a set of Gaussian functions to represent the input, it associates each target pixel with relevant Gaussian functions through Voronoi-guided selection. The target pixel is then reconstructed by aggregating the selected Gaussian functions with reference-aware bilateral weighting, which considers both geometric relevance and consistency with low-resolution features. We further introduce a Spectral Detail Enhancement module to improve spectral reconstruction. Extensive experiments on benchmark datasets demonstrate the effectiveness of GaussianHSI over state-of-the-art methods for arbitrary-scale hyperspectral image super-resolution.

## 二、方法论：GaussianHSI 框架

### 1. 总体思路

该研究提出了一个名为 **GaussianHSI** 的新框架，通过 **Voronoi 引导的双边二维高斯喷射（VBGS）** 与 **光谱细节增强模块（SDE）** 相结合，实现高质量的任意尺度高光谱图像重建。

框架包括三个阶段：

1. **编码器（Encoder）**：提取低分辨率 HSI 的潜在特征；
2. **空间重建分支（VBGS）**：通过连续高斯表示进行空间扩展；
3. **光谱增强分支（SDE）**：修复和增强各波段的光谱细节；
4. **解码器（Decoder）**：融合两支特征，输出高分辨率 HSI。

---

### 2. Voronoi-Guided Bilateral 2D Gaussian Splatting （VBGS）

- **高斯函数表示**  
  将每个像素建模为一个二维高斯函数，参数包含中心坐标 \((x_n, y_n)\)、协方差矩阵 \(\Sigma_n\)、特征向量 \(c_n\) 等。  
  图像值通过在目标坐标处聚合多个高斯的加权贡献得到。

- **Voronoi 引导选择**  
  为避免全局聚合带来的高计算成本与空间冗余，VBGS使用 **Voronoi 区域竞争机制**：通过 **马氏距离（Mahalanobis Distance）** 衡量目标像素与各高斯之间的相关性，选取前 *k* 个最相关高斯函数参与重建。

- **参考感知双边加权**  
  为保持与原低分辨率影像一致的光谱分布，引入一个 **参考特征加权项**，通过余弦相似度计算高斯特征与低分辨率特征的匹配度，再以指数函数调节其权重：
  \[
  w_{ij}(x, y) = \exp(\gamma s_{ij}(x, y))
  \]
  再根据这些加权系数对选出的高斯进行双边聚合，得到更平滑且光谱一致的重建结果。

---

### 3. Spectral Detail Enhancement （SDE）

- 对低分辨率光谱特征进行上采样，提取局部 *P×P* 的光谱—空间补丁；
- 通过交替的 **空间 MLP** 与 **光谱 MLP** 层进行两次特征变换：
  - 空间 MLP：融合空间位置的局部特征；
  - 光谱 MLP：学习各光谱通道间的相互关系；
- 最后将特征扁平化、融合，生成增强后的光谱信息。

---

### 4. 输出融合与解码

空间增强与光谱增强结果在通道维度拼接后，经 **基于自注意力的解码器**（参考 Swin Transformer）进行特征互动与最终映射，生成高分辨率光谱像素。

## 三、实验设计与验证

### 1. 数据集与场景设定

实验使用了三种常见遥感高光谱数据集：

- **Houston**（ITRES-CASI 1500 传感器，144 光谱波段）
- **Pavia**（ROSIS 传感器，102 光谱波段）
- **Chikusei**（VNIR-C 传感器，128 光谱波段）

每个数据集被划分为 120×120 的非重叠块，其中一半用于训练，另一半用于测试。

### 2. 基准与对比方法

与七至八种主流方法进行了对比，包括：

- CNN 类：MCNet、HDL  
- Transformer 类：MSDformer、ESSAformer、SQformer  
- 状态空间模型：HSRMamba  
- 以及最近基于高斯喷射的 GaussianSR 等。

还额外与任意尺度的 SR 方法比较（SRNO、AnyTSR），以验证泛化能力。

三项指标用于量化评估：

- PSNR（峰值信噪比）
- SSIM（结构相似度）
- SAM（光谱角度映射）

## 六、主要结论与发现

- **核心结论**：
  GaussianHSI 在多个基准数据集上对比现有方法均取得显著性能提升；
  在外分布尺度（×6、×8）下仍保持较高的 PSNR 与最低的 SAM；
  在 Houston 数据集上提升 PSNR 约 0.3–0.5 dB，SSIM 提升约 0.02–0.03。

- **关键发现**：  
  通过 Voronoi 引导与双边加权的连续高斯建模，可在任意尺度下实现结构连贯的空间重建，同时光谱细节增强模块有效保持光谱一致性。
