## 摘要
基于指令的图像编辑（Instruction-based image editing，IIE）旨在根据文本指令修改图像，同时保留与任务无关的内容。尽管近期扩散变换器取得了显著进展，现有方法仍常常遭遇过度编辑的问题，在与期望编辑无关的区域引入了不必要的变化。我们发现，这一局限性源于缺乏明确的编辑定位机制。具体来说，不同的编辑操作（例如添加、删除和替换）会产生不同的空间模式，而当前的 IIE 模型通常以与任务无关的方式处理定位。为了解决这一问题，我们提出了一种无需训练的、具备任务感知能力的编辑定位框架，该框架利用 IIE 模型中固有的源图像和目标图像流。在每个图像流中，我们首先获取基于注意力的编辑线索，然后根据这些注意力线索构建特征质心，以将标记划分为编辑区域和非编辑区域。基于最优定位本质上依赖于任务的观察，我们进一步引入一种统一的掩码构建策略，可针对不同编辑任务有选择地利用源图像和目标图像流。我们系统分析了所提出的见解与方法。在 EdiVal-Bench 上的广泛实验表明，我们的框架在强大的图像编辑骨干模型（包括 Step1X-Edit 和 Qwen-Image-Edit）之上，能够持续提升非编辑区域一致性，同时保持出色的指令遵循性能。

## Abstract
Instruction-based image editing (IIE) aims to modify images according to textual instructions while preserving irrelevant content. Despite recent advances in diffusion transformers, existing methods often suffer from over-editing, introducing unintended changes to regions unrelated to the desired edit. We identify that this limitation arises from the lack of an explicit mechanism for edit localization. In particular, different editing operations (e.g., addition, removal and replacement) induce distinct spatial patterns, yet current IIE models typically treat localization in a task-agnostic manner. To address this limitation, we propose a training-free, task-aware edit localization framework that exploits the intrinsic source and target image streams within IIE models. For each image stream, We first obtain attention-based edit cues, and then construct feature centroids based on these attentive cues to partition tokens into edit and non-edit regions. Based on the observation that optimal localization is inherently task-dependent, we further introduce a unified mask construction strategy that selectively leverages source and target image streams for different editing tasks. We provide a systematic analysis for our proposed insights and approaches. Extensive experiments on EdiVal-Bench demonstrate our framework consistently improves non-edit region consistency while maintaining strong instruction-following performance on top of powerful recent image editing backbones, including Step1X-Edit and Qwen-Image-Edit.

## 一、研究背景与核心问题

- **研究领域与目标**：  
  论文聚焦于 *指令式图像编辑*（Instruction-based Image Editing，IIE），即利用自然语言指令对现有图像进行修改，同时保持与任务无关的内容不变。这类任务广泛用于智能创作、交互设计、内容生成等场景。

- **问题动机**：  
  尽管近年来扩散变换器（Diffusion Transformer, DiT）提升了图像生成与编辑的能力，但现有方法在执行编辑时，经常**过度修改非目标区域**，导致背景、色调或视角等内容被不必要地改变。这种“过编辑”问题削弱了输出图像的可信度与一致性。

- **根本原因分析**：  
  研究指出此现象源于缺乏“**显式编辑定位机制**”，即模型并未区分哪些区域需要被更新、哪些区域应保持原样。  
  传统方法在定位时一般采用“任务无关”的统一方案，未考虑编辑类型之间（如添加、删除、替换）的差异性与空间特征分布差异。

## 二、方法论：任务感知的训练-free 编辑定位框架

### 核心思想
该工作提出一种**无需额外训练（training-free）**的、**基于任务感知的编辑定位框架**，核心目标是：
> 在不改变已有强大扩散模型结构的前提下，通过分析模型内部注意力与特征，显式推导出“应编辑区域”掩码，从而约束编辑过程，仅在所需区域更新潜变量。

### 整体流程（见论文 Fig.2）
1. **输入与结构**：  
   使用扩散 Transformer (DiT) 框架，输入包括文本指令、源图像、以及目标图像的噪声表示。模型的目标是逐步从噪声生成符合指令的目标图像。
2. **注意力线索提取（Attention-Based Semantic Estimation）**：  
   - 从 DiT 每层的联合注意力矩阵中分离出 *跨模态（文本-图像）* 和 *自模态（图像-图像）* 相关子矩阵；  
   - 通过类随机游走传播机制扩散语言信号，以得到更具空间一致性的**注意力图**；
   - 对不同图像流（source / target）分别产生粗略编辑区域掩码。
3. **特征质心划分（Feature-Based Semantic Assignment）**：  
   - 将注意力掩码视为初始语义指示器；
   - 依据掩码区域在潜空间中计算两类特征质心（编辑区 / 保留区）；
   - 根据余弦相似度，将每个 token 归类至最相似的质心，实现更精细的**语义分割式掩码**生成。
4. **任务感知掩码构建（Task-Aware Mask Construction）**：  
   不同编辑类型的定位来源不同：
   - **新增（Addition）**：仅使用目标流掩码（新增内容在目标特征中显现）；
   - **删除（Removal）**：仅使用源流掩码（需移除原有内容）；
   - **替换（Replacement）**：取两流掩码的并集，覆盖旧对象与新对象。
   后续通过保留最大连通区、孔洞填充与边界膨胀完成掩码后处理。
5. **掩码引导潜变量保留（Mask-Guided Latent Preservation）**：  
   - 在去噪步骤中，仅允许编辑区域的潜变量更新；非编辑区域则根据噪声调度函数 σₜ 与源图像潜变量构建**倒转潜变量**进行混合；
   - 实现了局部的潜空间冻结与更新控制，从而防止无关区域被篡改。

## 六、主要结论与发现

- 特征空间信号优于注意力得分：潜特征在编辑定位中更稳健、语义一致。  
- 编辑掩码的最优形态与编辑任务强相关：
  - 新增语义 → 目标流显现；
  - 删除语义 → 源流显现；
  - 替换语义 → 两流并存。
- 在 EdiVal-Bench 上：
  - **背景一致性** 提升 +3.18（Qwen 模型），+0.48（Step1X 模型）；
  - **主体一致性** 提升 +5.27（Qwen），+1.04（Step1X）；
  - **指令跟随度** 保持或略有提升；
  - **视觉质量** 基本不降。

## 八、不足与局限

- **掩码局限性**：  
  当基底模型输出存在强烈结构变动（如视角变化），源/目标流遮罩无法正确对应，出现失败样例（Fig.8）。
- **多区域编辑支持不足**：  
  当前后处理仅保留最大连通域，无法有效处理多目标编辑场景。
- **延迟权衡**：  
  尽管计算代价低，但增加若干步骤的特征计算与融合，在高分辨率图像上可能影响实时性。
- **非端到端优化**：  
  框架依赖固定掩码规则，未进行联合优化，理论上仍可进一步提升鲁棒性。
