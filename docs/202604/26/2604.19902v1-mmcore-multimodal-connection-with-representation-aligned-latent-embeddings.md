## 摘要
我们提出了 MMCORE，这是一个为多模态图像生成与编辑设计的统一框架。MMCORE 利用预训练的视觉-语言模型（VLM），通过可学习的查询标记（query tokens）预测语义视觉嵌入，这些嵌入随后作为扩散模型的条件信号。该简化的设计有效地将 VLM 的丰富理解与推理能力转移到视觉生成过程中。通过避免自回归模型与扩散模型之间的深度融合或从零训练，MMCORE 在保持高保真图像合成的同时显著减少了计算开销。MMCORE 无缝整合了文本到图像的合成与交错的图像生成，在复杂场景（如空间推理与视觉定位）中表现出强大的多模态理解能力。全面评估显示，MMCORE 在广泛的文本到图像以及单图/多图编辑基准上均持续优于现有最先进的方法。

## Abstract
We present MMCORE, a unified framework designed for multimodal image generation and editing. MMCORE leverages a pre-trained Vision-Language Model (VLM) to predict semantic visual embeddings via learnable query tokens, which subsequently serve as conditioning signals for a diffusion model. This streamlined design effectively transfers the rich understanding and reasoning capabilities of VLMs into the visual generation process. By obviating the need for deep fusion between autoregressive and diffusion models or training from scratch, MMCORE significantly reduces computational overhead while maintaining high-fidelity synthesis.   MMCORE seamlessly integrates text-to-image synthesis with interleaved image generation, demonstrating robust multimodal comprehension in complex scenarios such as spatial reasoning and visual grounding. Comprehensive evaluations indicate that MMCORE consistently outperforms state-of-the-art baselines across a broad spectrum of text-to-image and single/multi-image editing benchmarks.

- **研究目标**：
  - 构建一个**低资源、高保真、语义准确**的统一多模态图像生成系统；
  - 在不强行融合两种范式（AR + Diffusion）的前提下，实现跨模态对齐与协同。

## 二、方法论与核心思想

### 1. 总体框架

- **模型名称**：MMCORE（MultiModal COnnection with Representation Aligned Latent Embeddings）。
- **核心思想**：利用一个**预训练的多模态大语言模型（MLLM/VLM）**提取语义丰富的视觉潜在嵌入，通过**可学习的查询 token（query tokens）**机制将这些嵌入传递给一个扩散生成器作为条件输入。
- 该框架实现了**语义理解（由 MLLM/VLM 提供）与高质量生成（由 Diffusion 提供）**的解耦式协作。

### 2. 关键技术细节

- **(1) 语义可学习查询机制**：
  - 以固定数量 N 的 learnable query tokens 从 MLLM 的输出中提取跨模态语义特征；
  - 同时保留**文本 token 序列**，构成“双通道条件输入”；
  - N 在消融实验中从 1 到 128 变化，最优为 64。

- **(2) 表征对齐与蒸馏损失**：
  - 引入**视觉语义对齐约束**，将生成的视觉嵌入对齐到冻结的 SoTA 视觉编码器（如 SigLIP、ViT）的特征空间；
  - 对齐损失：
    \[
    L_{vis} = \frac{1}{N}\sum_i (1 - \cos(F_\theta(Q_i), v_i))
    \]
  - 总损失：
    \[
    L_{MLLM} = \lambda_t L_{llm} + \lambda_a L_{vis}
    \]

- **(3) 双路径条件机制（Dual-Pathway Conditioning）**：
  - **视觉潜变量**：捕获全局语义与跨模态联系；
  - **文本 token**：保留细粒度的语言约束；
  - 提升了复杂指令下的控制与一致性。

- **(4) 扩散头（Diffusion Head）改造与训练策略**：
  - 使用已有的 MMDiT 作为生成主干；
  - 采用**块级因果注意力掩码（block-causal attention mask）**控制历史帧依赖；
  - 引入**独立嵌入 Dropout（Independent Embedding Dropout）**策略以增强 multimodal 协调；
  - 训练流程分三阶段：
    1. MLLM 语义视觉对齐；
    2. 扩散头继续预训练（CPT）；
    3. SFT + RLHF 人类反馈微调。
