## 摘要
单幅图像去雾通常受到恢复质量与计算效率之间权衡的限制。尽管 CNN 网络在计算上高效，但难以学习针对浓密和非均匀雾霾的鲁棒先验。相反，扩散模型提供了强大的生成先验，但存在推理延迟严重和采样不稳定的问题。为克服这些局限，我们提出了 ZID-Net，这是一种新颖的架构，显式地将扩散监督与前馈推理解耦。为了实现高效推理，我们设计了一个频率-空间解耦的前馈主干。在该主干中，通道-空间拉普拉斯掩码（Channel-Spatial Laplacian Mask, CSLM）过滤由雾增强的噪声，从而提取纯净的结构细节，而轻量级全局上下文块（Lightweight Global Context Blocks, LGCBs）则建立长距离的空间依赖，以捕获雾霾的全局变化。动态特征仲裁块（Dynamic Feature Arbitration Block, DFAB）随后自适应地融合这些语义与结构特征，实现稳健重建。为在不增加推理解码成本的情况下向主干引入物理先验，我们在训练阶段引入了零推理先验传播头（Zero-Inference Prior Propagation Head, ZI-PPH）。ZI-PPH 利用条件扩散过程预测残余噪声，为主干提供与退化感知相关的结构监督。在测试阶段舍弃扩散分支后，ZID-Net 将扩散先验集成到一个纯前馈架构中，实现精确且高效的恢复。在合成 RESIDE 数据集上，ZID-Net 达到了 40.75 dB 的 PSNR，并在真实场景数据集上相比现有方法提升 1.13 dB。此外，在 StateHaze1k 遥感数据集上，ZID-Net 获得了 3.06 dB 的 PSNR 提升，推理时间仅为 19.35 毫秒。项目代码地址：https://github.com/XoomitLXH/ZID-Net。

## Abstract
Single image dehazing is often constrained by a trade-off between restoration quality and computational efficiency. While efficient, CNN networks struggle to learn robust priors for dense and non-homogeneous haze. Conversely, diffusion models provide strong generative priors but suffer from severe inference latency and sampling instability. To address these limitations, we propose ZID-Net, a novel framework that explicitly decouples diffusion supervision from feed-forward inference. For efficient inference, we design a frequency-spatial decoupled feed-forward backbone. Within this backbone, a Channel-Spatial Laplacian Mask (CSLM) filters haze-amplified noise to extract purified structural details, while Lightweight Global Context Blocks (LGCBs) establish long-range spatial dependencies to capture the global variations of haze. A Dynamic Feature Arbitration Block (DFAB) then adaptively fuses these semantic and structural features for robust reconstruction. To provide this backbone with physical priors without the inference cost, we introduce a Zero-Inference Prior Propagation Head (ZI-PPH) during training. ZI-PPH leverages a conditional diffusion process to predict residual noise, providing degradation-aware structural supervision to the backbone. By discarding the diffusion branch at test time, ZID-Net integrates diffusion priors into a pure feed-forward architecture for accurate and efficient restoration. ZID-Net achieves 40.75 dB PSNR on the synthetic RESIDE dataset and outperforms existing methods with a 1.13 dB gain on real-world datasets. Additionally, it yields a 3.06 dB PSNR gain on the StateHaze1k remote sensing dataset with an inference time of just 19.35 ms. The project code is available at: https://github.com/XoomitLXH/ZID-Net.

- **研究动机**：单幅图像去雾（Single Image Dehazing）任务旨在从受雾霾影响的图像中恢复清晰视图，但该问题天然具有高度不适定性。实际应用（如自动驾驶、遥感、医学成像）要求模型既具备高恢复质量，又能高效实时推理。

## 二、方法论与模型设计

### 1. 核心思想
- **“扩散监督与前馈推理解耦”**：在训练阶段引入扩散模型进行退化建模和先验学习，而测试阶段完全舍弃扩散分支，仅使用前馈单次推理的卷积架构。
- **模型名称**：ZID-Net（Zero-Inference Diffusion Prior Decoupling Network）。

### 2. 架构总体结构
- 模型由两部分组成：
  1. **频率–空间解耦前馈主干**（Feed-forward Backbone）；
  2. **训练阶段的辅助扩散先验分支**（Zero-Inference Prior Propagation Head, ZI-PPH）。

### 3. 关键模块与机制

- **通道–空间拉普拉斯掩码（CSLM）**  
  - 用色彩拉普拉斯算子提取 RGB 通道的高频残差，以物理滤波机制抑制雾增强噪声并保留结构边缘。  
  - 动态结合通道描述符与空间描述符，实现自适应噪声抑制和边缘强化。

- **轻量级全局上下文块（LGCB）**  
  - 用线性复杂度的通道空间注意机制替代传统 Transformer 的二次复杂度自注意力。  
  - 在瓶颈层（即最小分辨率层）实现全局语义聚合，建立远程空间依赖。  

- **动态特征仲裁块（DFAB）**  
  - 融合语义分支与结构分支的特征，通过 SE（Squeeze-and-Excitation）机制对通道进行加权重标定，平衡语义与物理信息的融合。

- **零推理扩散先验传播头（ZI-PPH）**  
  - 在训练阶段，ZI-PPH将清晰图与雾图的差异定义为退化残差，通过条件扩散过程生成带噪残差。  
  - 网络学习预测加入的噪声，以辅助主干建模雾的分布（公式 \( R_t = \sqrt{\alpha_t}R + \sqrt{1 - \alpha_t}\varepsilon \)）。  
  - 通过时序嵌入和双尺度融合增强语义引导，使主干学习退化感知特征。  
  - 测试阶段完全移除此分支，实现“零推理”扩散先验。

### 4. 损失函数设计
综合三项目标：
- **重建损失**：L1 像素误差。  
- **对比损失**：基于 VGG-19 特征空间的正负样本对比，提升感知质量。  
- **扩散监督损失**：训练阶段拟合扩散噪声，嵌入退化感知先验。  
最终组合：  
\[
\mathcal{L}_{total} = \lambda_1 \mathcal{L}_1 + \lambda_2 \mathcal{L}_{contrast} + \lambda_3 \mathcal{L}_{diff}
\]
其中 \(\lambda_1 = 1.0,\ \lambda_2 = 0.1,\ \lambda_3 = 0.35\)。
