## 摘要
大型扩散式变换器（DiTs）在执行全局编辑指令时表现良好，但局部编辑往往扩散至无关区域，因为联合注意力架构中没有显式通道指示网络应当在何处应用编辑。我们提出了 REDEdit，这是一种联合训练的、具备指令与区域感知的适配器框架，可以在不修改主干权重的情况下将一个冻结的 DiT 改造为精确的局部图像编辑器。在每个变换器块中部署轻量级的 Block Adapter，用于注入结构化条件流，将“编辑什么”（指令语义）与“编辑哪里”（空间掩码）进行分解；学习得到的 SpatialGate 选择性地将适配器信号引入编辑区域，同时保持图像其他部分几乎与原始图一致；Region-Aware Loss 使训练目标聚焦于变化的像素。由于这些组件使主干的内部表示在端到端过程中具备掩码感知能力，一个与编辑器联合训练的轻量级 MaskPredictor 头可以直接从指令和源图像中确定编辑区域，从而在推理阶段无需用户掩码。我们在两个互补基准上进行评估：MagicBrush（具有配对的真实目标）用于衡量像素级保真度与编辑精度，Emu-Edit Test（无真实图像，包含 9 种不同编辑类别）用于测试指令遵循与编辑类型的泛化能力。REDEdit 在两种任务上均取得了最新的最优结果，同时超越了无掩码和理想掩码基线。七种变体的消融实验明确分离了各组件的贡献。

## Abstract
Large diffusion transformers (DiTs) follow global editing instructions well but consistently leak local edits into unrelated regions, because joint-attention architectures offer no explicit channel telling the network where to apply the edit. We introduce REDEdit, a co-trained, instruction- and region-aware adapter framework that retrofits a frozen DiT into a precise local editor without modifying its backbone weights. A lightweight Block Adapter at every transformer block injects a structured condition stream that factorizes what to edit (instruction semantics) from where to edit (spatial mask); a learned SpatialGate routes the adapter signal selectively into the edit region while keeping the rest of the image near-identical to the source; and a Region-Aware Loss focuses the training objective on the changing pixels. Because these components make the backbone's internal representation mask-aware end-to-end, a thin MaskPredictor head trained jointly with the editor can ground the edit region directly from the instruction and source image eliminating any user-mask requirement at deployment. We evaluate on two complementary benchmarks: MagicBrush (paired ground-truth targets) to measure pixel-level preservation and edit accuracy, and Emu-Edit Test (no ground-truth images, 9 diverse edit categories) to stress-test instruction following and generalization across edit types. On both, REDEdit achieves state-of-the-art results, simultaneously outperforming mask-free and oracle-mask baselines. A seven-variant ablation cleanly isolates the contribution of each component.

## 1. 研究问题与背景  

- **核心问题**：  
  大规模扩散变换器（DiTs）在执行全局图像编辑任务时表现良好，但在局部编辑中存在“编辑泄露”问题——即模型会无意地修改目标区域之外的像素。其原因在于：联合注意力（joint attention）结构无法提供“编辑位置”的显式通道。  

- **动机**：  
  当前方法分为两大类：  
  1. **联合注意力型编辑器**（如 InstructPix2Pix、MagicBrush、UltraEdit、Emu-Edit 等）能处理自然语言指令，但无法控制编辑位置。  
  2. **掩码条件型方法**（如 BrushNet、FLUX-Fill）通过用户提供的二值掩码来约束编辑区域，但人工掩码创建成本高，且会引入额外延迟与错误。  

- **研究目标**：  
  设计一种可让 DiT 在**不修改主干权重**的情况下，自动学会**“编辑什么”与“编辑哪里”**的分离控制，实现**无掩码（mask-free）局部编辑**。

## 2. 方法论与技术细节  

### 核心思想  
提出 **REDEdit（Region-Aware Adapter Injection）** ——在冻结的 DiT 中注入区域感知适配器，实现空间和语义双重对齐。框架由五个关键组件构成：

1. **Block Adapter（块级适配器）**  
   - 在 DiT 的每个 transformer block 中嵌入一个轻量级**交叉注意力适配器**，输入为“结构化条件流”`c`。  
   - 计算公式（文字描述）：适配器通过交叉注意力计算条件响应 \( u = CA(q=h, k=c, v=c) \)，再经过前馈网络（FFN）并被空间门控向量 `g` 逐位置地调制，输出更新后的特征。  
   - 优点：主干冻结，可在不改动原模型的情况下实现空间选择性调节。  

2. **Condition Encoder（条件编码器）**  
   - 将“编辑内容”与“编辑位置”分解：  
     - 空间子流：由掩码 \( M \) 生成的空间令牌；  
     - 语义子流：来自语言-视觉编码器（Qwen2.5-VL）的语义令牌，经 Perceiver 跨注意力压缩生成固定长度语义向量；  
     - 二者经融合 transformer 融合为结构化条件流。  

3. **SpatialGate（空间门控）**  
   - 通过两层 MLP 将下采样的掩码映射为每个图像令牌的软门值 \([0,1]\)，控制适配器信号进入区域的强弱。  
   - 边界区域门值连续可变，避免硬掩码带来的接缝伪影。  

4. **Region-Aware Loss（区域加权损失）**  
   - 在标准流匹配速度预测损失上，加权编辑区域像素（权重 1+α，α=2），强化编辑区域梯度信号，防止训练被大量“未编辑像素”的零梯度稀释。  

5. **MaskPredictor（掩码预测头）**  
   - 小型交叉注意力解码器，输入为指令与源图像；  
   - 以 BCE + Dice 的解耦损失训练，用于从文本直接预测编辑区域。  
   - 使用**解耦训练策略**：编辑器路径始终用真值掩码训练，而 MaskPredictor 单独接受监督，从而避免“错误预测—编辑错误—梯度弱化”的反馈环。
