## 摘要
在图像中控制光照对于摄影和视觉内容创作至关重要。尽管闭源模型已经展现出令人印象深刻的光照控制能力，但开源替代方案要么需要大量控制输入（如深度图），要么不公开其数据和代码。我们提出一个完全开源且可复现的管线，用于在扩散模型中学习光照控制。我们的方法构建了一个数据引擎，将光照良好的图像转换为监督训练三元组，包括一个光照较差的输入图像、一条自然语言的光照指令，以及一个光照良好的输出图像。我们在这些数据上微调扩散模型，并在感知相似度、结构相似度和身份保持方面显著超越了基线模型 SD 1.5、SDXL 和 FLUX.1-dev。我们的工作提供了一个完全基于开源工具和公开数据的可复现解决方案。我们公开发布全部代码、数据和模型权重。

## Abstract
Controlling illumination in images is essential for photography and visual content creation. While closed-source models have demonstrated impressive illumination control, open-source alternatives either require heavy control inputs like depth maps or do not release their data and code. We present a fully open-source and reproducible pipeline for learning illumination control in diffusion models. Our approach builds a data engine that transforms well-lit images into supervised training triplets consisting of a poorly-illuminated input image, a natural language lighting instruction, and a well-illuminated output image. We finetune a diffusion model on this data and demonstrate significant improvements over baseline SD 1.5, SDXL, and FLUX.1-dev models in perceptual similarity, structural similarity, and identity preservation. Our work provides a reproducible solution built entirely with open-source tools and publicly available data. We release all our code, data, and model weights publicly.

## 二、方法论与技术细节

### 核心思想
将“光照控制”问题重新定义为**基于自然语言指令的监督图像编辑问题**，而非基于场景重渲染。  
研究通过**数据自动生成管线（Data Engine）**构造训练三元组：

\[
\text{Triplet} = (\text{输入图像}_\text{光照较差},\ \text{照明指令},\ \text{目标图像}_\text{良好光照})
\]

从而让模型学习如何根据指令在保持身份一致的前提下调节图像光照。

### 数据引擎流程（Algorithm 1概要）
1. **CLIP 过滤光照质量**：利用 CLIP 图像-文本相似度判断图像是否“光照良好”；设定阈值 0.21 过滤出约 1.2 万张高质量人脸图像。  
2. **前景分割**：采用 SAM3（Segment Anything Model v3）隔离主体，去除背景干扰。  
3. **反射率提取（Albedo Extraction）**：依照 Retinex 理论分离反射率与照明分量，获得光照中性图像。  
4. **合成光照退化（Illumination Degradation）**：  
   - 使用 MiDaS 获得单目深度。  
   - 基于 Lambertian 模型计算定向光照、阴影与环境光。  
   - 随机叠加十种阴影模式（如树叶遮挡、窗格阴影）以生成多样暗光场景。  
5. **照明指令生成**：采用多模态语言模型 Qwen3-VL 自动生成描述性指令，例如“柔和自然光从左前方照亮面部”。  

最终得到训练数据集三元组集 \( D_{train} = \{(x_{deg}, t_{instr}, x_{gt})\} \)。

### 模型与训练
- 使用 **Stable Diffusion 1.5** 的 **InstructPix2Pix** 架构作为基础。  
- 保留原始 VAE 与文本编码器，仅微调 U-Net 主体。  
- 优化器：AdamW，学习率 \(1\times10^{-5}\)，批量大小 24。  
- 训练时长约 250 个 epoch，约需 **8 × NVIDIA A100（80GB）** GPU **5.5 小时**。

- 所有数据按 Stable Diffusion 标准缩放至 **512×512**。

### Benchmark 与对比模型
- 对比的开源基线：
  1. **Stable Diffusion 1.5**
  2. **Stable Diffusion XL (SDXL)**

## 六、主要结论与发现

- 微调后的模型在定量指标上显著优于基线：
  - LPIPS: 0.30（vs. 0.63–0.65）
  - SSIM: 0.57（vs. 0.37–0.43）
  - Identity Score: 0.76（vs. 0.04–0.11），实现**7–17 倍身份保持提升**
- 模型能有效匹配照明指令并维持面部结构与身份一致性。  
- 可泛化至未见分布（CelebA-HQ），输出稳定自然。
