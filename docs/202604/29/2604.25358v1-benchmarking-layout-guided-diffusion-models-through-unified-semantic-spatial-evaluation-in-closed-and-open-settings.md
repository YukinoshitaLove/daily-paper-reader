## 摘要
评估基于布局指导的文本到图像生成模型，需要同时检验其与文本提示的语义一致性以及与所给布局的空间一致性。评估布局一致性通常需要收集细粒度标注，这既昂贵又耗时。因此，现有基准测试很少提供全面的布局评价，且往往在规模或覆盖范围上受限，使得模型的比较、排序及解释变得困难。在本文中，我们引入了一个封闭集基准（C-Bench），旨在隔离关键的生成能力，同时在提示结构和布局复杂度上提供不同层次的挑战。为配合这一受控环境，我们提出了一个开放集基准（O-Bench），利用真实世界的提示与布局来评估模型，为自然场景下的语义与空间一致性提供衡量标准。我们进一步开发了一套统一的评估协议，将语义准确性和空间准确性结合为单一评分，从而确保模型排序的一致性。基于我们的基准测试，我们对六个最先进的布局指导扩散模型进行了大规模评估，总计生成并评估了 319,086 张图像。我们据此建立了总体性能排名，并提供了文本与布局一致性的详细分解，以增强结果的可解释性。通过在不同场景与提示复杂度下的细粒度分析，我们揭示了当前模型的优势与局限。代码可在 https://github.com/lparolari/cobench 获取。

## Abstract
Evaluating layout-guided text-to-image generative models requires assessing both semantic alignment with textual prompts and spatial fidelity to prescribed layouts. Assessing layout alignment requires collecting fine-grained annotations, which is costly and labor-intensive. Consequently, current benchmarks rarely provide comprehensive layout evaluation and often remain limited in scale or coverage, making model comparison, ranking, and interpretation difficult. In this work, we introduce a closed-set benchmark (C-Bench) designed to isolate key generative capabilities while providing varying levels of complexity in both prompt structure and layout. To complement this controlled setting, we propose an open-set benchmark (O-Bench) that evaluates models using real-world prompts and layouts, offering a measure of semantic and spatial alignment in the wild. We further develop a unified evaluation protocol that combines semantic and spatial accuracy into a single score, ensuring consistent model ranking. Using our benchmarks, we conduct a large-scale evaluation of six state-of-the-art layout-guided diffusion models, totaling 319,086 generated and evaluated images. We establish a model ranking based on their overall performance and provide detailed breakdowns for text and layout alignment to enhance interpretability. Fine-grained analyses across scenarios and prompt complexities highlight the strengths and limitations of current models. Code is available at https://github.com/lparolari/cobench.

  作者希望构建一个体系化、可复用的评测框架，使研究者能**公平比较不同布局引导扩散模型**，并同时衡量其**语义与空间能力**。

## 二、方法论与核心思想

### 1. 整体框架  
论文提出了完整的评测系统，包含：
- **C-Bench（Closed-set Benchmark）**：  
  封闭、可控的评测集，用于诊断模型在特定生成能力（如颜色绑定、属性理解等）下的表现。  
- **O-Bench（Open-set Benchmark）**：  
  基于真实数据（Flickr30k Entities），测试模型在**开放自然场景**下的泛化能力。  
- **统一评估协议（Unified Evaluation Protocol）**：  
  将语义准确性与空间对齐度整合为单一指标，确保模型排名一致且可解释。  

### 2. 关键技术细节  

**（1）Prompt Generation Engine，PGE）文本提示生成模块**  
- 采用模板与大型语言模型（LLM）结合的方式生成文本提示。  
- 设置7类场景：  
  - 物体绑定（Object binding）  
  - 颜色绑定（Color binding）  
  - 属性绑定（Attribute binding）  
  - 物体关系（Object relationship）  
  - 小目标（Small bboxes）  
  - 重叠目标（Overlapped bboxes）  
  - 复杂组合（Complex compositions）  
- 模板严格限制对象、属性与关系，用 LLM（如 ChatGPT‑4o）生成复杂描述。

**（2）Layout Generation Engine，LGE）布局生成模块**  
- 基于约束的合成方式生成 bounding boxes：  
  - 对单目标随机放置；  
  - 对多目标遵从空间关系（如“在上方”“在左侧”）；  
  - 支持重叠和非重叠控制；  
  - 自动解析 LLM 生成文本中的对象（使用 spaCy 解析名词短语）。

**（3）评估指标设计**  

- **语义对齐分数 \(s_{text}\)**：  
  使用 **TIFA 指标**（通过 QA 模型 BLIP-2 分析生成图像回答问题的正确率）。  
  \[
  s_{text} = \frac{1}{N} \sum_i \frac{1}{M_i}\sum_j 1[A^{\text{VQA}}_{j,i}=A_{j,i}]
  \]
- **布局对齐分数 \(s_{layout}\)**：  
  基于 zero-shot 检测器（OWLv2），计算预测与参考 box 的 **IoU Accuracy@k 曲线的面积 (AUC)**。  
  \[
  s_{layout}(i) = \text{AUC}(\text{Acc@k}_{i}),\quad s_{layout} = \frac{1}{N}\sum_i s_{layout}(i)
  \]
- **统一分数 \(s_{unified}\)**：语义+空间的调和平均，类似 F1 分数：
  \[
  s_{unified} = \frac{2 s_{text} s_{layout}}{s_{text} + s_{layout}}
  \]
  此设计可惩罚语义与空间表现不平衡的模型。  

### 3. 自动化评估管线  
- 输入：Prompt + Layout  
- 模型生成图像  
- 自动问答分析 + 目标检测  
- 输出三种指标结果，并支持按场景与物体数量分解分析。

## 六、主要结论与发现

- **性能排名**（综合指标 \(s_{unified}\)）：  
  - MIGC（封闭集 0.7082，开放集 0.7548）为最优；  
  - BoxDiff、GLIGEN 紧随其后；  
  - Stable Diffusion 基线显著较低（0.25–0.45）。  
- **主要现象**：  
  - 所有模型在对象数量增加时性能下降；  
  - 错误主要来源于布局一致性不足；  
  - MIGC 和 GLIGEN 这类具布局训练的模型在空间一致性方面明显优于无布局模型。
