## 摘要
扩散模型近年来在图像修复任务（包括超分辨率）中表现出强大的性能。然而，其庞大的模型规模和迭代采样过程使得其在实际部署中计算成本高昂。本文提出了TOC-SR框架，用于通过首先构建紧凑的扩散骨干网络来实现高效的一步超分辨率模型。从一个具有16通道的潜变量扩散模型出发，我们通过特征级生成蒸馏构建参数高效的代理模块，并利用ε约束贝叶斯优化进行架构搜索，以在保持生成保真度的同时最小化模型复杂度。所得的紧凑扩散骨干网络相较于扩展的扩散模型，参数量减少6.6倍，GMACs减少2.8倍。随后我们将该骨干网络应用于超分辨率任务，并将扩散过程蒸馏为单步生成器。实验结果表明，所提出的方法能够在保持良好重建质量的同时，实现高效的超分辨率重建。

## Abstract
Diffusion models have recently demonstrated strong performance for image restoration tasks, including super-resolution. However, their large model size and iterative sampling procedures make them computationally expensive for practical deployment. In this work, we present TOC-SR, a framework for building efficient one-step super-resolution models by first discovering a compact diffusion backbone. Starting from a sixteen-channel latent diffusion model, we construct parameter-efficient surrogate blocks using feature-wise generative distillation and perform architecture discovery using epsilon-constrained Bayesian Optimization to minimize model complexity while preserving generative fidelity. The resulting compact diffusion backbone achieves a 6.6x reduction in parameters and a 2.8x reduction in GMACs compared to the expanded diffusion model. We then adapt this backbone for super-resolution and distill the diffusion process into a single-step generator. Experiments demonstrate that the proposed approach enables efficient super-resolution while maintaining strong reconstruction quality.

## 二、方法论与核心技术

### 1. 框架总体思路

TOC-SR 的整体流程分为 **三阶段**：
1. **潜空间容量扩展**：将 Stable Diffusion 1.5 的 4 通道潜空间扩展至 16 通道，以解决高频细节丢失问题；
2. **紧凑扩散骨干发现**：利用 **特征级生成蒸馏（Feature-wise Generative Distillation, FwKD）** 和 **ε-约束贝叶斯优化（ε-Constrained Bayesian Optimization）** 搜索最优轻量架构；
3. **任务适配与单步蒸馏**：将紧凑模型适配到 SR 任务中，并通过单步生成蒸馏（Step Distillation）将多步扩散过程浓缩为一次推理。

### 2. 关键技术细节

#### （1）16通道潜空间扩展

- 原 Stable Diffusion VAE 使用 4 通道潜变量（C=4），会导致细节损失。  
- 本文通过 latent-space distillation 的方式，从 FLUX 模型族的高容量 VAE 蒸馏到改进版 Stable Diffusion VAE：
  \[
  L_{VAE} = \lambda_{lat} \|z_s - z_t\|_2^2 + \lambda_{rec} \|x - D_s(z_s)\|_1
  \]
- 同时修改 U-Net 的输入/输出卷积层以支持 16 通道潜空间，并联合微调确保扩散过程保持生成一致性。

#### （2）ϵ-约束贝叶斯优化的紧凑骨干搜索

- 将 U-Net 拆分为 S 个阶段（encoder/bottleneck/decoder），为每个阶段设计参数约简的代理模块（surrogate blocks），通过 **结构可交换性**（保持输入输出维度一致）保证组装灵活。
- 针对每个模块，使用特征级蒸馏：
  \[
  L_{FwKD}^{(i,j)} = E_{F_i}\|O_{S_i,j} - O_{T_i}\|_2^2
  \]
- 组合形成候选架构 \( M(a) \)，并定义目标函数：  
  参数量最小化 \( f_{param}(a) \)  
  同时约束生成保真度 \( f_{acc}(a) ≤ ε \)。  
- 使用高斯过程（Gaussian Process）代理进行贝叶斯优化，在 ε 网格下探索 Pareto 前沿，最终选出最优 130M 参数模型（比基准减少 6.6×）。

#### （3）超分辨率任务适配

- 使用 LR–HR 成对数据训练条件扩散模型（Conditional Diffusion）：
  \[
  L_{SR} = E \|\epsilon - \epsilon_{\phi}(z_t, t, z_l)\|_2^2
  \]
- LR 输入上采样后经同一 VAE 编码为条件潜变量。

#### （4）单步蒸馏（Step Distillation）

- 基于教师（多步扩散）→ 学生（单步生成器）训练：
  \[
  L_{distill} = E\,\|G_{\psi}(z_T, z_l) - z_{T>0}\|_2^2
  \]
- 通过一次前向推理恢复高分辨率潜变量，实现单步超分辨率生成。
