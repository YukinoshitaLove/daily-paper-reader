## 摘要
使用单一模型进行统一的图像复原常常因退化形式的多样性而出现任务干扰。为了解决这一问题，我们提出了 DACG-IR（Degradation-Aware Adaptive Context Gating，退化感知的自适应上下文门控）方法，该方法能够显式感知退化特征，以动态调节特征表示。我们的方法从输入构建退化感知的上下文表示，以调控注意力分布、频域特征和特征聚合。具体而言，一个轻量级多尺度退化感知模块提取粗略的退化信息，并生成分层提示。这些提示引导编码器和解码器模块中的注意力温度和输出门控，从而实现自适应特征提取。此外，空间-通道双门控自适应融合机制可优化编码器特征，抑制从浅层到深层的噪声传播。该设计有效抑制由退化引起的噪声，同时保留有用的结构信息。实验表明，DACG-IR 在单任务、全能模型、恶劣天气去除以及复合退化场景中均优于当前最先进的方法。代码地址：https://github.com/HlHomes/DACG-IR-code

## Abstract
Unified image restoration using a single model often faces task interference due to diverse degradations. To address this, we propose DACG-IR (Degradation-Aware Adaptive Context Gating), which enables explicit perception of degradation characteristics to dynamically modulate feature representations. Our method constructs degradation-aware contextual representations from the input to modulate attention distribution, frequency-domain features, and feature aggregation. Specifically, a lightweight multi-scale degradation-aware module extracts coarse degradation information and generates layer-wise prompts. These prompts guide attention temperature and output gating in encoder and decoder blocks for adaptive feature extraction. Additionally, a spatial-channel dual-gated adaptive fusion mechanism refines encoder features, suppressing noise propagation from shallow to deep layers. This design effectively suppresses degradation-induced noise while preserving informative structures. Experiments show DACG-IR outperforms state-of-the-art methods in single-task, all-in-one, adverse weather removal, and composite degradation settings. Code: https://github.com/HlHomes/DACG-IR-code

## 一、研究问题与背景

- **研究动机**：传统的图像复原模型通常针对单一类型的退化（如雾、噪声、雨、模糊、低照度等）设计。现实世界中的图像退化往往复杂且多样，单一模型难以兼顾不同退化类型，导致**任务干扰（task interference）**和性能退化。  
- **核心问题**：如何让一个“统一的图像复原模型”能够**同时**处理多种退化形式，而不会出现不同任务之间的负面干扰。
- **研究目标**：构建一种能**显式感知退化模式**并**自适应调节特征表示**的网络框架，使模型在多退化场景下仍能保持稳健和高保真复原能力。

## 二、方法论：核心思想与技术细节

### 1. 核心思想
提出 **Degradation-Aware Adaptive Context Gating（DACG-IR）** 框架，通过学习退化感知特征，引导注意力机制、特征融合及频域调制，实现退化自适应的图像复原。

要点概述：
- 在输入图像中直接提取多尺度退化特征；
- 生成层级退化提示（prompts）用于引导注意力；
- 在解码阶段采用空间–通道双门控融合抑制噪声；
- 综合空间与频域信息，实现多任务统一处理。

### 2. 模块组成与主要机制
DACG-IR 由四个关键模块组成：

#### (a) Degradation-Aware Module（DAM）
- 从输入图像提取多尺度退化统计（均值与标准差）。
- 经**深度卷积+逐点卷积**生成不同层级退化提示 \( P_1, P_2, P_3, P_4 \)。
- 通过**全局平均与标准差池化（GAP/GSP）**形成全局退化向量，捕获退化强度与变异性。
- 生成多层提示引导后续模块的特征调制。

#### (b) Context Gated Dual-Domain Modulation（CGDM）
- 在网络瓶颈层结合**空间域与频域**特征处理。
- 使用 **2D FFT** 将特征映射至频域，通过退化提示生成频域调制掩码以校正全局退化。
- 结构公式描述：
  \[
    Y_{\text{out}} = F_l + Conv_{1x1}([Y_{\text{spa}}, Y_{\text{freq}}])
  \]
  实现局部细节与全局退化的协同修复。

#### (c) Adaptive Gated Fusion（AGF）
- 改进 U-shaped 网络中的 skip connection。
- 通过**空间门 + 通道门**生成自适应掩码：
  \[
    A = \sigma(S + C), \quad F'_{{enc}} = F_{enc} \odot A
  \]
- 动态过滤浅层噪声特征，减轻退化信息在网络中的传播。

#### (d) Context Adaptive Gated Attention（CAGA）
- 基于多头自注意力（MHSA），引入**自适应温度尺度控制**：
  \[
    \tau_h = e^{\theta_{\text{base}} + W_{\tau}(P_i)}
  \]
- 使用退化提示进行**输出门控**：
  \[
    O_{gated} = O_{attn} \odot \sigma(Linear(P_i))
  \]
- 可根据退化强度调整注意力敏锐度，使特征聚合更稳定、更具退化感知能力。

## 六、主要结论与发现

- DACG-IR 在所有统一复原场景中均取得 **当前最优或次优结果**；
  - **五任务平均PSNR** 达 31.07 dB，明显高于 PromptIR、DA-RCOT 等模型；
  - 对复杂气候退化，PSNR 提升约 0.46 dB 至 32.11 dB；
  - 对复合退化场景（CDD11），轻量版 DACG-IR-S 也优于 BioIR-T。
- **模块贡献验证**：
  - AGF 单独可提升 Restormer 平均约 +1.52 dB；
  - CGDM + CAGA协同提升模型可达 +2.27 dB。
- 综上，DACG-IR 有效抑制退化噪声，增强特征区分性和结构保真度。
