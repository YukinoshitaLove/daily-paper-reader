## 摘要
现有的单幅图像去雾方法在处理均匀薄雾图像时表现良好，但在面对空间上雾浓度变化较大、不同区域存在突变的非均匀雾图像时往往效果不佳。为解决这一根本性问题，我们提出了一种新的多分支深度神经网络框架，称为浓度划分与图像融合网络（Concentration Partitioning and Image Fusion Network，简称 CPIFNet），将复杂的非均匀去雾问题分解为若干可处理的均匀子问题。我们的核心思想是，将单张非均匀雾图像视为由多个局部区域组成的复合体，每个区域表现出近似均匀的雾特征。CPIFNet 采用两阶段结构，包括图像增强网络（Image Enhancement Network，IENet）阶段和图像融合网络（Image Fusion Network，IFNet）阶段。在第一阶段，多个 IENet 分支在不同浓度水平的均匀雾数据集上独立训练，生成在对应雾密度区域恢复效果最佳的增强模型。在第二阶段，IFNet 通过深度特征堆叠与融合智能地聚合所有增强输出中的优选区域，生成统一的高质量去雾结果。此外，我们引入了一个综合损失函数，结合重建损失、感知损失、结构损失和色彩损失，以联合监督两个阶段的训练。

## Abstract
Existing single image dehazing methods have demonstrated satisfactory performance on homogeneous thin-haze images; however, they often struggle with non-homogeneous hazy images that exhibit spatially varying haze concentrations and abrupt density transitions across different regions. To address this fundamental limitation, we propose a novel multi-branch deep neural network framework, termed Concentration Partitioning and Image Fusion Network (CPIFNet), which decomposes the challenging non-homogeneous dehazing problem into a set of tractable homogeneous sub-problems. Our key insight is that a single non-homogeneous hazy image can be viewed as a composite of multiple local regions, each exhibiting approximately homogeneous haze characteristics. CPIFNet employs a two-stage architecture consisting of an Image Enhancement Network (IENet) stage and an Image Fusion Network (IFNet) stage. In the first stage, multiple IENet branches are independently trained on homogeneous haze datasets of different concentration levels, producing enhancement models that excel at restoring regions matching their respective haze densities. In the second stage, the IFNet intelligently aggregates the advantageous regions from all enhancement outputs through deep feature stacking and merging, yielding a unified high-quality dehazed result. Furthermore, we introduce a comprehensive loss function incorporating reconstruction, perceptual, structural, and color losses to jointly supervise both stages.

## 一、研究问题与整体含义

- **研究动机**：  
  传统单图像去雾算法（包括基于物理模型的 DCP、CAP 及深度学习网络如 AOD-Net、FFA-Net、DeFormer 等）在处理均匀薄雾图像时效果良好，但在真实场景中常出现**非均匀雾分布**，即不同区域雾浓度差异显著，导致现有模型难以同时处理稠密雾与稀薄雾区域。  
  这种空间非均匀性使得模型容易**过增强或欠增强**，造成色彩失真和结构丢失。

- **核心问题**：如何设计一个能够同时适应不同雾浓度区域的去雾网络，使其在非均匀雾图像中实现统一且自然的视觉还原。

## 二、方法论与核心技术

### 1. 总体框架思想
- 提出了 **CPIFNet (Concentration Partitioning and Image Fusion Network)**，将非均匀图像去雾问题**分解为多个均匀子问题**：
  - 每个局部区域视为具有近似相同雾浓度的子图像。
  - 针对不同浓度水平分别训练独立的去雾子网络。

- 整体结构为**两阶段多分支网络**：
  1. **图像增强阶段（IENet）**：针对不同雾浓度的子集进行独立训练。
  2. **图像融合阶段（IFNet）**：结合各分支输出的特征，通过深度堆叠与融合生成统一的最终图像。

### 2. 技术细节
- **IENet 架构**：
  - 输入大小：256×256×3。
  - 组成：初始卷积层（5×5）、多个残差模块（含三组3×3卷积+ReLU+跳连），最终卷积层输出增强系数。
  - 输出公式：  
    - 增强系数 \( R = \text{IENet}(I) \)  
    - 增强结果 \( J_E = R \times I \)
  - 每个 IENet 子网络仅优化特定浓度范围的图像。

- **IFNet 架构**：
  - 输入：多个 IENet 输出（如 {J₁, J₂, …, Jₙ}）。
  - 多层卷积特征提取后通过**通道堆叠**策略融合，减少维度后逐步整合生成最终结果 \( J_f = \text{IFNet}(J₁, J₂, …, Jₙ) \)。
  - 使用 ReLU 激活与层间特征叠加以保留多尺度信息。

### 3. 损失函数设计
联合监督两个阶段的训练，综合四类损失：
- **重建损失** \( L_m = ||J - Y||_1 \)：约束像素级一致性。  
- **感知损失**：使用 VGG 网络多层特征差异。  
- **结构损失** \( L_s = 1 - \text{SSIM}(J,Y) \)：保障结构保真。  
- **色彩损失**：基于像素 RGB 向量夹角余弦，维持色彩自然度。  
总体目标函数：  
\( L = L_m + L_p + L_s + L_c \)

## 八、不足与局限

- **计算代价**：多分支设计导致参数量与训练时间增加，在实时应用中需进一步模型压缩。
- **数据依赖性**：分支训练依赖雾浓度划分准确性，划分标准可能影响性能。
- **极端场景处理**：对超稠密雾（近零透光）区域仍存在恢复困难。
- **外部因素未考虑**：如动态天气、光照变化或多模态信息（红外/深度图），目前模型仅处理RGB静态图。
