## 摘要
近年来，生成模型的进展显著提升了图像编辑的能力，然而实现精确且直观的用户控制仍然困难。具体而言，用户常常难以同时表达精确的空间布局与特定的语义细节。自然语言指令能够有效传达诸如纹理和颜色等高层语义，但缺乏空间上的精确性。相反，自由手绘涂鸦（scribble）可提供粗略的空间边界，却无法表达细致的视觉属性。因此，要实现精确控制，需要结合这两种模态。然而，现有模型在同时理解抽象涂鸦与文本时表现不佳，主要是因为缺乏专门的训练数据。为此，我们提出了 ScribbleEdit，这是一种大规模的合成数据集，旨在通过结合自然语言指令与自由手绘涂鸦输入，促成更精确、可控的编辑。我们通过一个合成管线构建该数据集，该管线会自动生成用于修复（inpainting）的源-目标图像对，并与人工绘制的涂鸦及由多模态语言模型（VLM）生成的文本指令配对。借助 ScribbleEdit，我们评估并微调了基于扩散（diffusion）的模型和自回归统一多模态图像编辑模型。实验结果表明，现成模型在处理抽象涂鸦输入时表现欠佳，而在我们的合成数据集上进行微调后，其生成的编辑在空间对齐和语义一致性方面显著提升。

## Abstract
Recent progress in generative models has significantly advanced image editing capabilities, yet precise and intuitive user control remains difficult. Specifically, users often struggle to communicate both exact spatial layouts and specific semantic details simultaneously. While natural language instructions effectively convey high-level semantics like texture and color, they lack spatial specificity. Conversely, freehand scribbles provide rough spatial boundaries but cannot express detailed visual attributes. Consequently, achieving precise control requires combining both modalities. However, existing models struggle to jointly interpret abstract scribbles alongside text due to a lack of specialized training data.   In this work, we introduce ScribbleEdit, a large-scale synthetic dataset designed to bridge this gap by combining natural language instructions with freehand scribble inputs for more accurate, controllable edits. We construct this dataset through a synthetic pipeline that automatically generates source-target image pairs via inpainting, which are then paired with human-drawn scribbles and VLM-generated text instructions. Using ScribbleEdit, we evaluate and finetune both diffusion-based and autoregressive unified multimodal image editing models. Our experiments reveal that while off-the-shelf models struggle with abstract scribble inputs, finetuning on our synthetic dataset significantly improves their ability to generate spatially aligned and semantically consistent edits.

## 二、方法论与技术路线

### 1. 核心思想
通过**自动化的合成数据管线**，构建大规模涂鸦-文本联合编辑样本，解决缺乏训练样本的问题，使扩散模型或多模态模型能够学习如何根据涂鸦和文本进行精确的图像编辑。

### 2. 数据集构建流程

（如论文 Figure 1 所示，流程主要包括以下五步）

1. **数据源选择**：  
   - 选自 *Sketchy Database*（包含带标签的自然图像及多个人工涂鸦）。  
   - 共采样约 12,500 张图像，每张图像对应多个涂鸦。

2. **对象分割**：  
   - 使用 **Grounding DINO** 根据类别文本预测目标区域的 BBox。  
   - 用 **SAM (Segment Anything Model)** 在 BBox 内生成候选掩码，选最优 IoU。  
   - 对掩码进行 15px 膨胀，确保覆盖完整物体。

3. **输入图像生成**：  
   - 用 **LaMa inpainting 模型** 移除目标对象，生成缺失物体的背景图像，作为“编辑前图像”。

4. **文本指令生成**：  
   - 利用视觉语言模型 **Gemma 3 (4B)** 输入源图与目标图，自动生成自然语言指令，如“在图像中央添加一个红色苹果”。

5. **样本格式与划分**：  
   - 每个样本包含 `[输入图像, 目标图像, 手绘涂鸦, 文本指令]` 四元组。  
   - 共 **11,775 个样本**，分为训练/验证/测试集（约 9.4k / 1.16k / 1.17k 张图像）。  
   - 经过采样后得到约 **18,876 训练实例**。

### 3. 模型与算法细节

- **扩散模型改进版 (Image-conditioned ControlNet-Scribble)**  
  - 原 ControlNet-Scribble 不利用输入图像背景，会破坏原场景。  
  - 本文将输入图像的潜变量（VAE编码后得到的4通道latent）**与目标潜变量拼接**，并扩展 U-Net 输入通道从 4 → 8。  
  - 训练分两阶段：
    - 仅更新新层 3 轮，LR=1e-4；
    - 全模型微调 7 轮，LR=1e-5；
  - 使用 **AdamW**、权重衰减 1e-2、梯度裁剪和 **classifier-free guidance (p=0.1)**，推理时采用 50 步 **DDIM** 采样。

- **统一多模态模型 (BAGEL‑7B‑MoT)**  
  - 通过 **LoRA (rank=32, α=64)** 微调语言模型主干，LR=1e‑4（随训练余弦衰减），批次为64，训练5轮。  
  - 使用空间加权的 MSE 损失：编辑区域权重5倍以惩罚背景变动。  
  - 同样采用 classifier-free guidance 和 50 步推理。

## 六、主要结论与发现

1. **数据集贡献**：首次提供结合手绘涂鸦与自然语言指令的合成图像编辑数据，填补多模态编辑数据空白。

   - 现成模型（pretrained）在抽象涂鸦输入下失败率高，存在错位或语义错误。  
   - 在 ScribbleEdit 上微调后：
     - 扩散模型的 **FID 降至 28.47（最佳）**，背景保持更真实。  
     - BAGEL‑7B‑MoT 微调后表现整体最优，**CLIP 相似度 0.88**，说明语义理解强化。  
3. **误差分析**：  
   - 文本或涂鸦描述错误样本占比高（16% 和 34%）；  
   - 在清晰指导下，模型对涂鸦遵从度达82%，对应文本的正确率为57%。
