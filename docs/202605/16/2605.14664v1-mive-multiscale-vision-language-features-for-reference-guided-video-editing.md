## 摘要
参考引导的视频编辑以源视频、文本指令和参考图像作为输入，要求模型在忠实执行指令性编辑的同时保持原有运动和未编辑内容的完整性。现有方法主要分为两种范式，各自存在固有限制：分离式编码器在独立处理指令与视觉内容时会出现模态差距，而统一的视觉-语言编码器由于仅依赖最终层表示而丧失了细粒度空间细节。我们观察到，VLM（视觉语言模型）的各层在层次上编码互补信息——浅层捕获用于精确编辑的局部空间细节，而深层则编码用于理解指令的全局语义。基于这一洞察，我们提出了 MiVE（用于参考引导视频编辑的多尺度视觉语言特征），这是一个将 VLM 重新用作多尺度特征提取器的框架。MiVE 从 Qwen3-VL 提取层次化特征，并将其整合到统一的自注意力扩散 Transformer 中，从而消除了跨注意力设计中固有的模态不匹配。实验结果表明，MiVE 在人工偏好评测中排名最高，超越了学术方法和商业系统，达到了当前最先进的性能。

## Abstract
Reference-guided video editing takes a source video, a text instruction, and a reference image as inputs, requiring the model to faithfully apply the instructed edits while preserving original motion and unedited content. Existing methods fall into two paradigms, each with inherent limitations: decoupled encoders suffer from modality gaps when processing instructions and visual content independently, while unified vision-language encoders lose fine-grained spatial details by relying solely on final-layer representations. We observe that VLM layers encode complementary information hierarchically -- early layers capture localized spatial details essential for precise editing, while deeper layers encode global semantics for instruction comprehension. Building on this insight, we present MiVE (Multiscale Vision-language features for reference-guided video Editing), a framework that repurposes VLMs as multiscale feature extractors. MiVE extracts hierarchical features from Qwen3-VL and integrates them into a unified self-attention Diffusion Transformer, eliminating the modality mismatch inherent in cross-attention designs. Experiments demonstrate that MiVE achieves state-of-the-art performance by ranking highest in human preference, outperforming both academic methods and commercial systems.

## 一、研究背景与核心问题

- **研究领域**：参考引导视频编辑（Reference-Guided Video Editing），旨在根据文本指令与参考图像修改视频内容，同时保持未编辑部分的运动动态与语义一致性。  
- **现有问题**：
  - **模态差距问题（modality gap）**：传统方法往往采用分离式编码器，文本与视觉信息独立处理，后期的交叉注意机制无法充分融合语义。  
  - **空间细节缺失**：统一式视觉语言模型（VLM）虽能同时编码多模态输入，但通常只利用最后一层语义特征，难以保留用于精确编辑的低层空间信息。  
- **核心目标**：如何在视频编辑中既精确理解文本编辑指令，又保持时序一致性与空间细节？

## 二、方法概述：MiVE 框架

### 1. 核心思想
MiVE（**Multiscale Vision-language features for reference-guided video Editing**）通过将 **Qwen3-VL** 的多层特征重新利用为多尺度输入，在统一的自注意力扩散Transformer（Diffusion Transformer, DiT）中实现深度模态融合，从根本上消除模态不匹配。

### 2. 技术组件
MiVE 由三大模块构成：

#### （1）多层视觉语言上下文提取（Multi-Level Context Extraction）
- 从 Qwen3-VL 的**首层**和**末层**提取特征：  
  - 早期层 → 捕获细粒度空间信息（如轮廓、局部纹理）。  
  - 深层 → 捕获全局语义与任务理解。  
- 将这两层特征归一化（RMSNorm）后经线性投影拼接，再经过融合层生成条件 token（conditional tokens）`c`。  
- 这些 token 在扩散过程中保持时序不变（固定时间嵌入），作为语义锚点。

#### （2）参考感知潜编码（Reference-Aware Latent Encoding）
- 使用预训练的VAE编码源视频、目标视频与参考图像，获得潜变量：  
  - 在时间维度上**将参考帧置于首位**，以保证风格一致性。  
  - 训练时输入包括参考帧、噪声化目标帧与控制源帧；推理时利用噪声采样初始化视频潜变量。

#### （3）统一自注意力反推结构（Unified Self-Attention Backbone）
- 将条件token与视频潜token拼接为统一输入序列，通过多个DiT块进行去噪。  
- 使用**自适应层归一（AdaLN）**区分干净（参考/条件）与噪声token：
  - 固定参考与条件时间嵌入（t=0）。  
  - 按当前扩散步t调节噪声token。  
- 输出视觉token后反patch化，Decoder生成最终视频。

### 3. 训练目标
- 使用 Flow Matching 损失（Lipman et al., 2023），在统一空间中进行连续显式建模。  
- 冻结VLM和VAE，仅优化Adapter和DiT部分。

## 三、实验设计

### 1. 数据集构建
- 共计约 **30,000 对视频编辑样本**，来自两部分：
  - **OpenVE-3M 精选版**：六类编辑任务（字幕、风格迁移、添加/删除物体、背景变化等），保留24K对高质量样本。  
  - **合成人像数据**：通过分割与背景替换生成6K对样本，增强人像编辑能力。
- 所有样本经 **Qwen3-VL-8B 自动质检过滤**，评分≥8.5方可保留。

### 2. Benchmark 设计
- **自建基准集（60段720P视频）**：
  - *简单场景（30个）*：局部修改任务（物体添加/删除、局部颜色变化等）。  
  - *复杂场景（30个）*：全局风格变化、光照重构、背景替换等，考验语义与时序一致性。  
- 对部分商业视频数据因版权限制，未完全公开，但可按请求获取。

### 3. 对比方法
- 学术系统：VACE、LucyEdit、VideoCoF、Wan-Animate 等。  
- 商业系统：Kling O1。  
- MiVE 同时在有/无参考图像条件下对比不同体系方法。

### 4. 评测方式
- **自动化VLM评分（Gemini-3-Flash）**：六个维度（IA、CC、TS、PR、VA、SC），0–10制。  
- **独立验证（InternVL3.5-8B）**：用于交叉评估消除偏差。  
- **人工评分**：30名受试者，1–5评分综合人类偏好。  
- 不采用SSIM/LPIPS，认为不适合编辑类生成任务。
