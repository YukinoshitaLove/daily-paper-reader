## 摘要
我们研究了扩散变换器（Diffusion Transformers, DiTs）在图像生成任务中的离群（outlier）token。先前的研究表明，视觉变换器（Vision Transformers, ViTs）可能产生少量范数较高的 token，这些 token 会吸引不成比例的注意力，但携带的局部信息有限；然而，它们在生成模型中的作用仍有待深入探讨。我们发现，这一现象同时存在于现代表征自编码器（Representation Autoencoder, RAE）–DiT 流程的编码器与去噪器中：预训练的 ViT 编码器可能产生离群表征，而 DiT 本身也可以在内部形成离群 token，尤其是在中间层中。此外，简单地掩蔽高范数 token 并不能提升性能，这表明问题并不仅仅由少数极端值引起，而更与局部 patch 语义的破坏有关。为了解决这一问题，我们提出双阶段寄存器（Dual-Stage Registers, DSR），这是一种基于寄存器的干预方法，可同时作用于两个组件：在可行时使用训练寄存器，否则采用递归的测试时寄存器；对于去噪器，还引入了扩散寄存器。通过在 ImageNet 以及大规模文本到图像生成任务上的实验，我们的方法一致地减少了离群伪影并提升了生成质量。我们的结果强调了对离群 token 的控制是构建更强大的 DiT 模型的重要组成部分。

## Abstract
We study outlier tokens in Diffusion Transformers (DiTs) for image generation. Prior work has shown that Vision Transformers (ViTs) can produce a small number of high-norm tokens that attract disproportionate attention while carrying limited local information, but their role in generative models remains underexplored. We show that this phenomenon appears in both the encoder and denoiser of modern Representation Autoencoder (RAE)-DiT pipelines: pretrained ViT encoders can produce outlier representations, and DiTs themselves can develop internal outlier tokens, especially in intermediate layers. Moreover, simply masking high-norm tokens does not improve performance, indicating that the problem is not only caused by a few extreme values, but is more closely related to corrupted local patch semantics. To address this issue, we introduce Dual-Stage Registers (DSR), a register-based intervention for both components: trained registers when available, recursive test-time registers otherwise, and diffusion registers for the denoiser. Across ImageNet and large-scale text-to-image generation, these interventions consistently reduce outlier artifacts and improve generation quality. Our results highlight outlier-token control as an important ingredient in building stronger DiTs.

## 2. 方法论：核心思想与技术细节
### 2.1 基本思想
- 离群 token 并非仅仅是几枚数值极端的 token，而是表征了**局部 patch 语义的损坏**；
- 简单的“掩蔽高范数 token”无效，因此应从结构上引入稳定机制；
- DSR 通过在 **编码器与去噪器两个阶段** 添加寄存器 (register token)，吸收注意力汇聚点，重整局部语义结构。

### 2.2 双阶段寄存器（DSR）框架
分为以下三种寄存器策略：
1. **训练寄存器 (Trained Registers)**：在可训练的 ViT 编码器中（如 DINOv2）添加专用 register tokens，在训练阶段共同优化；
2. **测试时寄存器 (Test-Time Registers, TTR)**：对于无寄存器的预训练模型（如 SigLIP2），在推理阶段追加非训练寄存器，用来吸收异常激活；
   - 对一些严重的模型（如 SigLIP2-So400），需采用**递归 TTR**两次干预；
3. **扩散寄存器 (Diffusion Registers)**：在去噪扩散 Transformer 中引入 36 个可训练寄存器 token，参与训练但在推理时去除。

### 2.3 技术流程文字说明
- **训练阶段**：
  1. ViT 编码器加入训练或测试时寄存器；
  2. 编码器输出特征作为扩散 Transformer 输入；扩散 Transformer 内另设训练寄存器；
  3. 扩散模型预测噪声、执行去噪过程；寄存器在中层稳定注意力分布；
- **推理阶段**：
  - 去除 encoder 和 diffusion 的寄存器输出；
  - 保留正常 image tokens 进行最终解码；

  作者定义掩蔽范数高于阈值 τ 的 token 的损失函数 \(L_{mask}\)，但发现性能无提升，证明问题不止数值异常——激励 DSR 的提出。

## 6. 主要结论与发现
- 离群 token 普遍存在于：
  - ViT 编码器后期层；
  - 扩散 Transformer 中间层；
- 这些离群并非单纯极值，而反映特征语义损坏；
- 简单掩蔽无效；
- DSR 能有效抑制离群，恢复局部语义稳定：
  - **SigLIP2-B 上 FID 从 5.89 ↓至 4.58**；
  - **GenEval 得分从 42.6 ↑至 46.6**；
  - 训练效率显著提升，可在原始 Epoch 的四分之一内达到相当质量；
- 控制离群 token 被确认为提升稳定性与生成质量的重要技术要素。
