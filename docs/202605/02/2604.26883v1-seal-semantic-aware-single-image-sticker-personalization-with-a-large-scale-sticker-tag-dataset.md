## 摘要
在基于扩散的个性化文本到图像生成中，从单张参考图像合成目标概念具有挑战性，特别是在贴纸个性化任务中，提示往往需要显式的属性编辑。仅有一个参考图像时，测试时微调（TTF）方法容易过拟合，导致出现\textit{视觉纠缠}——背景伪影被吸收进学习到的概念中，以及\textit{结构僵化}——模型记忆了参考图像特定的空间布局，从而失去上下文控制能力。为解决这些问题，我们提出了\textbf{语义感知的单图像贴纸个性化}（\textbf{SEAL}），这是一种可插拔、与架构无关的适配模块，可无缝集成到现有的个性化生成流程中，而无需修改其基于 U-Net 的扩散主干网络。SEAL 在嵌入自适应过程中引入了三项关键组件：（1）语义引导的空间注意力损失，（2）分裂-合并令牌策略，以及（3）结构感知的层级限制。为了支持具有属性级控制能力的贴纸领域个性化，我们构建了 StickerBench——一个带有结构化标签的大规模贴纸图像数据集，遵循六属性架构（外观、情绪、动作、相机构图、风格、背景）。这些注释为在保持目标身份一致的情况下控制不同上下文提供了一致的接口，从而系统性地评估身份解缠与上下文可控性。实验结果表明，SEAL 在保持上下文可控性的同时，持续提升了身份保持能力，凸显了在测试时自适应过程中显式空间与结构约束的重要性。代码、StickerBench 数据集以及项目页面将公开发布。

## Abstract
Synthesizing a target concept from a single reference image is challenging in diffusion-based personalized text-to-image generation, particularly for sticker personalization where prompts often require explicit attribute edits. With only one reference, test-time fine-tuning (TTF) methods tend to overfit, producing \textit{visual entanglement}, where background artifacts are absorbed into the learned concept, and \textit{structural rigidity}, where the model memorizes reference-specific spatial configurations and loses contextual controllability. To address these issues, we introduce \textbf{SE}mantic-aware single-image sticker person\textbf{AL}ization (\textbf{SEAL}), a plug-and-play, architecture-agnostic adaptation module that integrates into existing personalization pipelines without modifying their U-Net-based diffusion backbones. SEAL applies three components during embedding adaptation: (1) a Semantic-guided Spatial Attention Loss, (2) a Split-merge Token Strategy, and (3) Structure-aware Layer Restriction. To support sticker-domain personalization with attribute-level control, we present StickerBench, a large-scale sticker image dataset with structured tags under a six-attribute schema (Appearance, Emotion, Action, Camera Composition, Style, Background). These annotations provide a consistent interface for varying context while keeping target identity fixed, enabling systematic evaluation of identity disentanglement and contextual controllability. Experiments show that SEAL consistently improves identity preservation while maintaining contextual controllability, highlighting the importance of explicit spatial and structural constraints during test-time adaptation. The code, StickerBench, and project page will be publicly released.

## 一、研究背景与问题动机

- **研究背景**：  
  扩散模型（Diffusion Models）在文本到图像生成中表现出色，但在“个性化生成”（Personalization）场景中，尤其是“单图像”任务时仍面临严重挑战。贴纸个性化需求中，用户往往只提供一张参考图像并要求修改其属性（情绪、动作、背景等），因此缺乏多视角样本来支撑模型区分“主体身份”和“环境噪声”。

- **核心问题**：  
  - 单图像场景下的**过拟合风险**：模型仅从单张参考学习，易将背景或布局特性误吸入概念表示。  
  - 出现两种主要失效模式：  
    1. **视觉纠缠（Visual Entanglement）**：主体与背景绑定，背景伪影出现在生成结果中。  
    2. **结构僵化（Structural Rigidity）**：模型记忆参考图像的空间构型，缺乏灵活的上下文可控性。  

- **研究动机**：  
  希望在**无需更改扩散模型主干结构**的前提下，通过轻量的“语义感知适配机制”，改进单图像个性化生成的可控性与泛化能力。

## 二、方法论概述与核心技术

### 总体思想
提出一种**可插拔、架构无关的适配模块**——**SEAL（Semantic-aware single-image sticker personalization）**。它在测试时微调阶段调节概念嵌入的学习过程，从语义层面和结构层面约束模型，避免过拟合。

### 模块组成（见图 2）
1. **语义引导空间注意力损失（Semantic-guided Spatial Attention Loss）**  
   - 基于 Segment Anything Model (SAM) 提取目标对象的掩码，约束概念 token 的 cross-attention 分布仅聚焦在对象区域。  
   - 包含两项损失：  
     - **背景抑制损失**：惩罚注意力泄漏到背景。  
     - **对齐损失（基于 Soft-IoU）**：鼓励注意力区域与目标掩码重合。  
   - 通过归一化与平方强化处理提高稳定性和边界敏感度。

2. **分裂-合并令牌策略（Split-merge Token Strategy）**  
   - 将单个概念嵌入拆分为 K 个辅助嵌入（实验中 K=5），并行优化后再平均合并。  
   - 目的：
     - 稳定单样本训练过程。  
     - 捕捉多样属性。  
     - 避免嵌入陷入局部最优或单一模式。

3. **结构感知层限制（Structure-aware Layer Restriction）**  
   - 仅在 U-Net 的**语义层（中间层）**上施加空间注意力约束，避开输入端的低层（容易编码结构纹理）和输出端的细节层。  
   - 通过层索引范围 \([|L|/4, 3|L|/4]\) 定义选取策略，保持架构无关性。

### 总体目标函数
\[
L_{total} = L_{diffusion} + \lambda_{spatial} L_{spatial}
\]
- 其中 \(L_{diffusion}\) 为标准噪声预测目标（与扩散模型一致），\(L_{spatial}\) 为多层平均的空间约束项。  
- 该模块可无缝嵌入已有 Test-time Fine-tuning (TTF) 方法（如 DreamBooth、CoRe、Custom Diffusion 等）。
