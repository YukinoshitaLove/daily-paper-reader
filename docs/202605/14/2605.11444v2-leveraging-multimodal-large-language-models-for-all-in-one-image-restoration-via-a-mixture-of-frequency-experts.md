## 摘要
全能图像复原旨在使用统一框架从受到多种未知退化影响的输入中恢复干净的图像。近期方法通过识别退化特征来引导复原过程，表现出强大的性能。然而，许多方法将退化视为离散类别，这限制了其建模复合退化中连续关系结构的能力。为解决这一问题，我们提出了一种由多模态大语言模型（MLLM）引导的图像复原框架，利用多模态嵌入指导低层次复原。具体而言，我们通过MLLM引导融合模块（MGFB）将MLLM提取的特征注入到编码器-解码器架构中，以增强感知退化的表示。此外，我们引入了一个频率专家混合（MoFE）模块，该模块根据MLLM引导的上下文线索自适应地组合频率专家。为了进一步改进专家路由，我们设计了一个由MLLM引导的路由器，并加入关系对齐损失，以鼓励路由模式与退化输入的嵌入空间关系保持一致。大量的基准实验表明，所提出的方法在各种复原场景下均取得了优异的性能，在具有挑战性的CDD11数据集上建立了新的最新结果，性能相比先前方法提升最高达1.35 dB。

## Abstract
All-in-one image restoration seeks to recover clean images from inputs affected by diverse and unknown degradations using a unified framework. Recent methods have shown strong performance by identifying degradation characteristics to guide the restoration process. However, many of them treat degradations as discrete categories, which limits their ability to model the continuous relational structure that arises in composite degradations. To address this issue, we propose a multimodal large language model (MLLM)-guided image restoration framework that exploits multimodal embeddings as guidance for low-level restoration. Specifically, MLLM-derived features are injected into an encoder-decoder architecture through an MLLM-guided fusion block (MGFB) to enhance degradation-aware representations. In addition, we incorporate a mixture-of-frequency-experts (MoFE) module that adaptively combines frequency experts using MLLM-guided contextual cues. To further improve expert routing, we design an MLLM-guided router with a relational alignment loss that encourages routing patterns consistent with the embedding-space relationships of degraded inputs. Extensive experiments on multiple benchmarks show that the proposed method achieves strong performance across diverse restoration settings and establishes a new state of the art on the challenging CDD11 dataset, outperforming previous methods by up to 1.35 dB.

- **研究主题**：全能图像复原（All-in-One Image Restoration, AIO-IR）旨在以统一框架处理由多种未知退化（如噪声、模糊、雾化、雨雪等）破坏的图像。  
- **问题痛点**：

  - 现有的统一复原方法（如 PromptIR、OneRestore 等）往往将退化类型视为**离散类别**，无法建模退化间的**连续谱关系**。  
- **研究动机**：  
  - 现实中的图像退化常呈现“连续性与叠加性”特征（如低光 + 雾 + 雨），需要能够理解退化组合规律的模型。

- **核心目标**：构建一个由 MLLM 引导的统一图像复原框架，突破离散分类约束，实现对复杂、混合退化的稳健建模。

## 2. 方法论：核心思想与关键技术

### （1）总体框架概览
论文提出 **MMFE-IR（MLLM-guided Mixture-of-Frequency Experts for Image Restoration）**，其核心包括三大模块：
1. **MLLM引导融合模块（MGFB）**：在编码器–解码器结构中注入MLLM特征，实现语义引导下的退化感知表示；
2. **频率专家混合模块（MoFE）**：在频域上分解多频特征，利用专家混合结构自适应处理不同退化类型；
3. **关系对齐路由器（MLLM-guided Router）**：用MLLM嵌入间相似性约束专家组合模式，使复原过程保持语义连续性。

### （2）输入与多模态嵌入
- 使用预训练 MLLM（Qwen2.5-VL）提取三种嵌入：
  - **EImage**：由视觉token池化获得的图像特征；
  - **EJoint**：视觉–语言联合嵌入；
  - **EAnswer**：语言解码器输出的高层语义特征。
- 模型以**视觉问答式 prompt**（如“What kinds of degradation in this image?”）提取语义向量。

### （3）MLLM-Guided Fusion Block (MGFB)
- 作用：解决高层语义与低层视觉特征之间的表征差距。
- 机制：
  - 在**编码器**阶段使用 EImage 增强结构抽取；
  - 在**解码器**阶段使用 EJoint 强化语义感知重建；
  - 采用**交叉注意力机制**实现信息融合：
    \[
    F_l^{'} = F_l + \text{Softmax}\left(\frac{QK^\top}{\sqrt{d}}\right)V
    \]
    其中 Q 来自视觉特征，K、V 来自 MLLM 嵌入。

### （4）Mixture of Frequency Experts (MoFE)
- 启发：不同退化对应不同的频率特征（如噪声为高频、雾为低频）。
- 流程：
  1. 将输入图像转换至频域（DWT小波分解）；
  2. 忽略低频分量，仅建模高频部分；
  3. 通过多个轻量卷积专家 \( M_i \) 处理高频子带；
  4. 路由器根据 EAnswer 的语义生成权重：
     \[
     S = \sigma(W_r E_{Answer})
     \]
  5. 输出为各专家加权和：\(\sum_i S_i \cdot M_i(F_{HF})\)。

### （5）关系一致性损失 (MLLM-Guided Loss)
- 目标：保持路由器输出与 MLLM 嵌入空间的语义关系一致。
- 定义：
  \[
  L_{MGL} = \| \text{Sim}(E_{Answer}) - \text{Sim}(S) \|_1
  \]
  其中 Sim(·) 表示批内样本对的余弦相似度。
- 总损失：
  \[
  L_{total} = L_{rec} + \lambda L_{MGL}
  \]
  (γ=0.1)

### （6）训练策略
- 采用**渐进式多尺度训练**：逐步提升输入分辨率（128² → 224²），使模型平滑适应高分辨率复原任务。

## 8. 不足与局限

- **算力成本偏高**：需要多GPU和长时训练，MLLM嵌入计算存在延迟。
