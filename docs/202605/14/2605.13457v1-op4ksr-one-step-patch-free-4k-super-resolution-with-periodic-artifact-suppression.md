## 摘要
基于扩散模型的真实世界图像超分辨率（Real-ISR）在感知质量上取得了显著进展；然而，直接将图像超分辨率提升至4K仍受限于极高的内存消耗。因此，现有方法多采用基于图像块（patch-based）的推理方式，这会牺牲全局上下文信息，并引入语义混淆、空间不一致以及严重的推理延迟。我们提出了OP4KSR，一种构建于强大Flux骨干网络之上的单步、无图像块（patch-free）4K超分辨率方法。通过利用极度压缩的F16 VAE，OP4KSR能够在实际GPU算力预算下实现可行的4K推理，同时保持全局空间-语义一致性，并实现高效推理。然而，将这种单步架构直接应用于4K的过程中会本质性地引发周期性伪影。我们将这一问题追溯至RoPE基频配置不匹配和token内部空间模糊性，这两者由于缺乏迭代细化而进一步加剧。为抑制这些伪影，我们结合了RoPE基频重缩放（RFR）与基于自相关的周期性损失（$\mathcal{L}_\text{AP}$）。此外，我们构建了专用的训练数据集以及三个评测基准（一个合成数据与两个真实场景）以推动4K超分辨率研究。大量实验表明，OP4KSR在保持高效推理的同时获得了具有竞争力的感知质量，仅使用单张NVIDIA H20 GPU即可在5.75秒内生成$4096\times4096$的输出。

## Abstract
Diffusion-based real-world image super-resolution (Real-ISR) has achieved remarkable perceptual quality; however, directly super-resolving images to 4K remains limited by extreme memory consumption. Consequently, prior methods adopt patch-based inference, sacrificing global context and introducing semantic confusion, spatial inconsistency, and severe latency. We propose OP4KSR, a one-step patch-free 4K SR approach built upon the powerful Flux backbone. By leveraging the extreme-compression F16 VAE, OP4KSR makes 4K SR inference tractable under practical GPU budgets, preserving global spatial-semantic coherence while enabling highly efficient inference. However, adapting this one-step architecture intrinsically triggers severe periodic artifacts. We trace this to a RoPE base frequency allocation mismatch and intra-token spatial ambiguity, both exacerbated by the lack of iterative refinement. To suppress these artifacts, we couple RoPE base frequency rescaling (RFR) with an autocorrelation-based periodicity loss ($\mathcal{L}_\text{AP}$). Furthermore, we curate a dedicated training dataset alongside three benchmarks (one synthetic and two real-world) to advance 4K SR research. Extensive experiments demonstrate that OP4KSR achieves competitive perceptual quality with efficient inference, generating a $4096\times4096$ output in only 5.75 seconds on a single NVIDIA H20 GPU.

- **研究动机**：当前基于扩散模型的真实场景图像超分辨率（Real-ISR）在感知质量上取得突破，但当分辨率扩展至 **4K（4096×4096）** 时，模型面临**显著的内存消耗**与计算负担。

- **总体意义**：论文旨在实现**单步、无分块的真实场景4K超分辨率**，兼顾全局一致性与极低算力成本。

## 二、方法论：核心思想与技术设计

- **方法名称**：OP4KSR（One-Step Patch-Free 4K Super-Resolution）。
- **核心思想**：在 Flux 框架上构建能够直接从低分辨率（LR）图像生成高分辨率（HR）图像的**单步推理架构**，并通过高压缩编码与伪影抑制策略解决内存与视觉失真问题。

### 关键技术组件

1. **F16 极度压缩 VAE 编码器**
   - 相比标准 F8 VAE，压缩率更高，显著减少显存占用，使 4K SR 在单张 GPU 上运行可行。
   - 结合 UltraFlux 架构，在保持全局语义连续性的同时提升推理速度。

2. **中间时刻对齐（Mid-timestep alignment）**
   - 借鉴 OMGSR 思路，在扩散流匹配框架中将输入 LR 图像的潜变量对齐到**中间时刻 t=0.3**，消除扩散模型预期噪声与实际图像之间的分布差异。

3. **周期伪影抑制机制**
   - 发现单步推理在光滑区域（如水面、人脸）产生**周期性网格伪影**，其根因包括：
     - **RoPE（旋转位置编码）基频不匹配**：默认频率 θ=10,000 导致相邻 token 难以区分。
     - **Unpack 操作引发空间盲区**：子像素通道共享位置编码，导致重复图案。
   - 为此引入两种抑制策略：
     - **RoPE 基频重标定（RFR）**：将 θ 降至 100，使相邻位置具有更显著相位差，恢复局部结构辨识能力。
     - **自相关周期损失（$\mathcal{L}_{AP}$）**：在像素域划分4个象限，计算不同空间间隔（8、16、24、32、40像素）的二维自相关差异，惩罚周期性重复模式，以物理方式抑制伪影。

4. **总体优化目标**
   - 训练时联合损失包括：VAE 对齐损失（$\mathcal{L}_{vae}$）、像素重建损失（MSE）、感知损失（LPIPS）、频域损失（Wav）、对抗损失（ADV）以及周期损失（$\mathcal{L}_{AP}$）。
