## 摘要
自然图像是连续的，但大多数生成模型在离散网格上进行合成，限制了生成分辨率的灵活性。连续神经场（Neural Fields）能够实现无分辨率限制的渲染，但以往的方法仅在解码阶段通过插值模块引入连续性，使得生成的潜在空间仍然是离散且偏向重建任务的。我们提出了 RaPD（Resolution-agnostic Pixel Diffusion），该方法在连续的 Neural Image Field（NIF）潜在空间中执行扩散。RaPD 通过语义表征引导（Semantic Representation Guidance）来实现面向生成的潜在学习，并使用坐标查询注意力渲染器（Coordinate-Queried Attention Renderer）进行基于坐标的、具备尺度感知的渲染，从而弥合了重建与生成之间的差距。一个经去噪的潜在表示可以仅通过更改查询坐标在任意分辨率下进行渲染，同时保持固定的扩散计算成本。实验结果表明，该方法在生成质量和分辨率可扩展性方面均优于现有方法。

## Abstract
Natural images are continuous, yet most generative models synthesize them on discrete grids, limiting resolution-flexible generation. Continuous neural fields enable resolution-free rendering, but prior methods introduce continuity only at the decoding stage as an interpolation module, leaving the generative latent space discretized and reconstruction-oriented. We propose RaPD (Resolution-agnostic Pixel Diffusion), which performs diffusion in a continuous Neural Image Field (NIF) latent space. RaPD bridges this reconstruction-generation gap with Semantic Representation Guidance for generation-aware latent learning and a Coordinate-Queried Attention Renderer for coordinate-conditioned, scale-aware rendering. A single denoised latent can be rendered at arbitrary resolutions by changing only the query coordinates, keeping diffusion cost fixed. Experiments demonstrate superior generation quality and resolution scalability.

## 一、研究问题与背景

- **研究动机**：  
  现有主流图像生成模型（如 VAE-based latent diffusion、pixel diffusion）大多在**离散网格或压缩潜空间**中操作，其空间分辨率固定，生成图像的清晰度和细节受限。  
  尽管**连续神经场（Neural Image Field, NIF）**允许以任意分辨率渲染图像，但以往方法仅在解码阶段通过插值实现连续性，潜空间仍是离散的、面向重建而非生成。

- **核心问题**：  
  如何建立一种能在**连续潜空间中进行扩散**的生成框架，使模型获得：
  - 与分辨率无关的生成能力；
  - 高分辨率的一致性与细节保真；
  - 与语义一致的内容生成。

- **研究目标**：  
  将扩散过程引入连续隐式图像表示，使模型可以在固定计算成本下生成任意分辨率的高质量图像。

## 二、方法论概述

### 1. 总体框架：RaPD（Resolution-Agnostic Pixel Diffusion）

RaPD 在 **连续神经图像场潜空间（NIF latent space）**中执行扩散，其结构由三个核心组件构成：

- **(1) Semantic Representation Guidance (SRG)**  
  - 用于将原本面向重建的 NIF 潜表示转化为面向生成的语义潜空间。  
  - 包含两部分机制：  
    - **语义蒸馏（Semantic Distillation）**：从冻结的 DINOv2 ViT-B/14 教师模型中提取语义特征，通过余弦相似和距离矩阵相似损失指导潜表示学习，保证语义一致性。  
    - **潜密度感知时间步调整（Latent-Density-Aware Timestep Shift）**：在扩散训练中根据高密度连续潜表示调整时间步采样分布，增强模型在早期重噪阶段的稳定性。

- **(2) Coordinate-Queried Attention Renderer (CQAR)**  
  - 一种基于查询坐标的注意力式隐式解码器。  
  - 传统 LIIF 使用点式 MLP、CLIF 使用浅层 CNN，而 CQAR 使用 **窗口化 Transformer 注意力机制**，允许跨像素的语义推理。  
  - 构造过程：
    - 为目标坐标位置构造 token：结合最近邻潜特征、相对偏移、像素尺度和绝对坐标；
    - 对输出坐标 token 执行 Swin-style window attention，实现局部到全局的语义一致渲染。

- **(3) 流配合训练（Flow Matching Diffusion）**  
  - 将流匹配方法应用于连续潜空间，学习高维潜向量的“去噪速度场”，避免传统离散扩散无法泛化至新分辨率的限制。

### 2. 算法流程（两阶段训练）

**Stage-1：语义引导的多分辨率 NIF 学习**  
1. 编码器 Eϕ（EDSR 架构，无下采样）将输入图像映射为高密度潜空间 z。  
2. CQAR 渲染器生成目标坐标下的 RGB 输出。  
3. 同时优化重建损失（L1 + LPIPS）与语义蒸馏损失，使用梯度自适应权重平衡。

**Stage-2：扩散训练（Flow Matching）**  
1. 固定 Eϕ 与 CQAR。  
2. 对归一化潜表示 ˜z 进行流匹配训练，使用 PixGen 风格的 DiT 架构。  
3. 条件输入为 Qwen3-1.7B 文本嵌入，优化目标为去噪速度预测。  

**推理阶段**  
- 仅需一次扩散生成潜表示，再通过 CQAR 根据任意坐标网格进行渲染，扩散成本固定，渲染成本随查询坐标数线性变化。

## 三、实验设计

### 1. 数据集与评测基准

- **训练数据**：
  - 约 3600 万张来自 **BLIP3o** 数据集的图像用于预训练；
  - 额外 6 万张高质量 512×512 图像用于微调。
- **评测基准**：
  - **GenEval**（用于评估文本-图像一致性与构成能力）；
  - **DPG-Bench**（测量对复杂长文本描述的生成能力）；
  - 使用 **rFID** 衡量重建质量。

### 2. 对比方法

- **Latent diffusion 模型**：LDM、SD3、FLUX.1-dev、PixArt-α、DALL-E 3。  
- **Pixel diffusion 模型**：PixelFlow、PixNerd、DeCo、PixelGen。  
- 复现基线以统一环境和随机种子，保证公平性。

## 六、主要结论与发现

- **性能提升显著**：  
  - GenEval 总分 0.85，超过 PixelGen (0.79)、DeCo (0.83) 等基线；  
  - DPG-Bench 平均得分 81.6，优于所有开源 pixel-diffusion 模型。  
- **分辨率无关生成**：  
  - 能在 512→4096 像素范围内生成高保真图像，仅改变坐标查询而不增加扩散成本。  
- **语义一致性强化**：  
  - 生成结果在多物体、计数、颜色与位置关系任务上表现突出；  
  - DINOv2 蒸馏提供更稳定的全局语义对齐。  
- **高效率**：  
  - 与 PixelGen/DeCo 随分辨率平方增长的计算量不同，RaPD 的推理时间近似恒定。

## 七、方法与实验亮点

- **创新性**：首个在连续神经图像场潜空间中执行扩散的模型，实现真正意义上的分辨率无关生成。  
- **语义指导机制**：将 DINOv2 特征结构融入潜空间，使连续潜变量同时具备像素与语义表达能力。  
- **解码设计优雅**：CQAR 通过坐标查询注意力实现跨像素语义推理，改进了传统隐式解码器的局限。  
- **成本固定性**：无需随输出分辨率调整扩散步数，实现生成成本与分辨率解耦。  
- **实验全面、复现公平**：覆盖多基准和消融分析。

- **训练数据规模有限**：仅使用 3600 万张图像，明显小于 SD3、FLUX 等商用巨模，生成质量上限仍受限。
