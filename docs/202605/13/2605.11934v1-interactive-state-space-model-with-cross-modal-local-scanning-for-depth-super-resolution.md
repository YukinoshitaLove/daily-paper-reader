## 摘要
引导式深度超分辨率（Guided Depth Super-Resolution，GDSR）利用高分辨率（HR）RGB图像引导，从低分辨率（LR）输入中重建高分辨率深度图。现有方法要么独立建模每种模态，要么依赖计算开销巨大的、具有二次复杂度的注意力机制，从而阻碍了高效且语义交互的联合表示的建立。本文观察到，不同模态的特征图在特征提取过程中呈现出语义层面的相关性。这激发我们开发一种更灵活的方法，使模态间能够进行稠密且具语义感知的深层交互。为此，我们提出了一种以交互式状态空间模型（Interactive State Space Model）为核心的新型GDSR框架。具体而言，我们设计了一种跨模态局部扫描机制，使RGB与深度特征之间能够进行细粒度的语义交互。借助Mamba架构，我们的框架以线性复杂度实现全局建模。此外，我们引入了跨模态匹配变换模块，以利用两种模态的代表性特征提高交互建模质量。大量实验结果表明，我们的方法在性能上具有与当前最先进方法竞争的水准。

## Abstract
Guided depth super-resolution (GDSR) reconstructs HR depth maps from LR inputs with HR RGB guidance. Existing methods either model each modality independently or rely on computationally expensive attention mechanisms with quadratic complexity, hindering the establishment of efficient and semantically interactive joint representations. In this paper, we observe that feature maps from different modalities exhibit semantic-level correlations during feature extraction. This motivates us to develop a more flexible approach enabling dense, semantically-aware deep interactions between modalities. To this end, we propose a novel GDSR framework centered around the Interactive State Space Model. Specifically, we design a cross-modal local scanning mechanism that enables fine-grained semantic interactions between RGB and depth features. Leveraging the Mamba architecture, our framework achieves global modeling with linear complexity. Furthermore, a cross-modal matching transform module is introduced to enhance interactive modeling quality by utilizing representative features from both modalities. Extensive experiments demonstrate competitive performance against state-of-the-art methods.

## 1. 核心问题与研究背景

- **现状问题**：
  - 现有方法多数**单独建模每个模态**（RGB 与深度），缺乏真正的语义层交互；
  - 或依赖复杂的注意力机制，导致**计算复杂度呈平方级增长**；
  - 模态间交互常通过简单操作（加和、拼接），语义关联不足，无法充分利用跨模态信息。
- **研究动机**：观察发现 RGB 与深度特征在提取过程中具有明显的**语义级相关性**，因此希望利用这种相关性设计**高效的跨模态交互结构**，实现更精细的语义融合与高效的全局建模。

## 2. 方法论与核心思想

### 2.1 总体框架
论文提出一个基于 **Mamba 状态空间架构** 的新型 GDSR 框架，核心由两部分组成：
- **交互式状态空间模型 (Interactive State Space Model, ISSM)**  
- **跨模态匹配变换模块 (Cross-Modal Matching Transform, CMMT)**  

总体采用 U-shaped 编码-解码结构，通过堆叠多个 Mamba Block 完成多尺度上下文提取与融合。

### 2.2 关键机制：交互式状态空间模型 (ISSM)
- 基于 Mamba 架构进行**线性复杂度的全局建模**；
- 包含新的 **跨模态局部扫描机制 (Cross-Modal Local Scanning, CMLS)**，用于 RGB 和深度特征的细粒度语义交互；
- **主要步骤**：
  1. 输入特征经层归一化（LN）、线性映射和深度卷积处理；
  2. 通过 **重排（rearrange）** 通道以聚集语义相关的特征；
  3. 进行 **patchify** 操作，引入局部归纳偏置；
  4. 应用 **S6 selective scanning**（Mamba 的核心扫描单元），实现全局依赖建模。
- 结果通过 **门控机制（gating）**与线性映射进一步增强表征能力，实现语义相关特征的强化与不相关特征的抑制。

### 2.3 跨模态匹配变换模块 (CMMT)
- 作为 FFN（前馈网络）后的补充操作，使得两模态之间进行语义匹配与特征融合；
- 主要流程：
  1. 对 RGB 与深度特征卷积增强局部上下文；
  2. 展平特征后计算**相似度矩阵 M**（基于欧氏距离）；
  3. 选择相似度最高的特征向量并排序；
  4. 从对方模态（如 RGB）中选取 Top-C/r 的相关特征进行拼接；
  5. 通过卷积与 Sigmoid 门控实现筛选与融合；
- **创新点**：相比传统注意力机制，CMMT 在融合上更高效，且能**自适应选取最相关的跨模态特征**。

### 2.4 总体输出
- 最终输出为：

  D_{hr} = D_{up} + R

  其中 \( D_{up} \) 为上采样的初始深度图，\( R \) 为网络预测的残差修正图。
