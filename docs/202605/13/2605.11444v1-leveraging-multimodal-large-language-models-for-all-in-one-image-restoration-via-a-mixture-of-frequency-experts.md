## 摘要
全能图像复原的目标是在统一框架下，从受到多种未知退化影响的输入中恢复出清晰的图像。最新方法通过识别退化特征来指导复原过程，已展现出良好性能。然而，许多方法将退化视为离散类别，这限制了其对复合退化中连续关系结构的建模能力。为了解决这一问题，我们提出了一种由多模态大型语言模型（MLLM）引导的图像复原框架，利用多模态嵌入作为低层复原的指导。具体而言，MLLM 提取的特征通过 MLLM 引导融合块（MGFB）注入编码器-解码器架构中，以增强具备退化感知的表示。此外，我们引入了频率专家混合（MoFE）模块，基于 MLLM 引导的上下文线索自适应地组合不同频率专家。为了进一步改进专家路由，我们设计了带有关系对齐损失的 MLLM 引导路由器，鼓励路由模式与退化输入在嵌入空间中的关系保持一致。在多个基准测试上的大量实验结果表明，所提出的方法在各种复原场景中均表现出优异性能，并在具有挑战性的 CDD11 数据集上建立了新的最先进成果，较之前方法最高提升 1.35 dB。

## Abstract
All-in-one image restoration seeks to recover clean images from inputs affected by diverse and unknown degradations using a unified framework. Recent methods have shown strong performance by identifying degradation characteristics to guide the restoration process. However, many of them treat degradations as discrete categories, which limits their ability to model the continuous relational structure that arises in composite degradations. To address this issue, we propose a multimodal large language model (MLLM)-guided image restoration framework that exploits multimodal embeddings as guidance for low-level restoration. Specifically, MLLM-derived features are injected into an encoder-decoder architecture through an MLLM-guided fusion block (MGFB) to enhance degradation-aware representations. In addition, we incorporate a mixture-of-frequency-experts (MoFE) module that adaptively combines frequency experts using MLLM-guided contextual cues. To further improve expert routing, we design an MLLM-guided router with a relational alignment loss that encourages routing patterns consistent with the embedding-space relationships of degraded inputs. Extensive experiments on multiple benchmarks show that the proposed method achieves strong performance across diverse restoration settings and establishes a new state of the art on the challenging CDD11 dataset, outperforming previous methods by up to 1.35 dB.

## 2. 方法论与技术框架

### 2.1 总体思路
- 本文提出 **MLLM 引导的频率专家混合（MMFE-IR）框架**，旨在利用多模态语义特征指导像素级复原。
- 方法结合 **多模态嵌入（视觉 + 语言）指导的特征融合模块（MGFB）** 与 **频率域专家混合模块（MoFE）**。
- 通过一个带有 **关系对齐损失 (Relational Alignment Loss, LMGL)** 的**MLLM引导路由器**，保持退化样本在专家分配上的语义连续性。

### 2.2 关键模块与算法流程

- **MLLM 特征提取**
  - 使用预训练的 Qwen2.5-VL 模型，通过视觉问答（VQA）形式提取三类嵌入：
    - `EImage`：视觉特征（来自图像解码器最终层）
    - `EJoint`：视觉-语言联合特征（跨模态融合）
    - `EAnswer`：语言输出的语义特征（文本层信息）
  
- **MLLM-Guided Fusion Block (MGFB)**
  - 将以上嵌入通过跨注意力机制注入编码器解码器：
    - 编码器使用 `EImage` 以强化结构特征提取；
    - 解码器使用 `EJoint` 以提供语义退化感知；
  - 融合公式（简述）：  
    `Fl_MGFB = Fl + Attn(Fl, E)`  
    其中 `Fl` 为原始层特征，`E` 为对应的 MLLM 嵌入。

- **Mixture-of-Frequency-Experts (MoFE)**
  - 在频率域而非空间域操作，通过离散小波变换 (DWT) 将输入图像分解为多个子带（LH、HL、HH）。
  - 定义 N 个频率专家，每个专家负责不同频率成分（高频纹理/结构等）。
  - 路由权重由 `EAnswer` 决定：
    - 路由生成公式： `S = σ(Wr * EAnswer)`
    - 输出为加权专家聚合：`Σ Si * Mi(Fl_HF)`
  - 最终将聚合结果重新与空间特征融合，提升高低频信息整合。

- **MLLM-guided Router Optimization**
  - 通过关系对齐损失保证路由输出在样本间保持与 MLLM 嵌入的语义相似性：
    - 损失函数 `LMGL = ||Sim(EAnswer) - Sim(S)||1`
  - 总损失 `Ltotal = Lrec + λ * LMGL`，其中 `Lrec` 为重建损失（L1+傅里叶域混合）。

- **训练策略**
  - 使用渐进多尺度训练：分阶段提高输入分辨率、减小 batch size，提高稳定性与跨尺度泛化。

## 7. 方法优点与创新点

- **多模态连续退化建模**：利用 MLLM 嵌入空间自然表示退化间连续关系，解决传统离散化缺陷。  
- **频率域专家混合**：创新性地在频率域操作 MoE，有效分离不同退化频率特征。  
- **跨层引导融合（MGFB）**：让高层语义直接指导低层像素复原，弥合语义与视觉间鸿沟。  
- **关系对齐损失 (LMGL)**：保证路由语义一致性，促进专家选择的结构化优化。  
- **渐进多尺度训练策略**：提升稳定性与高分辨率泛化能力。  
- **高效性**：增加少量参数即可获得明显 PSNR 提升，且小型 MLLM 同样有效。

## 8. 局限与不足

- **实验未涵盖视频复原或时序一致性任务**，暂限于静态图像。
