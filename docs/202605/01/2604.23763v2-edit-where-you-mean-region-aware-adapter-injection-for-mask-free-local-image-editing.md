## 摘要
大型扩散变换器（DiTs）能够很好地执行全局编辑指令，但在进行局部编辑时常常出现泄漏到无关区域的情况，因为联合注意力结构中没有显式通道指示网络编辑的位置。我们提出了 AdaptEdit，这是一种协同训练的、具备指令与区域感知能力的适配器框架，可将冻结的 DiT 改造为精确的局部编辑器，而无需修改其主体权重。在每个变换器模块中，一个轻量级的 Block Adapter 注入结构化的条件流，将“编辑什么”（指令语义）与“编辑哪里”（空间掩码）进行分解；学习得到的 SpatialGate 有选择地将适配器信号引入编辑区域，同时使图像的其他部分与源图像保持几乎相同；而 Region-Aware Loss 则将训练目标集中在发生改变的像素上。由于这些组件使主体的内部表示在端到端训练中具备掩码感知能力，因此与编辑器联合训练的轻量级 MaskPredictor 头可以直接根据指令和源图像确定编辑区域——从而在部署时无需用户提供掩码。我们在两个互补的基准上进行了评估：MagicBrush（成对的真实目标）用于测量像素级保持与编辑精度；Emu-Edit Test（无真实图像，涵盖 9 种不同编辑类别）用于压力测试指令跟随与跨编辑类型的泛化能力。在两个基准上，AdaptEdit 均取得了最新的最佳成果，同时超越了无掩码与理想掩码基线。我们还进行了七种变体的消融实验，清晰地分离了每个组件的贡献。

## Abstract
Large diffusion transformers (DiTs) follow global editing instructions well but consistently leak local edits into unrelated regions, because joint-attention architectures offer no explicit channel telling the network where to apply the edit. We introduce AdaptEdit, a co-trained, instruction- and region-aware adapter framework that retro-fits a frozen DiT into a precise local editor without modifying its backbone weights. A lightweight Block Adapter at every transformer block injects a structured condition stream that factorizes what to edit (instruction semantics) from where to edit (spatial mask); a learned SpatialGate routes the adapter signal selectively into the edit region while keeping the rest of the image near-identical to the source; and a Region-Aware Loss focuses the training objective on the changing pixels. Because these components make the backbone's internal representation mask-aware end-to-end, a thin MaskPredictor head trained jointly with the editor can ground the edit region directly from the instruction and source image -- eliminating any user-mask requirement at deployment. We evaluate on two complementary benchmarks: MagicBrush (paired ground-truth targets) to measure pixel-level preservation and edit accuracy, and Emu-Edit Test (no ground-truth images, 9 diverse edit categories) to stress-test instruction following and generalization across edit types. On both, AdaptEdit achieves state-of-the-art results, simultaneously outperforming mask-free and oracle-mask baselines. A seven-variant ablation cleanly isolates the contribution of each component.

## 一、研究背景与核心问题

- **研究动机：**  
  随着 **扩散 Transformer（Diffusion Transformer, DiT）** 在图像生成与编辑任务中表现出色，其在执行全局编辑指令时能保持高质量的生成效果。然而，当任务需要**局部编辑（local editing）**时，DiT 模型常会出现“编辑泄漏”（edit leakage）问题，即编辑信号扩散到非目标区域，导致背景被意外修改。  
- **核心问题：**  
  当前模型通常依赖人工提供的**显式掩码（mask）**以指出编辑区域，但这会：
  - 降低可用性（需人类标注）；
  - 阻碍自动化编辑；
  - 限制模型对自然语言指令的空间理解。  
  因此，本文提出的目标是构建一个：
  > **无需显式掩码（mask-free）且能自动识别编辑区域的局部编辑扩散模型。**

## 二、方法论：AdaptEdit 框架

### 1. 核心思想
AdaptEdit 将冻结的扩散 Transformer 主体转化为一个**区域感知、指令感知的局部编辑器**。  
其核心思想是：在每个 Transformer block 内注入轻量化的适配器（Adapter），以区分 _“要编辑什么（What）”_ 和 _“要编辑哪里（Where）”_。

### 2. 关键组成模块

- **Block Adapter（块级适配器）**  
  在每层 Transformer 内添加独立的小适配模块，用于注入外部条件（即文本指令与空间位置信号），无需修改主干权重。  

- **SpatialGate（空间门控机制）**  
  用于在每个空间位置上选择性地激活适配器信号：
  - 在预测的编辑区域内，允许编辑信号通过；
  - 在背景区域，抑制更新，确保图像保持原样；
  - 实现 “soft gating” 控制，以避免编辑边界产生不自然的硬切割。

- **MaskPredictor（掩码预测头）**  
  在训练阶段利用人工或真实掩码作为监督目标，使网络学习从图片和文本指令中预测“应编辑的区域”。  
  部署时该模块能**自动生成掩码**，无需用户输入。  

- **Region-Aware Loss（区域感知损失）**  
  训练损失仅集中于被编辑区域的像素，从而：
  - 强化编辑区域的拟合精度；
  - 减少非编辑区的梯度干扰；
  - 提高局部保持性（preservation）。

### 3. 整体流程描述
1. **输入：** 源图像 + 文本编辑指令。  
2. **特征提取：** 利用冻结的 DiT 主干获得视觉与语言隐空间特征。  
3. **信号注入：** Block Adapter 结合 SpatialGate 将语义与空间信号融合到每层。  
4. **掩码预测：** MaskPredictor 根据当前隐状态估计编辑区域。  
5. **输出：** 通过反向扩散生成编辑后图像。
