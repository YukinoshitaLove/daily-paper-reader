## 摘要
现有的对象移除工具通常依赖手动掩码或文本提示，这使得非专业用户在复杂场景中难以精确移除对象，并常常导致移除不完全或背景补全不自然。为了解决这一问题，我们提出了 ClickRemoval，这是一款基于预训练 Stable Diffusion 模型并完全由用户点击驱动的开源交互式对象移除工具。无需额外训练、手绘掩码或文本描述，ClickRemoval 通过在去噪过程中调制自注意力机制来定位目标对象并恢复背景。实验结果表明，ClickRemoval 在多项定量指标和用户研究中均取得了具有竞争力的表现。我们已在 Apache-2.0 许可下发布完整软件包，网址为 https://github.com/zld-make/ClickRemoval。

## Abstract
Existing object removal tools often rely on manual masks or text prompts, making precise removal difficult for non-expert users in complex scenes and often leading to incomplete removal or unnatural background completion. To address this issue, we present ClickRemoval, an open-source interactive object removal tool built on pretrained Stable Diffusion models and driven solely by user clicks. Without additional training, hand-drawn masks, or text descriptions, ClickRemoval localizes target objects and restores the background through self-attention modulation during denoising. Experiments show that ClickRemoval achieves competitive results across quantitative metrics and user studies. We release a complete software package at https://github.com/zld-make/ClickRemoval under the Apache-2.0 license.

## 二、方法论：ClickRemoval 的设计与原理

### 2.1 核心思想  
ClickRemoval 通过“用户点击 → 语义距离图 → 注意力重定向”的链路，实现对目标区域的抑制与背景自然还原。  
该方法不改变 Stable Diffusion 的结构，仅在**推理阶段**注入注意力引导信号。

### 2.2 模型结构与模块组成  
整体框架包括四个核心子模块（见论文 Figure 1）：

1. **M2N2 语义距离映射（Semantic Distance Map）**  
   - 将用户点击位置转换为语义相关性图。  
   - 基于 Stable Diffusion 的自注意力特征计算各像素与点击点的语义距离。  
   - 使用 Markov 传播与 Flood Fill 改进局部鲁棒性，得到：
     - 对象图 \( M_{obj} \)：指示需要去除的区域；
     - 背景图 \( M_{bg} \)：表示与目标语义不同的背景特征区域。

2. **SGAR（Self-Guided Attention Redirection）**  
   - 在去噪循环中实时**调整注意力权重**，抑制目标相关注意力通道。  
   - 对每步的自注意力张量 \( S_{self} \) 重新加权：
     \[
     S'_{self} = S_{self} \odot B(G_{bg}(t)) + B(P_{obj})
     \]
     其中 \( G_{bg}(t) \) 控制背景权重随时间衰减，\( P_{obj} \) 对目标施加负惩罚。

3. **SGAS（Self-Guided Attention Scheduling）**  
   - 负责时间调度：在不同去噪阶段逐步控制 SGAR 的作用强度。
   - **早期阶段**：保持原始生成；  
     **中期**：启用背景引导与对象抑制；  
     **后期**：移除所有引导，仅依赖扩散模型自然修复。

4. **ARG（Attention Redirection Guidance）**  
   - 类似 ERG 方法的可控合成策略：以线性方式融合原始与调制噪声预测：
     \[
     \epsilon_{ARG} = (1 - r)\epsilon(x,t) + r\epsilon'(x,t)
     \]
     - 参数 \( r \in [0, 1] \) 表示用户可控的“移除强度系数”。

### 2.3 实现版本  
为平衡性能与速度，ClickRemoval 提供三种配置：

| 版本 | 基础模型 | 特点 |
|------|------------|------|
| **SD1.5** | Stable Diffusion 1.5 | 轻量实时版 |
| **SD2.1** | Stable Diffusion 2.1 | 平衡型 |
| **SDXL1.0** | Stable Diffusion XL | 高分辨率高质量版 |
