## 摘要
基于指令的图像编辑（Instruction-based Image Editing，IIE）模型近年来在根据自然语言指令修改特定图像区域方面表现出强大的能力，而这一过程隐含地需要识别编辑应当施加的位置。这表明此类模型在本质上执行了语言调控的视觉语义定位。在本研究中，我们探讨这种隐式的语义定位能否被用于零样本指代表达图像分割（Referring Image Segmentation，RIS），该任务要求根据自然语言描述在像素级别上定位目标对象。通过系统分析，我们发现，这些模型的内部表示在最早的去噪步骤（denoising timestep）就出现了显著的前景-背景可分性，远在任何可见图像变换发生之前。基于这一发现，我们提出了一个无需训练的框架，通过利用预训练图像编辑模型的中间表示，将其重新用于 RIS 任务。我们的方法将定位分解为两个互补组成部分：基于注意力的空间先验，用于估计应聚焦的位置；以及基于特征的语义判别，用于确定应分割的内容。通过利用特征空间的可分性，该框架仅需一次去噪步骤即可生成精确的分割掩码，无需完成完整的图像合成。在 RefCOCO、RefCOCO+ 和 RefCOCOg 等数据集上的大量实验表明，我们的方法在现有零样本基线之上取得了优异性能。

## Abstract
Instruction-based image editing (IIE) models have recently demonstrated strong capability in modifying specific image regions according to natural language instructions, which implicitly requires identifying where an edit should be applied. This indicates that such models inherently perform language-conditioned visual semantic grounding. In this work, we investigate whether this implicit grounding can be leveraged for zero-shot referring image segmentation (RIS), a task that requires pixel-level localization of objects described by natural language expressions. Through systematic analysis, we reveal that strong foreground-background separability emerges in the internal representations of these models at the earliest denoising timestep, well before any visible image transformation occurs. Building on this insight, we propose a training-free framework that repurposes pretrained image editing models for RIS by exploiting their intermediate representations. Our approach decomposes localization into two complementary components: attention-based spatial priors that estimate where to focus, and feature-based semantic discrimination that determines what to segment. By leveraging feature-space separability, the framework produces accurate segmentation masks using only a single denoising step, without requiring full image synthesis. Extensive experiments on RefCOCO, RefCOCO+, and RefCOCOg demonstrate that our method achieves superior performance over existing zero-shot baselines.

- **研究动机**

- **研究意义**  
  - 当前 zero-shot RIS 方法多依赖多阶段流水线（如 SAM + CLIP），存在几何偏向和语言理解能力不足的问题。

## 二、方法论与技术框架

### 1. 核心思想
- 通过对现代图像编辑模型（如 FLUX.2-klein、Step1X-Edit）内部表征的系统分析，发现：
  - 在最早的去噪步骤（denoising timestep）即出现明显的前景-背景可分性；
  - 即语义定位能力在图像生成开始前就已形成。  
- 基于此，提出一个 **无需训练（training-free）** 的框架，直接利用预训练模型的中间特征实现分割。

### 2. 整体流程
框架由两个互补模块构成：

#### （1）注意力驱动的空间先验（Attention-based Region Proposal, ARP）
- 从多模态自注意力（MMSA）矩阵中提取图像到文本的 **cross-attention** 子矩阵，得到粗略空间分布（CAM）。  
- 引入 **基于自注意力的亲和传播（Affinity Propagation）** 机制，通过随机游走扩散相似区域的响应，生成更平滑、连贯的 **Refined Attention Map (RAM)**。  
- 数学形式：
  - 对应的传播过程建模为随机游走的转移矩阵 \(T(t,l)\)，用以扩散视觉 token 间的注意力权重。

#### （2）特征驱动的语义定位（Feature-based Semantic Localization, FSL）
- 提取 MMSA 输出的 clean image token 特征并进行归一化。  
- 基于 RAM 提供的先验掩码：
  - 构造前景原型 \(P_{fg}\) 与背景原型 \(P_{bg}\)，通过掩码平均池化计算；  
  - 对每个像素执行 **余弦相似度** 分类，生成最终分割结果：
    \[
    S_{h,w} = \arg\max_{c \in \{fg, bg\}} P_c^\top \hat{F}_{up,t,l,h,w}
    \]
- 整体仅需单次去噪步骤完成高质量分割。

### 3. 分析基础与指标
- 使用类似 **Fisher 判别准则** 的特征可分性度量（Separability Score）：
  \[
  J = \frac{(\mu_{fg} - \mu_{bg})^2}{\sigma^2_{fg} + \sigma^2_{bg}}
  \]
  以衡量不同时刻模型的前景/背景特征分离能力。

## 三、实验设计与评估方案

### 1. 数据集与任务场景
- 三个标准指称分割数据集：  
  - **RefCOCO**：句子简短，多含“left/right”等方位词；  
  - **RefCOCO+**：去除方位描述，更注重语义性；  
  - **RefCOCOg**：复杂长文本指令，难度最高。  

### 2. 评价指标
- **oIoU**（总体交并比）：偏向大对象性能。  
- **mIoU**（平均交并比）：衡量整体分割精度的平均表现。

## 六、主要结论与发现

- 图像编辑模型在最初的去噪步骤即具有稳定的语义分离能力，表明 **语义定位在生成之前就已完成**。  
- 通过单步去噪、无训练框架即可实现 **高精度、低计算成本** 的零样本指称分割。  
- 在 RefCOCO/RefCOCO+/RefCOCOg 上性能显著优于最强零样本基线 RefAM：  
  - oIoU 提升约 5–7 个点；  
  - mIoU 在各数据集均为最佳。  
- 验证了生成式模型内部语义表征可自然用于判别式任务，如视觉定位与分割。

## 八、不足与局限

- **边界精度受限**：在复杂细节或精细形状上分割不够准确。  
- **实例混淆现象**：当场景中存在多个同类对象时，模型易激活全部实例。  
- **语义覆盖不完全**：对于部分复杂指令，模型可能仅捕获部分目标区域。  
- **依赖底层编辑模型能力**：不足主要源自预训练编辑模型本身，而非框架设计。  
- **泛化性风险**：尚未验证其他类型编辑架构（如非DiT模型）的适应性。
