## 摘要
我们提出了HICNet，一种参考引导的曝光校正框架。一个轻量级、内容无关的编码器将每张图像蒸馏成一个紧凑的光照嵌入，捕获区域亮度、边缘对比度以及高阶亮度矩。源图像与参考图像之间的嵌入差异驱动一个多尺度调制网络，该网络结合了基于FiLM的全局调整和光度通道重平衡，实现细粒度、光照感知的光谱门控，产生与曝光匹配的输出，同时忠实保留场景细节。跨批量对比损失对光照流形进行排序，增强了对多样光照条件的鲁棒性。在没有真实参考或内在分解的情况下训练，HICNet在公共基准上达到了更优的准确性，并能很好地泛化到完全未见过的场景。

## Abstract
We present HICNet, a reference-guided exposure correction framework. A lightweight, content-agnostic encoder distills each image into a compact illumination embedding capturing regional brightness, edge contrast, and higher-order luminance moments. The embedding difference between a source and its reference drives a multi-scale modulation network that combines FiLM-based global adjustment with Photometric Channel Rebalancing for fine-grained, illumination-aware spectral gating, producing exposure-matched outputs while faithfully preserving scene details. A cross-batch contrastive loss orders the illumination manifold, bolstering robustness to diverse lighting conditions. Trained without ground truth or intrinsic decomposition, HICNet attains better accuracy on public benchmarks and generalizes well to entirely unseen scenes.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题定义**：曝光校正是计算摄影中的基础任务，旨在将欠曝光或过曝光的图像恢复至视觉自然的状态。实际场景中，光照不均、相机设置不当或动态范围有限导致细节损失、对比度低、视觉不适。
- **现有方法局限**：
  - **监督方法**（如Afifi et al., ECM）依赖成对的高质量真实数据，数据集获取成本高，且泛化性差。
  - **基于Retinex分解的方法**依赖强先验（如空间平滑性），在复杂场景中适应性受限。
  - **无监督方法**（如Zero-DCE, QuadPrior, UEC）通常基于简单曲线或刚性假设，缺乏显式、可控的曝光风格表示，且没有有效解决源图像与参考图像之间的语义内容差异。
- **核心挑战**：在不依赖真实标注的情况下，如何将光照风格与语义内容解耦，实现鲁棒的跨图像曝光校正。
- **本文贡献**：提出HICNet，一种参考引导的曝光校正框架，通过内容无关的曝光编码器（CAEE）提取光照嵌入，结合多尺度调制网络（FiLM + 光度通道重平衡PCR）进行自适应调整，并引入跨批次对比损失来结构化光照流形。无需配对数据或内在分解，在公开基准上达到更优精度，并能泛化到未见场景。

## 2. 方法论：核心思想、关键技术细节
### 2.1 整体框架
- 输入：源图像 \(I_s\)（曝光不当）和参考图像 \(I_r\)（期望曝光分布，可能场景不同）。
- 输出：校正图像 \(\hat{I}\)，保留 \(I_s\) 内容，对齐 \(I_r\) 的曝光统计。
- 训练时无真实三元组 \((I_s, I_r, \hat{I}^*)\)。

### 2.2 内容无关曝光编码器（CAEE）
- 目的：将图像压缩为低维光照描述符 \(z \in \mathbb{R}^{66}\)，忽略语义，仅捕获“曝光风格”。
- 三个互补信号：
  1. **区域池化亮度**：灰度图经三个Conv-InstanceNorm-ReLU块+自注意力层+2×2自适应平均池化，得到256维 \(F_{\text{reg}}\)，指示明暗区域位置。
  2. **梯度局部对比度**：固定Sobel算子计算梯度幅值 \(G\)，经小型CNN压缩为32维 \(F_{\text{ctr}}\)，反映曝光引起的边缘变化。
  3. **全局直方图矩**：计算灰度图的均值、标准差、偏度、峰度，以及欠/过曝光像素比例（阈值0.05/0.95），得到6维向量 \(S\)。
- 描述符：拼接后经两层MLP得到 \(z = \Phi([F_{\text{reg}}; F_{\text{ctr}}; S]) \in \mathbb{R}^{66}\)。

### 2.3 多尺度曝光调制网络（MSEM）
- 采用三阶段U-Net（编码器-解码器），特征图通道数32/64/64。
- 计算光照偏移 \(\Delta z = z_r - z_s\)。
- **特征级线性调制（FiLM）**：每个尺度 \(i\) 用MLP预测参数 \((\alpha_i, \beta_i)\)，进行仿射变换：\(\tilde{F}_i = \alpha_i \odot F_i + \beta_i\)，保持内容拓扑。
- **光度通道重平衡（PCR）**：将 \(\Delta z\) 映射为每通道门控值 \(a_i = \sigma(W_i \Delta z + b_i) \in (0,1)\)，然后重平衡：\(\hat{F}_i = (1 + a_i) \odot \tilde{F}_i\)，突出曝光敏感频带。
- **解码与重建**：上采样后经3×3卷积输出残差 \(R\)，最终 \(\hat{I} = \text{clip}(I_s + R, 0, 1)\)。

### 2.4 训练损失
- **重建损失**：\(\mathcal{L}_{\text{pix}} = \|\hat{I} - I_r\|_1\)。
- **暗通道先验损失**：\(\mathcal{L}_{\text{dc}} = \|D(\hat{I}) - D(I_r)\|_1\)，其中 \(D(\cdot)\) 为局部16×16邻域最小值。
- **对比照明损失**：使用动量教师（EMA）空间定义正样本集，采用多正例NT-Xent损失 \(\mathcal{L}_{\text{ctr}}\)，结构化光照流形。
- **总损失**：\(\mathcal{L} = \mathcal{L}_{\text{pix}} + \lambda_{\text{dc}} \mathcal{L}_{\text{dc}} + \lambda_{\text{ctr}} \mathcal{L}_{\text{ctr}}\)，其中 \(\lambda_{\text{dc}}=0.5, \lambda_{\text{ctr}}=0.1, \tau=0.07\)。
