## 摘要
低光照图像存在严重噪声、对比度损失和语义模糊，使得增强成为去噪与细节恢复的联合问题。我们提出PixIE，一种由视觉基础模型语义提示的前馈像素空间LLIE框架。PixIE首先执行跨尺度去噪以抑制噪声并保持结构，然后使用DINO提示像素块（DPPBs）细化细节，该模块通过补丁条件化的、空间连续的逐像素调制注入中间DINOv3特征。为使像素空间注意力在跨尺度上高效，我们引入空间通道压缩（SCC），联合减少空间令牌网格和通道维度。我们进一步提出多感受野像素嵌入（MRPE），在语义提示前提供邻域感知的像素表示，提高对超出逐点嵌入的信号相关噪声的鲁棒性。在LLIE基准上的实验表明，PixIE相较于近期最优方法平均PSNR提升1.9%-15.0%，LPIPS降低8.5%-44.4%。定性比较进一步显示更清晰的细节和更稳定的纹理，同时提升了重建保真度和感知质量。

## Abstract
Low-light images suffer from severe noise, contrast loss, and semantic ambiguity, making enhancement a joint problem of denoising and detail recovery. We propose PixIE, a feed-forward pixel-space LLIE framework semantically prompted by a vision foundation model. PixIE first performs cross-scale denoising to suppress noise and preserve structure, then refines details using DINO-Prompted Pixel Blocks (DPPBs), which inject intermediate DINOv3 features through patch-conditioned, spatially continuous per-pixel modulation. To make pixel-space attention efficient across scales, we introduce Spatial-Channel Compaction (SCC), which jointly reduces the spatial token grid and channel dimension. We further propose Multi-Receptive-Field Pixel Embedding (MRPE) to provide neighborhood-aware pixel representations before semantic prompting, improving robustness to signal-dependent noise beyond point-wise embeddings. Experiments on LLIE benchmarks show that PixIE improves average PSNR by 1.9-15.0% over recent state-of-the-art methods and reduces LPIPS by 8.5-44.4%. Qualitative comparisons further show sharper details and more stable textures, improving both reconstruction fidelity and perceptual quality.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：低光照图像同时遭受**严重噪声、对比度丢失和语义模糊**，导致增强任务必须联合解决**去噪与细节恢复**。现有方法（CNN、Transformer、Mamba、扩散模型等）在光照校正和噪声抑制上取得了进展，但**往往过度平滑纹理、产生颜色漂移或跨区域不一致**，尤其在极暗区域，噪声与纹理难以区分。
- **背景动机**：① 数据驱动模型受限于训练分布，难以捕获超出分布的先验；② 虽然已有一些工作尝试利用视觉基础模型（如CLIP、SAM、DINO）特征作为语义引导，但**通常采用全局池化或粗粒度补丁拼接**，与需要高保真细节的像素级恢复之间存在“语义-空间鸿沟”；③ 像素空间建模能保留高频细节，但**朴素像素注意力在高分辨率下计算代价高**，且**点式投影（1×1）缺乏空间感受野**，在强噪声下无法区分结构与噪声。
- **论文目标**：提出**PixIE**——首个将**基础模型语义直接注入像素空间恢复管线**的低光照增强框架，通过**空间连续的逐像素调制**实现语义感知的细节保持增强，同时利用有效的压缩机制控制计算成本。

## 2. 方法论：核心思想、关键技术细节、算法流程

### 2.1 整体架构（端到端，前馈式）

输入低光照图像 I ∈ R^(3×H×W) → 3×3 卷积提取特征 F0 ∈ R^(C×H×W) → 构建三尺度金字塔（s=1,2,3，分辨率依次减半，通道数加倍）→ 各尺度依次经过：

1. **跨尺度去噪（Cross-Scale Denoising）**：语义无关的噪声抑制与结构保持。
2. **多感受野像素嵌入（MRPE）**：聚合邻域上下文。
3. **N个DINO提示像素块（DPPBs，N=4）**：注入DINOv3语义进行逐像素调制。
4. **多尺度融合（Multi-Scale Fusion）**：上采样后拼接各尺度精化特征，经2个3×3卷积预测残差 R，最终输出 ˆI = I + R。

### 2.2 关键技术细节

#### ① 跨尺度去噪
- 使用**Restormer风格的Transformer块**（MDTA + GDFN），MDTA在通道维度做自注意力，复杂度 O(HW·C²)（线性于像素数）。
- **细到粗的特征传播**：每个尺度两个顺序块 T1、T2。对于 s>1，将上一尺度的块输出通过 PixelUnshuffle 下采样后与本尺度对应块输出拼接，再经通道投影。注意：**只传播去噪后的特征 ˜F^(s)**，不传播经过DPPB调制的特征，以避免语义调制带来的分布偏移。

#### ② 多感受野像素嵌入（MRPE）
- 在语义提示前对像素嵌入局部上下文：使用三种并行卷积核（1×1、3×3、5×5），通道分配比例（25% : 37.5% : 37.5%），强调大感受野，抑制点状噪声。
- 特征拼接后经线性投影混合，得到邻域感知的表示，再送入DPPB。

#### ③ DINO提示像素块（DPPB）
- 冻结的DINOv3 ViT-S/16，每输入一次前向传播，提取中间层{2,5,8,11}的补丁令牌，统一缩放到各尺度所需的令牌网格，拼接后作为条件。
- **空间连续的逐像素调制**：轻量MLP在每个令牌位置预测6C个AdaLN-Zero参数（缩放、平移、门控用于注意力和FFN），再通过**双线性上采样 + 3×3深度卷积**生成全分辨率平滑参数场，实现逐像素调制（而非补丁独立预测，避免块边界伪影）。相比PixelDiT的每补丁预测P²个参数集，MLP参数减为1/P²（P=16时256倍），且消除边界不连续。
- 更新公式（以注意力子层为例）：
  ```
  x ← x + γ_a ⊙ Attn(Mod(RMSNorm(x); α_a, β_a))
  ```
  FFN子层同理，且FFN保持全分辨率。

#### ④ 空间通道压缩（SCC）
- 仅用于注意力分支，FFN保持全分辨率以保留细节。
- 步骤：将特征 x ∈ R^(B×C×H×W) 折叠为不重叠 P×P 补丁 → 1×1投影压缩到注意力维度 D_attn（设为16）→ 在小令牌网格上执行MDTA（复杂度 O(HpWp·D_attn²)）→ 展开回原分辨率。
- 相比直接对全分辨率x做MDTA（O(HW·C²)），通过空间折叠（减少HW/ P²因子）和通道瓶颈（C²→D_attn²）大幅降低计算量，且对粗尺度高通道数不敏感。

## 6. 优点

1. **创新的语义注入方式**：通过空间连续的逐像素AdaLN-Zero调制，避免补丁边界伪影，实现精细的语义-像素对齐，优于全局/补丁级引导。
2. **高效的像素空间注意力**：SCC同时缩减空间和通道维度，使像素级注意力在粗尺度上依然高效，且不影响FFN的全分辨率处理。
3. **鲁棒的去噪-增强解耦设计**：先进行语义无关的跨尺度去噪，再注入语义提示，避免噪声干扰语义引导，提升稳定性。
4. **多感受野嵌入**：MRPE在语义提示前聚合邻域上下文，有效抵抗信号相关噪声。
5. **轻量且性能优越**：可训练参数仅11.84M，GFLOPs仅65.09（256×256），在多个基准上取得最优结果，复杂度与性能平衡出色。
6. **丰富的消融和分析**：对每个设计选择进行了充分验证，包括层选择、调制机制、通道分配等，保证了方法论的可信度。

## 7. 不足与局限

1. **未明确训练硬件与时长**：虽然报告了迭代次数和模型大小，但未提供GPU型号、数量、单卡训练时长等关键算力信息，不利于可复现性评估。
2. **极端场景失效**：① 极端非均匀人造光照下的色偏可能被过度校正；② 非常小的物体（<16像素）低于DINO补丁分辨率，接收不到有效语义引导；③ 复合退化（如噪声+模糊+低光）未专门训练，鲁棒性有限。
