## 摘要
近年来，基于 Diffusion Transformer（DiT）的视频生成技术在视频对象移除任务上取得了显著进展。然而，这些方法依然受到推理延迟较大的困扰。例如，尽管 MiniMax Remover 在视觉质量上达到了最新水平，但其运行速度仅约为 10FPS，主要原因是在整个时空标记空间上进行密集计算，即使只有很小的遮罩区域需要处理。本文提出了一种高效的微调框架 YOSE（You Only Select Essential Tokens）。YOSE 主要包括两个关键组件：批变长索引（Batch Variable-length Indexing, BVI）和扩散过程模拟器（Diffusion Process Simulator, DiffSim）模块。BVI 是一种可微分的动态索引算子，能够基于遮罩信息自适应地选择关键标记，从而实现不同样本间可变长度的标记处理。DiffSim 则为未遮罩标记提供扩散过程的近似机制，用以在 DiT 自注意力中模拟未遮罩区域的影响，从而保持遮罩区域的语义一致性。凭借这些设计，YOSE 实现了基于遮罩感知的加速，其推理时间与遮罩区域大小近似线性相关，而全标记扩散方法的计算量则与遮罩大小无关。大量实验表明，YOSE 在保持与基线相当的视觉质量的同时，在 70% 的情况下实现了最高 2.5 倍的加速。代码可在以下链接获得：https://github.com/Wucy0519/YOSE-CVPR26。

## Abstract
Recent advances in Diffusion Transformer (DiT)-based video generation technologies have shown impressive results for video object removal. However, these methods still suffer from substantial inference latency. For instance, although MiniMax Remover achieves state-of-the-art visual quality, it operates at only around 10FPS, primarily due to dense computations over the entire spatiotemporal token space, even when only a small masked region actually requires processing. In this paper, we present YOSE, You Only Select Essential Tokens, an efficient fine-tuning framework. YOSE introduces two key components: Batch Variable-length Indexing (BVI) and Diffusion Process Simulator (DiffSim) Module. BVI is a differentiable dynamic indexing operator that adaptively selects essential tokens based on mask information, enabling variable-length token processing across samples. DiffSim provides a diffusion process approximation mechanism for unmasked tokens, which simulates the influence of unmasked regions within DiT self-attention to maintain semantic consistency for masked tokens. With these designs, YOSE achieves mask-aware acceleration, where the inference time scales approximately linearly with the masked regions, in contrast to full-token diffusion methods whose computation remains constant regardless of the mask size. Extensive experiments demonstrate that YOSE achieves up to 2.5X speedup in 70% of cases while maintaining visual quality comparable to the baseline. Code is available at: https://github.com/Wucy0519/YOSE-CVPR26.

## 一、研究背景与核心问题

- **研究领域与背景**：  
  随着 *Diffusion Transformer（DiT）* 在图像与视频生成领域的兴起，基于扩散模型的视频对象移除（Video Object Removal, VOR）任务在视觉一致性和生成质量方面取得显著进展。代表性方法包括 **ROSE** 和 **MiniMax Remover**，后者在保持高视觉质量的同时仍存在推理延迟高的问题。

- **核心问题**：  
  当前 DiT 架构在推理阶段对整段视频的全部时空 token 执行密集计算，即使只有少量遮罩区域需要处理，也要在全空间执行注意力和去噪运算。这种 **全 token 计算的冗余** 导致：
  - 推理复杂度与遮罩比例无关；
  - 小遮罩场景计算浪费极大；
  - 实时视频修复难以实现。

- **研究动机**：  
  视频对象移除本质上是 **局部生成任务** —— 仅需在遮罩区域执行生成。作者期望通过 **mask-aware（遮罩感知）加速策略**，实现推理复杂度与遮罩比例线性相关的高效扩散算法。

## 二、方法论与技术方案

### 1. 方法总体结构
YOSE（You Only Select Essential Tokens）是一个轻量级、可插拔的 DiT 微调框架，可直接应用于现有视频对象移除模型（如 MiniMax Remover、VACE）。核心理念是：
> “只选择必要的标记进行扩散处理，同时模拟未遮罩区域的扩散影响，以保持全局语义一致性。”

### 2. 核心组件与流程
#### （1）Batch Variable-length Indexing（BVI）
- 可微分的动态索引算子，用于准确选取遮罩区域对应的 **关键标记（Essential Tokens）**。
- 实现步骤：
  - 根据遮罩生成 **前向索引** `Ind_F` 从视频 latent 中提取需处理的标记；
  - 经 DiT 模块处理后，再通过 **反向索引** `Ind_B` 将结果放回原视频位置；
  - 全过程通过 `grid_sample()` 实现可微分采样，保证梯度传播。
- 特点：支持每个样本具有不同 mask 长度的 **可变 token 序列**；在批处理时通过 padding 实现并行，从而使计算复杂度与遮罩比例线性相关。

#### （2）Diffusion Process Simulator（DiffSim）
- 目标：解决仅处理遮罩区域导致的语义不连续问题。
- 方法：
  - 定义未遮罩区域的 **残差 latent**： `Res_Nis = Lat_Nis − Lat_mask`；
  - 构建三个可学习参数：  
    - 组合参数 `G` 控制遮罩与残差的插值比；  
    - 缩放参数 `S` 与偏置参数 `Bias` 调整场景统计；  
  - 用于生成模拟的 Key–Value 表征 `KV = (1 + S)·[G·Lat_mask + (1−G)·Res_Nis] + Bias`；
  - 模拟未遮罩区域的扩散行为，无需参与完整 DiT 处理。
- 效果：在每个 DiT block 内将遮罩区与外部上下文融合，保持语义与时间一致性。

#### （3）边界融合策略（Fusion Strategy）
- 为缓解遮罩边缘伪影，通过**均值–方差对齐 + 加权融合**平滑过渡：
  1. 膨胀遮罩区域获取邻域；
  2. 对重叠区的预测与原始区域分别计算均值、标准差；
  3. 将预测区归一化、重标定以匹配外部分布；
  4. 最终通过加权融合掩码平滑边界。
- 有效消除接缝，实现自然衔接。

#### （4）损失设计
采用 **mask-aware flow-matching loss**：
\[
L_{FM_{mask}} = \frac{|\text{mask} \odot (\text{out} - \text{Noise-GT})|_2^2}{|\text{mask}|_1}
\]
仅在遮罩区对扩散轨迹展开约束，保证生成的局部区域精确拟合目标。
