## 摘要
组合式文本到图像（T2I）生成要求模型能够同时遵循多个描述不同图像区域的子提示（sub-prompts）。近期研究表明，扩散模型的\emph{起始噪声}携带着显著的语义信息：从文本预测出的“黄金”噪声（golden noise）可以显著提高提示的一致性。然而，我们观察到这种噪声预测本质上是全局性的：同一个网络需要用一个单一的文本嵌入来概括冗长、涉及多区域的提示，因此当提示描述空间上分离的实体时，这就成为性能瓶颈。我们提出了\textbf{Golden RPG}，一种区域感知的噪声预测器，它在冻结的 NPNet 基础上增加了两个可训练模块：（i）基于每个区域的\textbf{FiLM 适配器}，可根据各自的子提示调整预测噪声；（ii）在 Swin 主干网络的两个阶段之间插入的\textbf{区域交叉注意力（Region Cross-Attention）}层，使不同空间位置能够关注不同的子提示标记。为防止区域条件在提示已易于生成的样本上造成退化，我们进一步提出了\textbf{置信度自适应融合（Confidence-Adaptive Blending）}头，该模块可动态预测每个样本中区域信号应当覆盖全局信号的强度。我们在原始 RPG 基准（20 个提示，100 个样本）以及 T2I-CompBench 的四个多区域类别（1,200 张图像，六种对比方法）上进行了评估。Golden RPG 在所有类别上都获得了最高的跨区域一致性（Cross-Region-Coherence）分数，并在绝对的 CLIP-Score 和 CLIP-IQA 上与最强基线方法持平。配对用户研究进一步显示，本方法相较于最强基线有约 \boldsymbol{~67\%} 的偏好率。该适配器包含约 200 万个可训练参数，在 SDXL 模型上仅增加 0.6 秒的推理开销。

## Abstract
Compositional text-to-image (T2I) generation requires a model to honour multiple sub-prompts that describe distinct image regions. Recent work shows that the \emph{starting noise} of a diffusion model carries significant semantic information: ``golden'' noise predicted from text can substantially raise prompt fidelity. We observe that this noise prediction is, however, fundamentally global: the same network is asked to summarise a long, multi-region prompt with a single text embedding, which becomes the bottleneck whenever the prompt describes scenes with spatially-separated entities. We introduce \textbf{Golden RPG}, a region-aware noise predictor that extends a frozen NPNet with two trainable additions: (i) a per-region \textbf{FiLM adapter} that reshapes the predicted noise according to each sub-prompt; and (ii) a \textbf{Region Cross-Attention} layer injected between two stages of the Swin backbone, allowing different spatial locations to attend to different sub-prompt tokens. To prevent the regional conditioning from degrading samples whose prompts are already easy, we further propose a \textbf{Confidence-Adaptive Blending} head that dynamically predicts, per sample, how strongly the regional signal should override the global signal. We evaluate on the original RPG benchmark (20 prompts, 100 samples) and on four multi-region categories of T2I-CompBench (1{,}200 images, six competing methods). Golden RPG achieves the highest Cross-Region-Coherence score on every category, while matching the strongest baselines on absolute CLIP-Score and CLIP-IQA. A paired user study further shows a $\boldsymbol{\sim}$67\% preference over the strongest baseline. The adapter contains $\sim$2M trainable parameters and adds only $0.6$\,s of inference overhead on top of SDXL.

## 一、研究背景与核心问题

- **研究动机**  
  文本到图像（Text-to-Image, T2I）扩散模型近年来在图像生成的真实感和语义一致性上取得重大突破（如：Stable Diffusion XL、Latent Diffusion、SD3）。  
  然而，这些模型在处理**多对象、多区域组合性（Compositional）场景**时仍表现不佳，例如在生成“左边一个红苹果、右边一个蓝杯子”时，常出现对象颜色混淆、空间重叠等问题。

- **问题根源**  
  现有的“黄金噪声”（Golden Noise / NPNet）技术虽能根据文本预测起始噪声，从而改善语义一致性，但其噪声预测是**全局性的**：  
  同一网络用单个文本向量去描述跨多个区域的复杂提示，这使语义混合、区域对齐困难。

- **研究目标**  
  提出一种能够**对不同区域独立编码噪声语义**的模型，使扩散过程从最初的噪声层面就具备空间感知能力，从而提升复杂场景生成的区域一致性、物体定位、属性绑定等质量指标。

## 二、方法论：Golden RPG 的核心技术设计

### 1. 总体思路

Golden RPG 在保持 **SDXL 与 NPNet 冻结**的前提下，新增约 **200 万个参数** 的轻量模块，使起始噪声生成对多区域子提示（sub-prompts）具备差异化响应。

模型包含三大创新组件：

1. **区域 FiLM 适配器（Per-Region FiLM Adapter）**  
   - 基于 FiLM（Feature-wise Linear Modulation）机制。  
   - 为每个区域子提示生成独立的通道级缩放与偏移参数（γ, β）。  
   - 在噪声张量中按区域掩码（mask）调整局部噪声特征，使不同区域对应不同的语义偏向。  
   - 保持可微分与近似恒等初始化，防止早期训练不稳定。

2. **区域跨注意力层（Region Cross-Attention, RCA）**  
   - 插入至 Swin Transformer 主干的第 2 与第 3 阶之间。  
   - 使不同空间位置的特征可聚焦于各自子提示文本的 token 序列。  
   - 在训练初期初始化为近似恒等映射，贡献细粒度的空间-语义对齐。

3. **置信度自适应融合头（Confidence-Adaptive Blending Head）**  
   - 通过一个微型三层 MLP，基于全局与区域文本嵌入的统计量（如嵌入差异、内积、标准差等）预测每样本的融合权重 α ∈ [0, α_max]。  
   - 控制模型在“全局黄金噪声”与“区域调制噪声”之间的插值。  
   - 当提示不具备显著的区域性时，可自动降低区域信号强度以避免生成退化。

### 2. 训练目标与优化流程

- 损失函数由四部分构成：

  ```text
  L = L_mse + λ_r * L_rank + λ_d * L_div + λ_α * L_α
  ```

  - **L_mse**：输出噪声与最优候选噪声之间的均方误差；
  - **L_rank**：基于区域语义对齐分数（RSA）的排序损失；
  - **L_div**：鼓励区域间噪声统计差异性；
  - **L_α**：对置信度头 α 的软监督项。

- **训练流程概述**
  1. 对每个训练提示生成 5 个候选噪声并计算区域语义对齐（RSA）；
  2. 选取最佳、最差噪声为正负样本；
  3. 通过上述损失联合优化 FiLM、RCA、Confidence Head；
  4. 其余模块（NPNet、SDXL、编码器）全部冻结。
