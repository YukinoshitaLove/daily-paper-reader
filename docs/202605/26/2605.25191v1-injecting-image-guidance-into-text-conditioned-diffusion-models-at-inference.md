## 摘要
文本到图像扩散模型（如Stable Diffusion）能从文本生成高质量图像，但缺乏在推理时注入视觉引导（如草图、风格）而不重新训练的方法。现有方法要么需要计算昂贵的微调，要么依赖风格迁移技术，这可能与文本提示产生语义错位。我们提出了视觉概念融合（VCF），这是首个在推理时无需任何特定概念训练即可提供图像和文本提示双重条件的方法。VCF通过将CLIP图像特征与文本嵌入空间对齐，实现了向Stable Diffusion注入视觉概念。VCF由三个组件组成：（1）一个轻量级对齐器，利用InfoNCE和交叉注意力重建损失将图像令牌映射到文本嵌入流形；（2）一种融合策略，同时保留文本和视觉语义；（3）一个可选的提示-噪声优化（PNO）模块，用于测试时细化。我们的实验表明，VCF成功地从参考图像中迁移了风格、构图和色彩搭配等视觉属性，同时保持对提示的遵循。定量结果显示文本对齐（CLIP分数）和视觉对应（LPIPS）之间存在权衡，而VCF在参考保真度方面优于基线方法。

## Abstract
Text-to-image diffusion models like Stable Diffusion generate high-quality images from text, but lack a way to inject visual guidance (e.g. sketches, styles) at inference without retraining. Existing methods either require computationally expensive fine-tuning or rely on style transfer techniques that risk semantic misalignment with textual prompts. We introduce Visual Concept Fusion (VCF), the first method offering dual conditioning on both an image and text prompt at inference time without any concept-specific training. VCF enables visual concept injection into Stable Diffusion by aligning CLIP image features with the text embedding space. VCF consists of three components: (1) a lightweight aligner that maps image tokens to the text embedding manifold using InfoNCE and cross-attention reconstruction losses, (2) a fusion strategy that preserves both textual and visual semantics, and (3) an optional Prompt-Noise Optimization (PNO) module for test-time refinement. Our experiments demonstrate that VCF successfully transfers visual attributes including style, composition, and color palette from reference images while maintaining prompt adherence. Quantitative results show a trade-off between text alignment (CLIP score) and visual correspondence (LPIPS), with VCF outperforming baselines in reference fidelity.

## 1. 研究动机与核心问题
- **核心问题**：文本到图像扩散模型（如 Stable Diffusion）能够根据文字生成高质量图像，但用户难以仅靠文字精确控制画面的风格、构图、色调等视觉要素，通常需要昂贵的提示工程或模型微调。
- **现有方法的不足**：
  - **微调类方法**（如 DreamBooth、StyleDrop、ControlNet）：每引入一个新概念（风格、物体）就需要重新训练模型或适配器，计算成本高、泛化性差。
  - **训练自由类方法**（如 SDEdit、SkipInject）：多只能保留空间布局，难以迁移高层次风格或语义概念。
  - **直接注入图像特征**（朴素融合）会导致特征空间不匹配，生成质量低下。
- **论文目标**：首次实现**在推理时同时接受文本和图像双条件**，无需任何概念特定的训练，即可将参考图像的风格、构图等视觉概念注入生成过程。

## 2. 方法论（VCF 管线）
VCF 由三个关键组件构成，整体流程如图 2 所示。

### 2.1 图像‑文本对齐器（Image Aligner）
- **动机**：Stable Diffusion 使用 CLIP 文本编码器的预投影 token，而 CLIP 视觉编码器的预投影 token 与之分布不匹配（\(p_{\text{text}} \neq p_{\text{image}}\)），直接注入会破坏去噪过程。
- **网络结构**：一个轻量级 MLP（两层线性 + LayerNorm + ReLU），将图像 token \(I \in \mathbb{R}^{m \times d_{\text{image}}}\) 映射为对齐后的 token \(\hat{I} \in \mathbb{R}^{m \times d_{\text{text}}}\)。
- **训练目标**：组合损失函数
  - **全局对齐**：InfoNCE 损失，拉近图像均值嵌入与对应文本均值嵌入。
  - **局部对齐**：交叉注意力重建损失，让对齐后的图像 token 去重建原始文本 token。
  - 总体损失 \(L_{\text{align}} = \lambda L_{\text{InfoNCE}} + L_{\text{attn}}\)，其中 \(\lambda=0.2\)。
- 只有对齐器参与训练，扩散模型保持冻结。

### 2.2 文本‑图像融合策略
在得到对齐后的图像 token \(\hat{I}\) 后，与原始文本 token \(T\) 融合，送入扩散模型。论文考察三种策略：
1. **朴素融合**（均值加权）：将图像 token 平均成一个全局向量，再与每个文本 token 线性混合。会冲淡语义，仅作基线。
2. **拼接融合**（VCF 主方法）：直接将图像 token 拼接到文本 token 序列末尾，形成 \([T; \hat{I}]\)。保留各模态独立语义，实验效果最佳。
3. **交叉注意力融合**：让文本 token 作为查询去关注图像 token，再将融合后的特征缩放后混合回文本 token。会引入噪声和伪影，性能不如拼接。

### 2.3 提示‑噪声优化（PNO，可选）
- **目的**：在推理时进一步优化条件嵌入 \(T_{\text{final}}\) 和初始噪声 \(x_T\)，最大化生成图像与参考图像在 CLIP 空间中的余弦相似度。
- **做法**：在 DDIM 采样前进行少量梯度步（10–50 步），对 \(T_{\text{final}}\) 和 \(x_T\) 联合优化，同时加入噪声正则化项 \(L_{\text{reg}}\) 以防止退化。
- 该模块可单独作用于纯文本生成，也可与融合管线结合使用。

## 3. 实验设计
### 3.1 数据集与模型
- **基座模型**：Stable Diffusion v2 (768-ema-pruned)，分辨率 768×768，DDIM 50 步。
- **对齐器训练数据**：COCO Captions 随机 10% 子集（约 6 万图像‑描述对），按 80/10/10 划分训练/验证/测试。每个 epoch 随机抽取一幅图像的五条描述之一以增强鲁棒性。

### 3.2 评估指标
- **CLIP Score**：生成图像与文本提示之间的 CLIP 余弦相似度，衡量文本对齐程度。
- **LPIPS**（Learned Perceptual Image Patch Similarity）：生成图像与参考图像之间的感知相似度，值越低越相似（权重 \(w_l = 1\)）。

### 3.3 对比方法
- **SDv2 text‑only**：纯文本生成。
- **Naive fusion**：不经过对齐器，直接将图像特征平均后与文本 token 混合。
- **VCF**：本文方法（拼接融合 + 对齐器，可选 PNO）。

由于时间限制，论文未与 SDEdit 等训练自由基线进行定量对比。

## 6. 主要结论与发现
- VCF 能够成功将参考图像的风格、构图、色调等视觉属性迁移到生成结果中，同时保持对文本提示的基本遵循。
- 在抽象或模糊的文本提示下，VCF 的增益尤为明显，能将模糊描述转化为有视觉冲击力的图像。
- 文本对齐与视觉对应存在经典权衡：更强的视觉引导会略微降低 CLIP Score，但显著降低 LPIPS。
- 对齐器的联合损失（InfoNCE + 交叉注意力重建）是实现高质量融合的关键，单独使用某一种损失会导致文本失控或视觉丢失。

## 7. 优点
- **首次实现推理时双条件注入**：不需要任何概念特定的微调，一步到位支持多张参考图像（方向）。

- **设计合理、可解释性强**：通过模态对齐解决分布不匹配问题，融合策略简洁有效，PNO 提供额外的优化杠杆。
- **开源承诺**：实验可复现，权重、代码均将公开。

## 8. 不足与局限
- **视觉特征不可控**：无法指定要迁移参考图像的哪些特征（如仅取色彩而忽略构图），有时会造成不可预测的生成或文本‑图像冲突。

- **PNO 资源开销**：虽优化步数不高，但仍引入额外推理时间，未讨论实时性影响。
