## 摘要
单图像超分辨率（SISR）旨在从低分辨率输入重建高质量图像。尽管近期多模态方法提升了感知质量，但它们仍然对错误先验敏感且需要昂贵的标注。为解决这些问题，我们提出Simon-SR，一个利用可学习提示进行高效语义挖掘和鲁棒文本-图像融合的多模态SISR框架。该方法结合了对比提示学习与提示引导的空间自适应细化，以增强多模态对齐。实验表明，Simon-SR超越了最先进方法，在PSNR上最高提升0.50 dB，SSIM提升0.0133，LPIPS提升0.0695。代码将开源。

## Abstract
Single Image Super-Resolution (SISR) reconstructs high-quality images from low-resolution inputs. While recent multi-modal methods improve perceptual quality, they remain sensitive to erroneous priors and require expensive annotations. To address these issues, we propose Simon-SR, a multi-modal SISR framework leveraging learnable prompts for efficient semantic mining and robust text-image fusion. Our approach combines Contrastive Prompt Learning with Prompt-Guided Spatially Adaptive Refinement to enhance multi-modal alignment. Experiments demonstrate that Simon-SR surpasses state-of-the-art methods, achieving maximum improvements of 0.50 dB in PSNR, 0.0133 in SSIM, and 0.0695 in LPIPS. Code will be released.

## 2. 论文提出的方法论

- **核心思想**：将文本语义视为可学习的隐变量，而非固定的先验知识。通过对比提示学习（Contrastive Prompt Learning）自动从无标注图像中提取语义，再以提示引导的空间自适应调制（Prompt-Guided Spatially Adaptive Refinement）进行特征融合，避免错误先验并降低标注开销。
- **框架组成**：
  - **阶段一：对比提示学习（CPL）**
    - 为每张图像分配一组可学习提示 [X]₁...[X]ₘ，构建模板“A photo of a [X]₁...[X]ₘ [类别]”。
    - 使用冻结的 CLIP 文本编码器 𝒯(·) 和图像编码器 ℐ(·) 分别提取文本和视觉特征。
    - 通过图像到文本（Lᵢ₂ₜ）和文本到图像（Lₜ₂ᵢ）的双向对比损失进行优化，损失函数为：
      ```
      L_con = Σ_x L_i2t(x) + Σ_x L_t2i(x)
      ```
      其中 L_i2t 和 L_t2i 为基于余弦相似度的交叉熵损失。
    - 该过程仅优化提示嵌入，不依赖外部标注，从而缓解语义偏差。
  - **阶段二：提示引导的空间自适应细化（PSAR）**
    - 将优化后的提示与低分辨率图像分别编码为文本特征 𝒇_txt 和图像特征 𝒇_img。
    - **渐进式文本感知细化块（PTRBlock）**：采用 U-Net 结构，每一层通过交叉注意力计算空间相关性：
      ```
      A = σ( Q · K^⊤ / √(C/r) )
      ```
      其中 Q 由图像特征经 1×1 卷积投影得到，K 由文本特征经线性投影得到，σ 为 softmax。
    - **空间自适应仿射变换**：
      ```
      𝒇̂_img = ( I + ΔΓ ⊗ A ) ⊗ 𝒇_img ⊕ Δβ ⊗ A
      ```
      其中 ΔΓ 和 Δβ 分别由图像特征和文本特征通过线性投影生成，并初始化为零。该设计使得语义相关区域被选择性增强，无关区域保持原样。
    - 对抗训练：采用重建损失（L₁）、感知损失（VGG-19）和对抗损失（带有 CLIP 一致性项的判别器），总损失为 L = L_sr + L_percep + 0.02 × L_adv。
