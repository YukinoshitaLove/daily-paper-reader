## 摘要
近年来，基于视觉Transformer（ViT）的模型在图像超分辨率领域展现出卓越的性能。然而，ViT相对于空间分辨率的二次计算复杂度严重制约了其效率，导致高延迟和巨大的内存消耗。为缓解这一问题，人们提出了各种基于窗口的注意力机制；但这些机制本质上削弱了ViT的主要优势——长距离依赖建模。为了克服这些限制，我们提出了聚类单元级相似性Transformer（CUST），这是一种有效融合全局与局部信息的新型架构。具体而言，CUST使每个图像块能够在其局部窗口之外的更广泛区域范围内聚合并关注相似的块，从而捕获广泛的上下文理解。此外，它采用重叠注意力窗口来捕获局部依赖性，同时通过计算原始特征与其下采样-上采样对应特征之间的残差差异来显式提取高频细节。综合实验表明，我们提出的模型在计算效率和重建性能之间实现了实用的平衡。在现实约束下，与近期全局上下文或轻量级模型相比，它实现了更低的内存占用和更快的推理速度。代码已开源在 https://github.com/jwgdmkj/CUST。

## Abstract
Recently, Vision Transformer (ViT)-based models have exhibited remarkable performance in image super-resolution. However, the quadratic computational complexity of ViTs with respect to spatial resolution severely constrains their efficiency, leading to high latency and massive memory consumption. To alleviate this, various window-based attention mechanisms have been proposed; yet, they inherently compromise the long-range dependency modeling that is the primary advantage of ViTs. To overcome these limitations, we propose the Clustered Unit-level Similarity Transformer (CUST), a novel architecture that efficiently integrates global and local information. Specifically, CUST enables each patch to aggregate and attend to similar patches within a broadened regional scope outside its local window, thereby capturing extensive contextual understanding. Furthermore, it employs overlapping attention windows to capture local dependencies, while explicitly extracting high-frequency details by computing the residual difference between the original features and their downsampled-upsampled counterparts. Comprehensive experiments demonstrate that our proposed model achieves a practical balance between computational efficiency and restoration performance. It achieves a lower memory footprint and faster inference speed compared to recent global context or lightweight models under realistic constraints. Code is available at [https://github.com/jwgdmkj/CUST].

## 2. 方法论
### 整体架构
CUST 由浅特征提取卷积层、多个深层特征提取块和重建模块组成。每个深层特征提取块包含：
- **CANA**（跨窗口亲和近邻注意力）：捕获长距离语义依赖。
- **MEDA**（多频误差驱动密集注意力）：强化局部相关性与高频细节。
- 两层 ConvFFN 和层归一化。

### 2.1 CANA 模块
- **核心思想**：在扩大的搜索区域内，通过语义相似性对图像块进行跨窗口聚类，使每个块能关注到窗口外具有相似纹理/结构的远距离块。
- **关键步骤**：
  1. **代表性令牌生成**：对搜索区域（尺寸`GS×GS`）内每个窗口（`ws×ws`）做平均池化，得到`N`个代表性窗口令牌`X_mgrouped`。
  2. **相似性聚类**：计算区域内所有空间块与窗口令牌的点积相似度，通过 `Argmax` 将每个块分配至最相似的窗口令牌，再用 `Argsort` 形成聚类`C_n`。
  3. **键/值扩展**：为避免信息在聚类边界断开，将邻接聚类（`C_{n-1}`、`C_{n+1}`）的最近半区拼接到当前查询的键和值中。
  4. **掩码注意力**：通过注意力掩码`M`仅允许共享相同窗口 ID 的块之间进行交互，计算带掩码的软注意力输出，并经可学习门控`G_n`调制。
  - 公式示意：`Y_n = GELU(Softmax(Q_n K_exp^T / √d + M)·V_exp) ⊙ G_n`。

### 2.2 MEDA 模块
- **核心思想**：在下采样-上采样残差中显式提取高频误差信号，引导重叠窗口注意力关注纹理和边缘，以较小内存代价恢复精细局部结构。
- **算法流程** (Algorithm 1)：
  1. **高频误差提取**：`X_error = X - Up(Down(X))`，获得多尺度差异。
  2. **误差精炼与门控**：`X_error` 经过扩张卷积（Refiner）生成上下文特征，再经 1×1 卷积生成空间门控图`G`。
  3. **特征补偿**：`X_refined = X + α·(F_refine ⊙ G)`，α为可学习标量。
  4. **局部细节精炼**：对补偿后的特征应用重叠窗口自注意力（来自 HPI-Net 的重叠注意力），最终输出。

## 6. 主要结论与发现
- CUST 在重建质量（PSNR/SSIM）、推理速度和峰值内存之间取得了目前最先进的**实用平衡**。
- 与全局上下文模型（如 CATANet）相比，CUST 在 ×4 尺度上平均 PSNR 提升 0.094 dB，同时内存平均降低 83%。
- 与内存最优的 SRFormer-light 相比，CUST 在内存略增 0.87% 的情况下推理速度提速 29.8%。
- 消融证明 CANA 和 MEDA 相互协同，前者有效扩展感受野捕获长程语义，后者利用高频误差补偿窗口限制、降低内存负荷。
- 多频误差算法使得在较小窗口下仍可维持甚至提升性能，硬件效率显著提高。
