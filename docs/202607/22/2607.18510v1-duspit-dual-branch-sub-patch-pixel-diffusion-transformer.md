## 摘要
扩散Transformer在图像生成上表现出色，但大多在压缩的潜在空间中操作。像素空间扩散避免了这种信息损失，然而现有方法将每个原始图像块映射到单个令牌，迫使一种表示同时处理全局通信和细粒度细节。我们通过提出一种新架构来解决这个问题，\textbf{DuSPiT}，即\textbf{双}分支\textbf{子}块\textbf{像素}Transformer。该模型将全局结构推理与局部外观建模分离开来。DuSPiT使用一个紧凑的基础分支进行高效的全局推理，以及一个并行的、高容量的像素分支，组织成子块组，以保留详细的外观，两个分支通过交叉注意力进行交互。我们的结果显示，DuSPiT生成的图像具有更丰富的细节和更强大的细粒度结构，同时也比先前的像素空间扩散Transformer实现了更好的质量-效率权衡。

## Abstract
Diffusion Transformers achieve strong image generation performance, but most operate in compressed latent spaces. Pixel-space diffusion avoids this information loss, yet existing approaches map each raw image patch to a single token, forcing one representation to handle both global communication and fine-grained details. We address this issue by proposing a new architecture, \textbf{DuSPiT}, a \textbf{Du}al-branch \textbf{S}ub\textbf{P}atch \textbf{Pi}xel \textbf{T}ransformer. This model separates global structural reasoning from local appearance modeling. DuSPiT uses a compact base branch for efficient global reasoning and a parallel, high-capacity pixel branch, organized into subpatch groups, to preserve detailed appearance, with the two branches interacting through cross-attention. Our results show that DuSPiT generates images with richer details and stronger fine-grained structures, while also achieving a better quality--efficiency trade-off than prior pixel-space diffusion transformers.

### 2. 论文提出的方法论：DuSPiT

论文的核心思想是“解耦”，并由此提出了名为 **DuSPiT** 的双分支架构。

*   **核心思想**: 将全局结构推理与局部细节建模分配到两个独立但相互作用的并行分支中，避免相互掣肘。
*   **关键技术细节**:
    *   **双分支表征**:
        *   **基础分支 (Base Branch)**: 负责高效的全局推理。它使用紧凑的低维令牌 zᵢ ∈ ℝ^dp，与标准DiT类似，捕捉图像的整体布局和语义关系。
        *   **像素分支 (Pixel Branch)**: 专注于保留高保真外观细节。它引入高容量令牌 aᵢ ∈ ℝ^da，其中 da ≥ P²C (原始像素维度)，且 da ≫ dp，用于存储纹理、色彩等丰富信息。该分支不参与密集的全局自注意力计算，避免了巨大的计算开销。
    *   **子块组组织**: 像素分支的高维令牌并非无结构向量，而是被划分为 n × n 个子块组 (subpatch groups)。每个组对应图像块内的一个更精细空间区域，从而显式地保留了内容的内部空间结构。
    *   **层次化位置编码**: 为桥接两个分支，设计了一种新型位置编码方案：
        *   **块级编码**: 基础分支使用标准的2D RoPE编码全局位置 (u, v)。
        *   **子块级编码**: 为像素分支的每个子块组追加其相对块中心的偏移量 (Δx, Δy)，形成 `RoPE(u+Δx, v+Δy)`。这使模型既能感知全局布局，又能理解块内几何关系。
    *   **交叉注意力交互 (Cross-Attention)**: 这是两个分支的交互枢纽。在每个Transformer块中，像素分支的子块查询 (Query) 会与来自同一图像块的、融合了全局上下文的基础分支令牌（作为Key和Value）进行交叉注意力计算。这种**非对称**的信息流（基础分支指导像素分支）是关键设计，使基础分支成为一个稳定的结构性瓶颈。
    *   **输出解耦**: 最终的去噪预测仅从像素分支的高容量表征中产生，通过一个轻量级输出头投影回图像空间，基础分支不做直接输出。
