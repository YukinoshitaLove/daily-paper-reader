## 摘要
连续扩散模型已成为逼真图像超分辨率（SR）的主流范式，但它们通常将重建建模为连续信号级别的去噪，并通过外部条件模块引入语义先验。这使得直接利用现代多模态模型统一的基于token的扩展范式不够直接。自回归模型通过将图像建模为离散视觉token，提供了更原生的语义表示，但其因果解码对于高分辨率重建效率低下。离散扩散通过支持对视觉token进行非因果、并行预测，提供了一个有前景的中间方案。然而，由于两个特定任务的挑战，直接将离散扩散应用于SR仍然并非易事：（1）视觉token的长尾分布，使得稀有但感知关键的纹理被低估表示；（2）空间不一致的并行解码，可能引入孤立的伪影。为了解决这些问题，我们提出了DiMOO-SR，一个感知稀有的多模态离散扩散框架，用于逼真图像SR。在训练期间，逆频率采样（IFS）优先考虑被低估但信息丰富的token。在推理期间，空间一致性排名（SCR）利用局部邻域一致性来细化token置信度，以提高结构连贯性。在广泛使用的真实世界SR基准上的大量实验表明，DiMOO-SR仅需少量并行解码步骤即可实现有竞争力的感知质量，突显了离散扩散在生成式图像超分辨率方面的潜力。代码将在发表后发布。

## Abstract
Continuous diffusion models have become the dominant paradigm for photo-realistic image Super-Resolution (SR), but they typically formulate reconstruction as continuous signal-level denoising and incorporate semantic priors through external conditioning modules. This makes it less direct to exploit the unified token-based scaling paradigm of modern multimodal models. Autoregressive models provide a more native semantic representation by modeling images as discrete visual tokens, yet their causal decoding is inefficient for high-resolution reconstruction. Discrete diffusion offers a promising middle ground by enabling non-causal, parallel prediction over visual tokens. However, directly adapting discrete diffusion to SR remains non-trivial due to two task-specific challenges: (1) the long-tailed distribution of visual tokens, which under-represents rare but perceptually critical textures; and (2) spatially inconsistent parallel decoding, which may introduce isolated artifacts. To address these issues, we propose DiMOO-SR, a rarity-aware multimodal discrete diffusion framework for photo-realistic image SR. During training, Inverse Frequency Sampling (IFS) prioritizes under-represented but information-rich tokens. During inference, Spatial Consistency Ranking (SCR) refines token confidence using local neighborhood agreement to improve structural coherence. Extensive experiments on widely used real-world SR benchmarks demonstrate that DiMOO-SR achieves competitive perceptual quality with only a few parallel decoding steps, highlighting the potential of discrete diffusion for generative image super-resolution. The code will be released upon publication.

- **问题背景**：逼真图像超分辨率（SR）需同时恢复低频纹理与高频语义细节。当前主导的连续扩散模型（如 StableSR、PASD、SeeSR）通过连续信号去噪生成图像，但语义先验通常需借助外部条件模块（如 CLIP）注入，难以融入现代多模态大模型统一的 token 化序列生成范式。
- **范式对比**：自回归模型（如 PURE）将图像量化为离散 token 并利用大语言模型进行原生语义理解，但因果解码需逐 token 预测，对高分辨率重建效率极低。离散扩散模型（DDM）支持并行、非因果的 token 预测，成为兼顾语义理解与效率的“第三条路”。
- **本文核心问题**：将现有多模态离散扩散模型直接用于 SR 时存在两个瓶颈：
  1. 视觉 token 的长尾分布：常见平坦区域 token 占多数，稀有纹理 token 被欠训练，导致模型倾向重建平滑背景而忽略感知关键细节；
  2. 并行解码的空间不一致：独立基于点置信度选择 token，缺乏空间邻域一致性约束，易产生孤立伪影，破坏结构连贯性。
- **整体含义**：提出 DiMOO-SR，通过训练时逆频率采样（IFS）缓解 token 不平衡，推理时空间一致性排序（SCR）增强解码结构连贯性，展现离散扩散在生成式 SR 领域的潜力。

## 2. 方法论

### 2.1 整体框架
- 基于 Lumina-DiMOO 架构，使用双向注意力 Transformer 作为主干。
- 输入为 512×512 高分辨率图像，通过 VQ-GAN 编码为 32×32 的离散 token 图（词汇量 K=8192）。
- 多模态条件融入：利用 Qwen3-VL 提取图像文本描述，得到文本 token，与低分辨率条件 token、高分辨率 token 拼接为统一序列输入 Transformer，通过双向自注意力实现跨模态交互，无需额外交叉注意力模块。

### 2.2 训练：逆频率采样 (IFS)
- **动机**：视觉 token 的码本利用率呈严重长尾（头部 40% 索引占 90% 以上使用量），稀有 token 对应高频纹理，自信息高；均匀随机掩码使训练监督集中在头部 token，忽略纹理重建。
- **IFS 概率定义**：对于 token 序列中位置 i，采样概率与 token 的自信息指数温度加权相关：
  `P_IFS(i) = exp(α·I(x_i)) / Σ_j exp(α·I(x_j))`，其中 `I(k) = -log p(k)`，α 为信息温度。
- **混合采样策略**：纯 IFS 可能因高掩码率导致上下文不足而崩溃。采用伯努利混合（p=0.5）交替两种分支：
  - 全局结构分支：均匀随机掩码，掩码率 γ ~ U(0,1)，学习从零构建全局布局。
  - 硬细化分支：IFS 掩码，掩码率限制在 γ ~ U(0,0.6)（保留至少 40% 上下文），专注纹理细节。
- **防御性混合解释**：混合分布确保每个 token 都有非零采样概率，且 IFS 分量的重要性权重被 bounded（≤2），避免梯度方差暴增。

### 2.3 推理：空间一致性排序 (SCR)
- **动机**：标准并行解码独立按点置信度排序掩码 token，缺少空间共识，可能保留空间孤立的错误预测。
- **锚定置信场**：对已保留的视觉 token 设置信度为 1（可靠锚点），当前预测 token 用原始候选概率 `c_i`。
- **局部邻域一致性**：对位置 i，计算固定窗口（5×5）内平均置信度 `¯c_{N_i}`。
- **可靠性门控更新**：
  - 若 `c_i ≥ τ`（高置信），保持不变；
  - 若 `c_i < τ`，融合局部共识：`˜c_i = (1-λ)c_i + λ¯c_{N_i}`。
- 仅影响排序决策，不直接改变 token 值或像素，属于不引入额外参数的后处理模块（λ=0.7, τ=0.5）。
- 解码步数设为 T=5，大幅降低推理延迟。
