## 摘要
基于扩散的方法利用大型预训练稳定扩散模型作为强大的生成先验，在真实图像超分辨率中取得了令人印象深刻的性能。然而，这些方法仍面临两个关键局限。首先，现有的基于SD的单步和多步真实图像超分辨率方法对所有输入样本采用统一的处理范式，忽略了不同图像之间恢复难度的差异。其次，SD模型中VAE的剧烈分辨率降低（例如8倍下采样）导致细粒度细节的不可逆损失，这些细节无法被后续的扩散过程恢复。为了解决这些局限，我们提出了一种难度感知动态路由策略，克服了僵化、一刀切的处理范式。具体来说，我们首先设计了一个难度估计器来预测每张输入图像的恢复成本，从而实现自动分配到适当容量的网络。然后，我们通过调节SD骨干中VAE的空间下采样比，构建了一组具有不同模型容量的真实图像超分辨率网络，从而为困难样本保留更多高频信息，同时为简单输入保持效率。大量实验表明，与近期最先进的方法相比，所提出的模型具有优越的效率和有效性。

## Abstract
Diffusion-based methods have achieved impressive performance in real-world image super-resolution (Real-ISR) by leveraging large pre-trained stable diffusion (SD) models as powerful generative priors. However, these methods still face two key limitations. First, existing SD-based one-step and multi-step Real-ISR approaches adopt a unified processing paradigm for all input samples, ignoring the varying restoration difficulty across images. Second, the aggressive resolution reduction of the VAE in SD models (e.g., 8x downsampling) leads to irreversible loss of fine-scale details, which cannot be recovered by the subsequent diffusion process.   To address these limitations, we propose a Difficulty-aware Dynamic Routing (DDR) strategy that overcomes the rigid, one-size-fits-all processing paradigm. Specifically, we first design a difficulty estimator to predict the restoration cost of each input image, enabling automatic assignment to a network of appropriate capacity. Then, we construct a set of Real-ISR networks with varying model capacities by modulating the spatial downsampling ratio of the VAE in the SD backbone, thereby preserving more high-frequency information for challenging cases while maintaining efficiency for simpler inputs.   Extensive experiments have demonstrated the superior efficiency and effectiveness of the proposed model compared to recent state-of-the-art methods.

### 1. 论文的核心问题与整体含义

- **核心问题**：真实世界图像超分辨率中的现有扩散模型方法存在两个关键局限：
  1. **统一处理范式**：对所有输入图像不加区分地使用相同模型与推理策略，忽视了不同图像在退化复杂度与恢复难度上的显著差异，导致困难样本重建不足、简单样本计算浪费。
  2. **VAE下采样导致的细节丢失**：Stable Diffusion 中 VAE 激进的下采样（例如 8×）会不可逆地丢失高频细节，后续扩散过程难以恢复，尤其影响纹理、文字等细粒度结构的保真度。

- **整体含义**：提出**难度感知动态路由框架**，根据每张图像的恢复难度自适应分配不同容量的超分网络，从而在保证视觉质量的同时提升效率，实现“按需分配”的高效真实图像超分辨率。

### 2. 论文提出的方法论

- **核心思想**：
  - 放弃“一刀切”处理，根据图像的高频能量衰减量化恢复难度，并通过动态路由将困难样本送入细节保持能力更强的低压缩率VAE专家网络，简单样本送入高效率高压缩率VAE专家网络。

- **关键技术细节**：
  1. **难度估计器**：基于高频能量衰减。通过2D傅里叶变换分别计算低频图像与真实高分辨率图像在高频区域的频谱能量，定义难度  
     \(d = 1 - \frac{E_{\text{high}}(I_{\text{deg}})}{E_{\text{high}}(I_{\text{gt}})}\)  
     取值范围 \([0,1]\)，越大表示细节损失越严重、恢复越困难。
  2. **专家网络构建**：
     - **Expert‑D4**：使用 VAED4（4×下采样），保留更多细节，侧重高保真重建；
     - **Expert‑D8**：使用原 Stable Diffusion 的 VAED8（8×下采样），侧重高效推理。
     - 两者共享扩散UNet主干，通过LoRA高效微调。
  3. **混合训练策略**：Expert‑D4 使用 80% 困难样本 + 20% 简单样本训练，Expert‑D8 使用 80% 简单样本 + 20% 困难样本训练，增强各自鲁棒性。
  4. **损失函数**：结合 MSE、LPIPS 和 VSD 损失（权重 \(\lambda_1=2, \lambda_2=1\)）进行单步扩散训练（类似 OSEDiff 范式）。
  5. **推理路由**：给定输入图像，计算其难度分数 \(d\)，若 \(d \ge \tau\) 则分配给 Expert‑D4，否则分配给 Expert‑D8。阈值 \(\tau\) 默认为训练集难度中位数，用户可调节以平衡质量与速度。

### 6. 论文的主要结论与发现

- 现有统一处理方法在困难样本上性能下降明显，而简单样本上模型能力冗余。
- 所提出的 DDR‑SR 通过难度感知路由，动态匹配专家容量，在所有方法中取得最佳 SSIM、LPIPS 和 CLIPIQA，且在 DISTS、FID、MUSIQ 上排名前二。
- DDR‑SR 在保持细节（如文字、纹理、面部特征）方面明显优于其他方法，同时 Expert‑D4 具有最低的 FLOPs（1.81T）和较少的参数量（1.65B），实现高效与高质量的平衡。

### 7. 优点

- **首次将难度感知动态路由引入扩散超分**，改变“一刀切”范式，实现样本自适应推理。
- **VAE压缩比可调配**：通过不同下采样比的专家网络，灵活保留高频信息，缓解了 VAE 导致的结构细节丢失问题。
- **用户可控的效率‑质量权衡**：推理时可通过调整路由阈值或专家比例，适应不同应用场景需求。
- **参数高效训练**：基于 LoRA 微调大模型，避免全参数训练开销，且混合训练策略增强了专家网络的鲁棒性。

### 8. 不足与局限

- **难度估计器的稳健性未充分讨论**：仅在合成退化基准上训练和测试，真实场景下难以获得 ground truth 高频参考，推理时如何计算难度可能受限或需要近似。
- **路由阈值 \(\tau\) 的敏感性分析缺失**：虽然提出用户可调节，但实验中未展示不同 \(\tau\) 下性能与速度的变化趋势。
- **数据集局限**：训练和测试主要在特定真实世界基准（RealSR、DRealSR）上，未考察更多样化的真实退化类型或极低分辨率场景。
- **指标偏重感知质量**：在 PSNR 上未全面超越所有对比方法，且在 MANIQA、NIQE 无参考指标上略逊于某些多步方法，说明细节丰富度可能仍受制于单步生成范式。
- **仅处理 4× 超分任务**：未扩展到其他放大倍数，框架的通用性有待验证。
