## 摘要
临床心脏MRI通常以高面内分辨率但粗糙的层间分辨率采集，以减少扫描时间并适应屏气和心脏运动限制，这限制了三维分析和诊断准确性。我们提出STRMSR，一种参考和记忆引导的层间超分辨率框架，通过利用来自同一受试者的高分辨率参考视图和作为记忆的中间超分辨率结果，重建高分辨率心脏容积。我们的方法使用从粗到细的上下文匹配，在空间错位下建立低分辨率目标与参考/记忆图像之间的鲁棒对应。一个可学习的逐块动态特征聚合模块为每个局部块预测内容自适应的混合权重，有效融合动态信息同时抑制不可靠的特征迁移。存储在记忆库中的中间超分辨率结果确保了超分辨率三维容积的层间一致性。在WHS心脏MRI数据集上，采用两种参考协议（正交平面视图和长轴腔室视图）的实验表明，在4倍和8倍上采样因子下，我们的方法相对于基线方法有了一致的提升。

## Abstract
Clinical cardiac MRI is commonly acquired with high in-plane resolution but coarse through-plane resolution to reduce scan time and accommodate breath-hold and cardiac-motion constraints, which limits 3D analysis and diagnostic accuracy. We propose STRMSR, a reference- and memory-guided through-plane super-resolution (SR) framework that reconstructs high-resolution (HR) cardiac volumes by leveraging HR reference views acquired from the same subject and intermediate SR results as the memory. Our method uses coarse-to-fine contextual matching to establish robust correspondence between low-resolution target and reference/memory images under spatial misalignment. A learnable patch-wise dynamic feature aggregation module predicts content-adaptive mixture weights for each local patch, effectively fusing dynamic information while suppressing unreliable feature transfers. The intermediate SR results stored in the memory bank ensure slice-to-slice consistency for the super-resolved 3D volume. Experiments on the WHS cardiac MRI dataset under two reference protocols, orthogonal-plane views and long-axis chamber views, demonstrate consistent improvements over baselines at 4x and 8x upsampling factors.

## 论文核心问题与整体含义
- **研究动机**：临床心脏MRI（CMR）通常采用2D屏气采集，虽然面内（in-plane）分辨率高，但层间（through-plane）分辨率很粗糙，导致获得的容积数据高度各向异性，严重限制后续的三维可视化、量化分析和疾病诊断。
- **整体含义**：论文聚焦于心脏MRI的层间超分辨率（SR）问题，旨在从已有的各向异性低分辨率（LR）堆栈中恢复各向同性的高分辨率（HR）三维容积。其核心思想是利用同一次检查中常规获取的少量高分辨率参考视图（如长轴视图或正交平面视图）以及已重建切片的结果（记忆），指导并约束整个LR堆栈的超分辨率重建，从而在保持层间一致性的同时恢复精细的解剖细节。

## 方法论
- **框架名称**：STRMSR（Sp - Temporal Reference and Memory guided Super-Resolution）。
- **整体思路**：将LR容积视为沿层间方向的“视频”，首先借助HR参考图像对第一帧进行SR，之后将已重建的中间结果存入记忆库，用于指导后续帧的重建，实现层间SR信息的传播。

- **关键技术细节**：
  1. **双分支Transformer特征提取**：
     - 使用基于Swin Transformer的残差块（RSTB）构建编码器。
     - 目标LR图像和参考LR图像（由HR参考下采样得到）共享参数提取特征，保证嵌入空间一致；参考HR图像由独立分支提取特征，以保留分辨率特异性细节。
  2. **粗到细上下文匹配（CFCM）**：
     - 在特征金字塔的最粗级，将目标特征划分为块，通过多膨胀归一化相关找到参考特征中的最佳匹配块，然后在局部区域内进行密集的3×3 patch匹配，计算余弦相似度得到对应索引和置信度。
     - 利用索引从HR特征中提取对应patch，并置信度加权“折叠”（fold）生成翘曲的HR特征。
     - 将粗级的匹配中心传播到更细级别，并在小半径内重新进行局部密集匹配以更新对应关系，最终得到多尺度的翘曲HR特征。
  3. **逐块动态特征聚合（PDFA）**：
     - 对于多个翘曲的特征图（来自多个参考/记忆帧），将其分割成非重叠块，通过平均池化得到块描述符。
     - 用一个轻量MLP预测每个块的logit，通过softmax归一化为权重，然后广播到块内每个像素，对各特征图进行加权求和，实现内容自适应的特征融合，抑制不可靠的参考信息。
  4. **基于记忆的SR传播**：
     - 将LR目标图像与其对应的SR结果成对存入先进先出（FIFO）记忆库。后续帧在SR时同时参考外部HR视图和记忆库中的最近几个SR帧，以强制层间连续性。
  5. **损失函数**：结合图像域的L1重建损失与k空间损失（在复原图像与真实HR图像的二维FFT上计算L2），权重分别为1和0.001。

## 实验设计
- **数据集**：WHS心脏MRI数据集，共86位受试者，划分为训练集50例、验证集6例、测试集30例。
- **实验场景与协议**：
  - **正交平面参考协议（WHS-Ortho）**：以轴向图像为目标LR（×4或×8下采样），以矢状方向图像作为HR参考（同样经历下采样以保持HR层内分辨率），代表密集覆盖的参考模式。
  - **长轴参考协议（WHS-LAX）**：以短轴（SAX）图像为目标LR，长轴2腔、3腔、4腔视图为HR参考，模拟临床常规采集的稀疏参考模式。
- **对比基准方法**：
  - Bicubic：单图插值。
  - MsFF-Net：3D多分辨率CNN，融合HR参考体积。
  - MINet：使用单一配对参考的多对比度SR网络。
  - MASA：基于参考的图像SR，支持多参考融合与显式匹配。
  - McMRSR：基于Transformer的多尺度上下文匹配与聚合方法。
  - 所有基线均在相同数据划分和训练设置下重新训练。
- **评估指标**：PSNR、SSIM、MSE，并采用Wilcoxon符号秩检验进行统计显著性分析（p<0.001)。

## 主要结论与发现
- STRMSR在所有协议和放大倍数下均取得最优或具有竞争力的表现，尤其在挑战性更大的×8层间超分上，显著优于所有基线方法（提升0.55~1.17 dB PSNR）。
- 粗到细的上下文匹配比直接复用粗级对应关系效果提升明显（消融实验中去除CFCM导致PSNR下降0.60 dB）。
- 逐块动态特征聚合有效避免不可靠参考信息的干扰，去除后PSNR下降0.24 dB。
- 记忆传播机制增强了层间一致性，去除后PSNR降低0.12 dB，且时间剖面显示其能更好保持线性结构。
- 该方法在参考覆盖密集（正交平面）和稀疏（长轴视图）两种场景下均适用，显示出跨几何形态的泛化能力。
