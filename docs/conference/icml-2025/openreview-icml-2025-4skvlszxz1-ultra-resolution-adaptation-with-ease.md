## Abstract
Text-to-image diffusion models have achieved remarkable progress in recent years. However, training models for high-resolution image generation remains challenging, particularly when training data and computational resources are limited. In this paper, we explore this practical problem from two key perspectives: data and parameter efficiency, and propose a set of key guidelines for ultra-resolution adaptation termed URAE. For data efficiency, we theoretically and empirically demonstrate that synthetic data generated by some teacher models can significantly promote training convergence. For parameter efficiency, we find that tuning minor components of the weight matrices outperforms widely-used low-rank adapters when synthetic data are unavailable, offering substantial performance gains while maintaining efficiency. Additionally, for models leveraging guidance distillation, such as FLUX, we show that disabling classifier-free guidance, i.e., setting the guidance scale to 1 during adaptation, is crucial for satisfactory performance. Extensive experiments validate that URAE achieves comparable 2K-generation performance to state-of-the-art closed-source models like FLUX1.1 [Pro] Ultra with only 3K samples and 2K iterations, while setting new benchmarks for 4K-resolution generation. ### 2. 论文提出的方法论：核心思想、关键技术细节

URAE 由三大关键准则组成：

*   **准则 1：数据效率——合成数据优于真实数据**
    *   **核心思想**：利用强大教师模型（如 FLUX1.1 [Pro] Ultra）生成的合成数据训练学生模型，可显著促进训练收敛。
    *   **理论依据**：在 NTK 无限宽网络假设下，证明当训练数据由真实数据和合成数据混合组成时，最终参数与最优参数的距离受三项约束：初始误差衰减、标签噪声积累（合成数据可降低此项）以及参考模型与最优模型的固差距。若参考模型足够准确，合成数据能有效降低噪声影响。2K 场景下，教师模型可提供高质量合成数据，因此 URAE 完全使用合成数据进行训练。

*   **准则 2：参数效率——无合成数据时，微调权重矩阵的“次要成分”**
    *   **核心思想**：当无法获取高质量合成数据（如 4K 场景）时，传统的低秩适配器（LoRA）调优的是权重矩阵的“主要成分”（对应大奇异值），但超分辨率适配更需要学习细节纹理和局部排列，这些信息往往存在于“次要成分”（小奇异值）中。
    *   **技术细节**：
        1. 对权重矩阵 \(W\) 进行奇异值分解（SVD）：\(W = U\Sigma V\)。
        2. 提取与最小 \(r\) 个奇异值对应的成分：\(W_{\text{small}} = U_{[:, -r:]} \Sigma_{[-r:, -r:]} V_{[-r:, :]}\)。
        3. 在训练时，冻结剩余部分 \(W_{\text{res}}\)，仅更新等效于 \(W_{\text{small}}\) 的低秩矩阵 \(A,B\)，其中 \(A = U_{[:, -r:]}\sqrt{\Sigma_{[-r:, -r:]}}\)，\(B = \sqrt{\Sigma_{[-r:, -r:]}} V_{[-r:, :]}\)。
    *   **优势**：既保留了大部分参数以维持语义和布局能力，又能高效学习高频细节，相比 LoRA 在真实噪声数据上表现更优。

*   **准则 3：训练时禁用无分类器引导（CFG）**
    *   **核心思想**：对于像 FLUX 这样经过引导蒸馏的模型，在适配训练阶段必须将 CFG 尺度设置为 1（即禁用 CFG）。
    *   **理由**：蒸馏阶段模型学习的是包含空条件分支的输出（\(g>1\)），而适配阶段的目标是流匹配损失 \(\epsilon - z_0\)，与空条件无关。若训练时 \(g>1\)，会导致两个阶段的优化目标不一致，性能严重下降。设置 \(g=1\) 可保持目标一致。实验证明，推理时再开启 \(g>1\) 仍能良好泛化。

### 6. 论文的主要结论与发现

*   **合成数据显著提升收敛**：在 2K 场景下，完全使用教师模型生成的合成数据即可达到甚至超越使用大量噪声真实数据的效果，且仅需 3K 样本。
